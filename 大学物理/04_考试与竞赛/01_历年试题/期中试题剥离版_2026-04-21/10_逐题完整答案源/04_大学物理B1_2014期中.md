### 第1题
题目：
半径 \(r=1.5\text{ m}\) 的飞轮，初角速度 \(\omega_0=10\text{ rad/s}\)，角加速度 \(\alpha=-5\text{ rad/s}^2\)。求角位移再次为 0 的时刻，以及该时刻轮缘点的线速度。
分析：
这是匀角变速转动，先由 \(\theta(t)=\omega_0 t+\dfrac12 \alpha t^2\) 求非零时刻，再由 \(v=r\omega\) 求线速度。
解答步骤：
1. 角位移为 \(\theta=10t-\dfrac52 t^2\)。
2. 令 \(\theta=0\)，得 \(t(10-\dfrac52 t)=0\)，除去 \(t=0\)，有 \(t=4\text{ s}\)。
3. 此时角速度 \(\omega=\omega_0+\alpha t=10-5\times 4=-10\text{ rad/s}\)。
4. 轮缘点线速度大小为 \(|v|=r|\omega|=1.5\times 10=15\text{ m/s}\)。
最终答案：
再次回到原角位置的时刻是 \(4\text{ s}\)；线速度大小为 \(15\text{ m/s}\)。
易错点：
不要把“角位移为 0”误当成“角速度为 0”；本题求的是回到原角位置的时刻。

### 第2题
题目：
总长为 \(L\) 的细杆，线密度 \(\lambda=ax+b\)。求质心坐标 \(x_C\)。
分析：
直接用质心定义 \(x_C=\dfrac{\int x\,dm}{\int dm}\)，其中 \(dm=\lambda(x)\,dx\)。
解答步骤：
1. 总质量 \(M=\int_0^L (ax+b)\,dx=\dfrac12 aL^2+bL\)。
2. 对原点的一阶矩为 \(\int_0^L x(ax+b)\,dx=\dfrac13 aL^3+\dfrac12 bL^2\)。
3. 因此 \(x_C=\dfrac{\frac13 aL^3+\frac12 bL^2}{\frac12 aL^2+bL}\)。
4. 化简得 \(x_C=\dfrac{L(2aL+3b)}{3(aL+2b)}\)。
最终答案：
质心坐标为 \(\dfrac{L(2aL+3b)}{3(aL+2b)}\)。
易错点：
线密度不是常量，不能直接套均匀细杆的 \(L/2\)。

### 第3题
题目：
质量为 \(m\) 的质点沿 \(x\) 轴运动，速度满足 \(v=kx\)。求合力与坐标关系，以及从 \(x_0\) 到 \(x_1\) 所需时间。
分析：
已知 \(v(x)\) 时，应使用 \(a=v\dfrac{dv}{dx}\)；时间由 \(dt=\dfrac{dx}{v}\) 积分得到。
解答步骤：
1. 由 \(v=kx\) 得 \(\dfrac{dv}{dx}=k\)。
2. 因此 \(a=v\dfrac{dv}{dx}=kx\cdot k=k^2x\)。
3. 合力 \(F=ma=mk^2x\)。
4. 又 \(dt=\dfrac{dx}{kx}\)，积分得到 \(t=\dfrac1k\ln\left|\dfrac{x_1}{x_0}\right|\)。
最终答案：
\(F=mk^2x\)，所需时间为 \(\dfrac1k\ln\left|\dfrac{x_1}{x_0}\right|\)。
易错点：
不能把加速度误写成 \(dv/dx\)。

### 第4题
题目：
已知均匀长方形薄板绕边轴的转动惯量，求其绕过中心且垂直板面的轴的转动惯量。
分析：
先利用平行轴定理求过中心的板内主轴转动惯量，再用垂直轴定理。
解答步骤：
1. 设板边长为 \(a,b\)，质量为 \(m\)。
2. 与边平行的中心轴转动惯量分别为 \(I_x=\dfrac{mb^2}{12}\)，\(I_y=\dfrac{ma^2}{12}\)。
3. 由垂直轴定理可得 \(I_z=I_x+I_y\)。
4. 因而 \(I_z=\dfrac{m(a^2+b^2)}{12}\)。
最终答案：
过中心且垂直板面的转动惯量为 \(\dfrac{m(a^2+b^2)}{12}\)。
易错点：
边轴和中心轴不是同一根轴，中间必须做一次轴平移。

### 第5题
题目：
圆锥摆摆锤质量为 \(m\)，在水平面内以角速度 \(\omega\) 匀速转动，转动半径为 \(r\)。求经过 \(1/4\) 周期后，重力对悬点 \(O\) 的冲量矩大小。
分析：
重力矩大小恒为 \(mgr\)，但方向在水平面内随时间转动，因此要做矢量积分。
解答步骤：
1. 经过 \(1/4\) 周期的时间是 \(\Delta t=\dfrac{T}{4}=\dfrac{\pi}{2\omega}\)。
2. 冲量矩为 \(\vec J_M=\int_0^{\Delta t}\vec M\,dt\)。
3. 把力矩方向在水平面内分解并积分，可得矢量积分后的模为 \(\dfrac{\sqrt2\,mgr}{\omega}\)。
最终答案：
重力对悬点的冲量矩大小为 \(\dfrac{\sqrt2\,mgr}{\omega}\)。
易错点：
冲量矩不是“力矩大小乘时间”这么简单，因为力矩方向在持续变化。

### 第6题
题目：
倾角 \(30^\circ\) 的斜面放在水平面上，一质量 \(2\text{ kg}\) 的物体沿斜面下滑，加速度为 \(3.0\text{ m/s}^2\)，取 \(g=10\text{ m/s}^2\)。求物体与斜面间的滑动摩擦系数，以及地面对斜面的静摩擦力。
分析：
先对小物块沿斜面列方程求滑动摩擦力，再对斜面受力分析求地面静摩擦。
解答步骤：
1. 沿斜面方向有 \(mg\sin30^\circ-f_k=ma\)，故 \(f_k=2(5-3)=4\text{ N}\)。
2. 法向力 \(N=mg\cos30^\circ=10\sqrt3\text{ N}\)。
3. 所以滑动摩擦系数 \(\mu_k=\dfrac{f_k}{N}=\dfrac{4}{10\sqrt3}=\dfrac{2\sqrt3}{15}\)。
4. 再对斜面受力分析，地面静摩擦平衡来自物块对斜面的水平作用分量。
5. 静摩擦力大小为 \(f_s=N\sin30^\circ-f_k\cos30^\circ=3\sqrt3\text{ N}\)。
最终答案：
\(\mu_k=\dfrac{2\sqrt3}{15}\approx 0.231\)，地面对斜面的静摩擦力大小为 \(3\sqrt3\text{ N}\)。
易错点：
地面静摩擦要对斜面整体分析，不是重复对物块再列一遍斜面方程。

### 第7题
题目：
势能曲线如图，总能量为 \(E=10\text{ J}\)。若粒子开始时位于 \(x=5\text{ m}\)，求运动范围；并判断 \(x=6\text{ m}\) 处的平衡类型。
说明：
本题读数来自图片卷和已有整理稿交叉还原，交点约读为 \(x\approx 3\text{ m}\) 与 \(x\approx 8\text{ m}\)。
分析：
允许运动区由 \(U(x)\le E\) 决定；但粒子只会在与初始位置连通的那一段允许区内运动。平衡类型看势能极值。
解答步骤：
1. 从图上读出 \(E=10\text{ J}\) 与势能曲线的交点约在 \(x\approx 1,3,8\text{ m}\)。
2. 初始位置 \(x=5\text{ m}\) 落在右侧势阱内，因此粒子不会跳到左边区间，而只能在与 \(x=5\) 连通的允许区中运动。
3. 所以实际运动范围约为 \(3\text{ m}\le x\le 8\text{ m}\)。
4. 图上 \(x=6\text{ m}\) 处对应势能极小点，因此是稳定平衡位置。
最终答案：
运动范围约为 \(3\text{ m}\le x\le 8\text{ m}\)；\(x=6\text{ m}\) 为稳定平衡位置。
易错点：
允许区可能有多个不连通区间，必须结合初始位置判断粒子实际在哪一段运动。

### 第8题
题目：
一立方体静止时边长为 \(a\)。当它以速度 \(0.6c\) 沿着某一条边的方向相对地面运动时，地面上测得它的体积是多少。
分析：
只有沿运动方向的长度发生洛伦兹收缩，垂直于运动方向的两条边长度不变。
解答步骤：
1. 洛伦兹因子 \(\gamma=\dfrac{1}{\sqrt{1-0.6^2}}=\dfrac{1}{0.8}=1.25\)。
2. 沿运动方向边长变为 \(\dfrac{a}{\gamma}=0.8a\)。
3. 另外两条边仍为 \(a\)。
4. 所以体积 \(V=a\cdot a\cdot 0.8a=0.8a^3\)。
最终答案：
地面上测得体积为 \(0.8a^3=\dfrac45 a^3\)。
易错点：
不是三条边都收缩，只有与速度平行的那一条收缩。

### 第9题
题目：
飞船以 \(0.6c\) 的速度相对地面飞行，从飞船上沿速度方向抛出一物体，物体相对飞船的速度为 \(0.8c\)。求地面上测得该物体的速度。
分析：
使用一维相对论速度合成公式，不能直接做经典加法。
解答步骤：
1. 设飞船相对地面的速度为 \(u=0.6c\)，物体相对飞船速度为 \(u'=0.8c\)。
2. 地面测得速度为 \(v=\dfrac{u+u'}{1+\frac{uu'}{c^2}}\)。
3. 代入得 \(v=\dfrac{0.6c+0.8c}{1+0.48}=\dfrac{1.4}{1.48}c=\dfrac{35}{37}c\)。
最终答案：
地面上测得物体速度为 \(\dfrac{35}{37}c\approx 0.946c\)。
易错点：
高速问题不能直接把 \(0.6c\) 与 \(0.8c\) 做普通加法。

### 第10题
题目：
屋顶悬挂一定滑轮，A、B 两人从静止开始分别抓住绳两端向上爬。A 比 B 用力大。问：忽略滑轮质量及摩擦时谁先到屋顶；若不忽略，又如何。
分析：
理想滑轮时两侧张力相同；若滑轮有转动惯量或摩擦，两侧张力可不同。
解答步骤：
1. 忽略滑轮质量与摩擦时，绳上张力处处相同，设为 \(T\)。
2. 两人质量相同，则都满足 \(T-mg=ma\)，故二人的加速度相同。
3. 由于初始高度和初速度也相同，所以二人同时到屋顶。
4. 若不忽略滑轮质量或摩擦，滑轮转动需要两侧张力不相等。
5. 若 A 用力更大，通常表现为 A 侧张力更大，于是 \(a_A>a_B\)，所以 A 先到屋顶。
最终答案：
理想滑轮时 A、B 同时到达；若不忽略滑轮质量或摩擦，则 A 先到。
易错点：
在理想模型里，“谁更使劲”不会直接变成“谁受更大的绳拉力”。

### 第11题
题目：
半径为 \(R\) 的轮子沿 \(x\) 轴以角速度 \(\omega\) 作纯滚动。轮缘上一点 \(A\) 在 \(t=0\) 时与原点重合。求该点的坐标、速度、加速度与时间关系，并求轨迹最高点处的曲率半径。
分析：
轮缘点轨迹是标准摆线，令参数 \(\varphi=\omega t\) 即可。
解答步骤：
1. 摆线参数方程为 \(x=R(\varphi-\sin\varphi)\)，\(y=R(1-\cos\varphi)\)，其中 \(\varphi=\omega t\)。
2. 因而 \(x(t)=R(\omega t-\sin\omega t)\)，\(y(t)=R(1-\cos\omega t)\)。
3. 速度分量为 \(v_x=R\omega(1-\cos\omega t)\)，\(v_y=R\omega\sin\omega t\)。
4. 加速度分量为 \(a_x=R\omega^2\sin\omega t\)，\(a_y=R\omega^2\cos\omega t\)。
5. 最高点对应 \(\varphi=\pi\)，用参数曲线曲率半径公式可求得 \(\rho=4R\)。
最终答案：
\(x=R(\omega t-\sin\omega t)\)，\(y=R(1-\cos\omega t)\)；\(v_x=R\omega(1-\cos\omega t)\)，\(v_y=R\omega\sin\omega t\)；\(a_x=R\omega^2\sin\omega t\)，\(a_y=R\omega^2\cos\omega t\)；最高点曲率半径为 \(4R\)。
易错点：
摆线最高点不是速度为 0 的点；速度为 0 的是尖点。

### 第12题
题目：
飞船从北极沿地球表面切向飞出，初速度为 \(v_0\)，之后在万有引力作用下运动。已知轨道与 \(y\) 轴再次相交于 \(C\)，且 \(OC=3R\)。求飞船在 \(C\) 点的速度大小和它与初速度的夹角。
分析：
万有引力是中心力，所以机械能守恒和角动量守恒都可用。
解答步骤：
1. 初态半径为 \(R\)，速度与半径垂直，所以角动量 \(L=mRv_0\)。
2. 在 \(C\) 点半径为 \(3R\)，设速度大小为 \(v\)，与初速度夹角为 \(\theta\)，则 \(L=m(3R)v\sin\theta\)。
3. 由角动量守恒得 \(\sin\theta=\dfrac{v_0}{3v}\)。
4. 再由机械能守恒：\(\dfrac12 mv_0^2-\dfrac{GMm}{R}=\dfrac12 mv^2-\dfrac{GMm}{3R}\)。
5. 解得 \(v=\sqrt{v_0^2-\dfrac{4GM}{3R}}\)。
最终答案：
\(v=\sqrt{v_0^2-\dfrac{4GM}{3R}}\)，且 \(\sin\theta=\dfrac{v_0}{3v}\)。
易错点：
只用能量守恒只能求速度大小，求方向还必须结合角动量守恒。

### 第13题
题目：
圆柱体从桌边开始滚下，接触过程中静摩擦系数为 \(\mu=0.25\)。求未发生滑动时的支持力与转角关系，以及发生滑动的临界角。
说明：
按图可把圆柱视为绕桌边接触点 \(P\) 无滑动转动。
分析：
先用机械能守恒求 \(\dot\theta\)，再列径向方程求支持力，最后用 \(f=\mu N\) 求临界角。
解答步骤：
1. 圆柱对接触点的转动惯量为 \(I_P=I_C+mR^2=\dfrac12 mR^2+mR^2=\dfrac32 mR^2\)。
2. 由能量守恒有 \(mgR(1-\cos\theta)=\dfrac12 I_P\dot\theta^2\)，故 \(\dot\theta^2=\dfrac{4g}{3R}(1-\cos\theta)\)。
3. 径向列式可得 \(N-mg\cos\theta=-mR\dot\theta^2\)。
4. 代入上式得 \(N=\dfrac{mg}{3}(7\cos\theta-4)\)。
5. 切向方程结合转动方程可得所需静摩擦力 \(f=\dfrac13 mg\sin\theta\)。
6. 临界滑动条件是 \(f=\mu N\)，代入 \(\mu=0.25\) 后化简为 \(4\sin\theta=7\cos\theta-4\)。
7. 解得临界角 \(\theta_c=2\arctan\dfrac{3}{11}\approx 30.5^\circ\)。
最终答案：
未滑动时支持力 \(N(\theta)=\dfrac{mg}{3}(7\cos\theta-4)\)，临界角约为 \(30.5^\circ\)。
易错点：
临界条件不是 \(N=0\)，而是“维持无滑动所需的静摩擦力达到最大值”。

### 第14题
题目：
质量 \(2m\)、长度 \(l\) 的细杆静止放在光滑水平面上。质量 \(m\) 的小球以速度 \(v_0\) 垂直射向距杆端 \(l/4\) 处，与杆发生弹性碰撞。求碰后小球速度、杆的角速度和杆质心速度。
分析：
碰撞时间极短，外界冲量可忽略，因此沿碰撞方向线动量守恒；对杆质心角动量守恒；又因为是弹性碰撞，机械能守恒。
解答步骤：
1. 设碰后小球速度为 \(u\)，杆质心速度为 \(V\)，角速度为 \(\omega\)。
2. 线动量守恒：\(mv_0=mu+2mV\)，即 \(v_0=u+2V\)。
3. 对杆质心取角动量守恒：\(mv_0\dfrac l4=mu\dfrac l4+I_C\omega\)，其中 \(I_C=\dfrac1{12}(2m)l^2=\dfrac16 ml^2\)。
4. 化简得 \(\omega=\dfrac{3}{2l}(v_0-u)\)。
5. 弹性碰撞满足机械能守恒：\(\dfrac12 mv_0^2=\dfrac12 mu^2+\dfrac12 (2m)V^2+\dfrac12 I_C\omega^2\)。
6. 联立求解得 \(u=-\dfrac{v_0}{15}\)，\(V=\dfrac{8v_0}{15}\)，\(\omega=\dfrac{8v_0}{5l}\)。
最终答案：
小球反向弹回，速度为 \(-\dfrac{v_0}{15}\)；杆质心速度为 \(\dfrac{8v_0}{15}\)；角速度为 \(\dfrac{8v_0}{5l}\)。
易错点：
杆既有平动又有转动，不能只列线动量守恒和能量守恒而漏掉角动量守恒。

### 第15题
题目：
快速运动介子的总能量为 \(E=3000\text{ MeV}\)，静止能量为 \(E_0=100\text{ MeV}\)，固有寿命为 \(\tau=2\times 10^{-6}\text{ s}\)。求它在实验室系中的飞行距离。
分析：
先由 \(E=\gamma E_0\) 求出洛伦兹因子，再用时间延缓求实验室寿命，最后乘速度得到距离。
解答步骤：
1. 洛伦兹因子 \(\gamma=\dfrac{E}{E_0}=\dfrac{3000}{100}=30\)。
2. 所以 \(\beta=\sqrt{1-\dfrac1{\gamma^2}}=\sqrt{1-\dfrac1{900}}=\sqrt{\dfrac{899}{900}}\)。
3. 实验室系寿命为 \(t=\gamma\tau=30\times 2\times 10^{-6}=6\times 10^{-5}\text{ s}\)。
4. 飞行距离 \(s=vt=\beta ct=\beta\gamma c\tau\)。
5. 代入数值可得 \(s\approx 3\times 10^8\times 2\times 10^{-6}\times \sqrt{899}\approx 1.8\times 10^4\text{ m}\)。
最终答案：
实验室系中的飞行距离约为 \(1.8\times 10^4\text{ m}\)。
易错点：
固有寿命不是实验室里直接测到的寿命，要先乘时间延缓因子 \(\gamma\)。
