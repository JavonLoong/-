# 大学物理（2）期末考前速记公式清单

这份清单只服务于最后冲刺：不展开解释，只保留最该背熟的关系式和判断关键词。

## 1. 光学

### 1.1 干涉

```text
Delta phi = 2 pi delta / lambda
亮纹：delta = k lambda
暗纹：delta = (k + 1/2) lambda
双缝小角：delta ≈ d x / D
条纹间距：Delta x = lambda D / d
插片附加光程：(n - 1) t
图样平移量：Delta x_shift = D (n - 1) t / d
迈克耳孙：2 Delta d = N lambda
```

关键词：

- 双缝、相邻亮纹、中央亮纹、插片、迈克耳孙、光源宽度

### 1.2 薄膜干涉

```text
先判断两束反射光是否相差一次半波损失
若相差 0 次或 2 次：
反射亮：2 n t = k lambda
反射暗：2 n t = (k + 1/2) lambda

若相差 1 次：
反射亮：2 n t = (k + 1/2) lambda
反射暗：2 n t = k lambda
```

关键词：

- 增透膜、反射最弱、反射最强、等厚、等倾

### 1.3 衍射与光栅

```text
单缝暗纹：a sin(theta) = m lambda
中央明纹宽：2 f lambda / a
光栅主极大：d sin(theta) = k lambda
缺级：k = m d / a
分辨本领：R = lambda / Delta lambda = kN
圆孔分辨极限：theta_min ≈ 1.22 lambda / D
```

关键词：

- 单缝、中央明纹、光栅、缺级、分辨、望远镜口径

### 1.4 偏振

```text
自然光过偏振片：I = I0 / 2
马吕斯定律：I = Iin cos^2(theta)
多片偏振片：逐片连乘 cos^2
布儒斯特角：tan(iB) = n2 / n1
iB + r = 90°
波片相位延迟：Delta phi = 2 pi Delta n t / lambda
四分之一波片：Delta n t = lambda / 4
二分之一波片：Delta n t = lambda / 2
```

快判断：

- `Delta phi = 0` 或 `pi`：线偏振
- `Ax = Ay` 且 `Delta phi = ±pi/2`：圆偏振
- 其他一般是椭圆偏振

## 2. 量子物理

### 2.1 波粒二象性

```text
光子能量：E = h nu = h c / lambda
维恩位移：lambda_m T = b
斯特藩-玻尔兹曼：M = sigma T^4
光电效应：h nu = W + Kmax = W + e Us
红限频率：nu0 = W / h
红限波长：lambda0 = h c / W
康普顿：Delta lambda = lambdaC (1 - cos phi)
反冲电子动能：K = h c (1/lambda - 1/lambda')
德布罗意：lambda = h / p
电子经电压 U 加速：lambda = h / sqrt(2 m e U)
不确定关系：Delta x Delta px >= hbar / 2
布拉格公式：2 d sin(phi) = k lambda
```

关键词：

- 峰值波长、总辐射本领、截止电压、逸出功、散射角、电子波长、单缝宽度、掠射角

### 2.2 薛定谔方程与势阱

```text
psi_n(x) = sqrt(2/a) sin(n pi x / a)
E_n = n^2 h^2 / (8 m a^2)
Psi(x,t) = sum c_n psi_n(x) exp(-i E_n t / hbar)
P(E_n) = |c_n|^2
Eavg = sum |c_n|^2 E_n
P(x1 ~ x2) = integral |Psi|^2 dx
Phi_p(x) = 1 / sqrt(2 pi hbar) · exp(i p x / hbar)
```

快判断：

- 看见叠加态，先找 `c_n`
- 概率是 `|c_n|^2`
- 平均能量是加权平均，不是再解方程

### 2.3 氢原子与量子数

```text
E_n = -13.6 / n^2   (eV)
1 / lambda = R (1 / nl^2 - 1 / nu^2)
最多谱线数：Nmax = n (n - 1) / 2
n = 1, 2, 3, ...
l = 0, 1, ..., n - 1
m_l = -l, ..., +l
m_s = ±1/2
L = sqrt[l(l + 1)] hbar
L_z = m_l hbar
每层最大电子数：2n^2
每个亚层最大电子数：2(2l + 1)
```

快判断：

- 可见光优先看跃迁到 `n = 2`
- 用 `|m_l| <= l <= n - 1` 反推 `l`
- “角动量大小”是 `L`，不是 `L_z`

## 3. 最后五分钟只提醒自己这些

1. 双缝、单缝、光栅三套条件一定分开。
2. 迈克耳孙题别漏 `2`。
3. 自然光先减半，再马吕斯。
4. 黑体题先求温度比，再四次方。
5. 光电效应的 `eUs` 是最大动能。
6. 康普顿比值题里 `lambdaC` 会约掉。
7. 势阱题的概率来自系数模方。
8. 氢原子可见光常看 `n -> 2`。
