第 22 章 光的干涉

双缝、薄膜、空间相干性、迈克耳孙干涉仪，本质都围绕“光程差”。

物理图像

  • 干涉题先别急着套公式，先找出哪两束光在叠加。
  • 一旦写出光程差 \(\delta\)，大多数题已经完成了一半。
  • 亮纹 / 暗纹的判断只看相位差，也就是光程差。

必背公式

相位差 \( \Delta \phi = \dfrac{2\pi \delta}{\lambda} \)
亮纹 \( \delta = k\lambda \)
暗纹 \( \delta = \left(k + \dfrac{1}{2}\right)\lambda \)
双缝近似 \( \delta \approx \dfrac{dx}{D} \)
条纹间距 \( \Delta x = \dfrac{\lambda D}{d} \)
插片附加光程 \( \Delta = (n - 1)t \)
图样平移 \( \Delta x_{\text{shift}} = \dfrac{D(n-1)t}{d} \)
迈克耳孙 \( 2\Delta d = N\lambda \)
一条规则就够： 干涉图样整体向插片所在那一路平移。
高频题型
  1. 双缝第 \(k\) 级亮纹 / 暗纹位置：先写 \(\delta \approx dx / D\)，再联立亮暗条件。
  2. 插片后条纹平移：先算附加光程 \((n - 1)t\)，再换成屏上位移。
  3. 空间相干性：条纹刚好消失时，记 \(b_{\max} \approx \lambda L / d\)。
  4. 迈克耳孙干涉仪：镜面移动 \(\Delta d\) 引起光程差变化 \(2 \Delta d\)，最容易漏这个系数。
  5. 薄膜干涉：先数相位反转次数，再决定 \(2nt\) 对应亮纹还是暗纹条件。
薄膜干涉的相位反转判断表
两束反射光半波损失差 反射亮纹条件 反射暗纹条件
相差 0 次或 2 次 \(2nt = k \lambda\) \(2nt = (k + 1/2) \lambda\)
相差 1 次 \(2nt = (k + 1/2) \lambda\) \(2nt = k \lambda\)
最容易错的点
  • 把相位差和光程差混掉，没有先统一到 \(\delta\)。
  • 忘记双缝小角近似 \(\delta \approx dx / D\)。
  • 插片题只改某一条纹位置，忘了本质是整套图样平移。
  • 迈克耳孙干涉仪漏掉光程差中的 \(2\)。
  • 薄膜干涉没有先判断反射时是否发生相位反转。

第 23 章 光的衍射

单缝看包络，光栅看“单缝包络 + 多缝干涉主极大”。

物理图像

  • 单缝衍射：缝内部各部分互相干涉。
  • 光栅衍射：主极大位置由 \(d \sin(\theta) = k \lambda\) 决定。
  • 缺级：主极大刚好落在单缝暗纹位置上，被包络压没。

必背公式

单缝暗纹 \( a\sin\theta = m\lambda \)
中央明纹宽 \( \Delta x_{\text{center}} = \dfrac{2f\lambda}{a} \)
光栅主极大 \( d\sin\theta = k\lambda \)
分辨本领 \( R = \dfrac{\lambda}{\Delta \lambda} = kN \)
缺级条件 \( a\sin\theta = m\lambda,\quad d\sin\theta = k\lambda,\quad k = \dfrac{md}{a} \)
分辨极限 \( \theta_{\min} \approx \dfrac{1.22\lambda}{D} \)
遇到“刚好分辨开两条谱线”： 立刻想到 \(R = kN\)。
高频题型
  1. 单缝中央明纹宽度：只要给了 \(a\)、\(f\)、\(\lambda\)，直接 \(2f \lambda / a\)。
  2. 光栅主极大和级次：先用 \(d \sin(\theta) = k \lambda\)，再判断 \(|\sin(\theta)| \le 1\)。
  3. 缺级：一定联立单缝暗纹条件，不能只写光栅主极大条件。
  4. 分辨本领：级次越高、总缝数越多，越容易分辨。
  5. 望远镜 / 显微镜分辨：看到口径和最小角距，想到 \(1.22 \lambda / D\)。
典型题骨架：主极大 + 缺级
已知：在 \(\theta = 30^\circ\) 处出现 \(2\) 级主极大
\(d \sin 30^\circ = 2\lambda\)
\(d = 4\lambda\)

若 \(3\) 级主极大缺级：
\(3 = md / a\)

取最小 \(m = 1\)
\(a = d / 3 = 4\lambda / 3\)

再看可见级次：
\(\left| k\lambda / d \right| \le 1\)
最容易错的点
  • 把单缝暗纹条件和光栅主极大条件写成同一个式子。
  • 缺级题只写 \(d \sin(\theta) = k \lambda\)，忘了联立单缝条件。
  • 只会算主极大，不会结合角域限制做级次筛选。
  • 分辨极限题忘了 \(1.22\)。

第 24 章 光的偏振

偏振片筛方向，波片改相位差，这是整章最重要的两句话。

物理图像

  • 自然光：振动方向杂乱；线偏振光：方向固定。
  • 偏振片是在筛掉不合方向的振动分量。
  • 波片主要改变相位差，不是简单改光强。

必背公式

自然光过偏振片 \( I = \dfrac{I_0}{2} \)
马吕斯定律 \( I = I_{\text{in}}\cos^2\theta \)
布儒斯特角 \( \tan i_B = \dfrac{n_2}{n_1} \)
反射偏振关系 \( i_B + r = 90^\circ \)
相位延迟 \( \Delta \phi = \dfrac{2\pi\,\Delta n\,t}{\lambda} \)
四分之一波片 \( \Delta n\,t = \dfrac{\lambda}{4} \)
二分之一波片 \( \Delta n\,t = \dfrac{\lambda}{2} \)
偏振类型快判断：
\(\Delta \phi = 0\) 或 \(\pi\) 是线偏振；\(A_x = A_y\) 且 \(\Delta \phi = \pm \pi/2\) 是圆偏振；其余一般是椭圆偏振。
高频题型
  1. 多片偏振片：自然光先减半，之后逐片乘 \(\cos^2(\theta)\)。
  2. 反射偏振：看到“反射光完全偏振”，立刻想到布儒斯特角。
  3. 波片题：先沿快慢轴分解，再看相位差是否变成 \(\pi/2\) 或 \(\pi\)。
  4. 两块四分之一波片同轴串联：等效于一块半波片。
三片偏振片题的通用写法
\(I_1 = I_0 / 2\)
\(I_2 = I_1 \cos^2 \theta_{12}\)
\(I_3 = I_2 \cos^2 \theta_{23}\)

别跳步骤。闭卷里最容易丢分的不是不会，而是第一步没先把自然光除以 2。

最容易错的点
  • 自然光经第一片偏振片后忘了先减半。
  • 布儒斯特角题把入射角、反射角、折射角混掉。
  • 波片题只背结论，不先拆成快慢轴两分量。
  • 判断偏振类型时只看相位差，不看振幅关系。

第 26 章 波粒二象性

本章大多数题都能收束到几类：黑体、光电效应、康普顿、德布罗意、不确定关系、X 射线衍射。

物理图像

  • 光既有波动性，也有粒子性。
  • 电子等微观粒子也有波动性，所以才会有电子衍射和德布罗意波长。
  • 很多填空题看着花，其实只是换壳考同一条公式。

必背公式

光子能量 \( E = h\nu = \dfrac{hc}{\lambda} \)
维恩位移 \( \lambda_m T = b \)
斯特藩-玻尔兹曼 \( M = \sigma T^4 \)
光电效应 \( h\nu = W + K_{\max} = W + eU_s \)
红限频率 \( \nu_0 = \dfrac{W}{h} \)
红限波长 \( \lambda_0 = \dfrac{hc}{W} \)
康普顿散射 \( \Delta \lambda = \lambda_C(1-\cos\phi) \)
反冲电子动能 \( K = hc\left(\dfrac{1}{\lambda} - \dfrac{1}{\lambda'}\right) \)
德布罗意 \( \lambda = \dfrac{h}{p} \)
加速电子波长 \( \lambda = \dfrac{h}{\sqrt{2meU}} \)
不确定关系 \( \Delta x\,\Delta p_x \ge \dfrac{\hbar}{2} \)
布拉格公式 \( 2d\sin\phi = k\lambda \)
高频题型
  1. 黑体辐射：先用 \(\lambda_m T = b\) 算温度比，再用 \(T^4\) 算总辐射本领比。
  2. 光电效应：看到截止电压、逸出功、红限频率，直接写 \(h \nu = W + eU_s\)。
  3. 康普顿散射：如果问两个散射角下波长改变量比值，\(\lambda_C\) 会约掉。
  4. 德布罗意：电子经电压加速时常用 \(\lambda = h / \sqrt{2meU}\)。
  5. 不确定关系：单缝越窄，横向动量不确定度越大。
  6. X 射线衍射：先看题里给的是掠射角还是与法线夹角，别直接上手。
典型题骨架：光电效应
第 1 步：光子能量
\(E = hc / \lambda\)

意思：一个光子的能量由波长决定。\(h\) 是普朗克常量，\(c\) 是光速，\(\lambda\) 是入射光波长。

怎么用：题目给波长时，先用这个把“光的波长”换成“每个光子的能量”。波长越短，光子能量越大。

第 2 步：能量分账
\(E = W + eU_s\)

意思：光子能量分成两份：一份用来把电子从金属里“抠出来”，叫逸出功 \(W\)；剩下的变成光电子最大动能。

为什么是 \(eU_s\)：截止电压刚好能把最快的光电子拦住，所以最大动能 \(K_{\max} = eU_s\)。

易错点：\(eU_s\) 不是逸出功，它是最大动能。

求逸出功
\(W = hc / \lambda - eU_s\)

意思：总光子能量减去最大动能，剩下就是金属的逸出功。

怎么用：题目给入射光波长 \(\lambda\) 和截止电压 \(U_s\)，通常就是让你求 \(W\)。

红限频率
\(\nu_0 = W / h\)

意思：刚好能发生光电效应的最低频率。这个时候电子刚好被打出来，最大动能为 0。

怎么判断：如果入射光频率 \(\nu < \nu_0\)，不管光强多大，都不会发生光电效应。

红限波长
\(\lambda_0 = hc / W\)

意思：刚好能发生光电效应的最大波长，也叫截止波长。

怎么判断：如果入射光波长 \(\lambda > \lambda_0\)，光子能量不够，电子出不来。

最容易错的点
  • 黑体题把峰值波长与温度关系写反。
  • 光电效应里把 \(eU_s\) 当成逸出功，而不是最大动能。
  • 康普顿题不会抓 \(1 - \cos \phi\) 这个核心。
  • 德布罗意题把光子公式和电子加速公式混用。
  • 不确定关系题不会把缝宽当作 \(\Delta x\) 的量级估计。

第 27 章 薛定谔方程与无限深方势阱

重点不是求解微分方程本身，而是“本征态、叠加态、测量概率、时间演化”。

物理图像

  • 粒子被限制在 \(0 < x < a\) 的势阱里，只能取离散能级。
  • 如果初态是本征态叠加，时间演化只是在每个本征态前乘一个相位因子。
  • 测量某能量的概率就是对应系数模方。

必背公式

本征函数 \( \psi_n(x) = \sqrt{\dfrac{2}{a}}\sin\!\left(\dfrac{n\pi x}{a}\right) \)
本征能量 \( E_n = \dfrac{n^2 h^2}{8ma^2} \)
初态展开 \( \Psi(x,0) = \sum_n c_n\,\psi_n(x) \)
时间演化 \( \Psi(x,t) = \sum_n c_n\,\psi_n(x)e^{-iE_n t/\hbar} \)
能量概率 \( P(E_n) = |c_n|^2 \)
平均能量 \( E_{\text{avg}} = \sum_n |c_n|^2 E_n \)
区间概率 \( P(x_1 \sim x_2) = \int_{x_1}^{x_2} |\Psi(x,t)|^2\,dx \)
动量本征函数 \( \Phi_p(x) = \dfrac{1}{\sqrt{2\pi\hbar}}\,e^{ipx/\hbar} \)
高频题型
  1. 已知初态，写出 \(\Psi(x,t)\)：给每个本征态乘上 \(\exp(-iE_n t/\hbar)\)。
  2. 求某个能量的概率：就是对应系数模方 \(|c_n|^2\)。
  3. 求平均能量：\(\sum |c_n|^2 E_n\)，不是重新解势阱。
  4. 区间概率：对 \(|\Psi|^2\) 在给定区间积分。
  5. 动量题：把波函数拆成若干平面波，读出可能动量值与概率。
最常用的展开技巧
\(\sin u \cos u = \tfrac{1}{2} \sin 2u\)
\(\sin u (1 - \cos u) = \sin u - \tfrac{1}{2} \sin 2u\)

很多看起来复杂的初态，最终只是 \(\psi_1\) 和 \(\psi_2\) 的线性组合，别被表面形式吓住。

典型题骨架：势阱初态展开
若
\(\Psi(x,0) = \sqrt{8/(5a)} (1 - \cos(\pi x / a)) \sin(\pi x / a)\)

先化简：
\(\sin u (1 - \cos u) = \sin u - \tfrac{1}{2} \sin 2u\)

则
\(\Psi(x,0) = \dfrac{2}{\sqrt{5}} \psi_1(x) - \dfrac{1}{\sqrt{5}} \psi_2(x)\)

于是
\(P(E_1) = 4/5\)
\(P(E_2) = 1/5\)
\(\bar{E} = (4/5) E_1 + (1/5) E_2\)
最容易错的点
  • 忘了势阱定义区间是 \(0 < x < a\)。
  • 归一化系数写错，导致后面概率全歪。
  • 把 \(c_n\) 和 \(|c_n|^2\) 混掉。
  • 区间概率直接对 \(\Psi\) 积分，而不是对 \(|\Psi|^2\) 积分。
  • 把“可能动量值”和“平均动量”混成同一个概念。

第 28 章 原子中的电子

氢原子、量子数、角动量量子化、谱线条数，是这一章几乎固定出现的几类题。

物理图像

  • 氢原子最核心的结构是离散能级，不是连续取值。
  • 光谱线来自能级跃迁，能量差越大，光子能量越高，波长越短。
  • 量子数是描述电子量子态的标签，不是随便给的四个数字。

必背公式

氢原子能级 \( E_n = -\dfrac{13.6}{n^2}\,\mathrm{eV} \)
里德伯公式 \( \dfrac{1}{\lambda} = R\!\left(\dfrac{1}{n_l^2} - \dfrac{1}{n_u^2}\right) \)
最多谱线数 \( N_{\max} = \dfrac{n(n-1)}{2} \)
主量子数 \( n = 1,2,3,\ldots \)
角量子数 \( l = 0,1,\ldots,n-1 \)
磁量子数 \( m_l = -l,\ldots,+l \)
自旋量子数 \( m_s = \pm \dfrac{1}{2} \)
角动量大小 \( L = \sqrt{l(l+1)}\,\hbar \)
z 分量 \( L_z = m_l\hbar \)
每层最多电子数 \( 2n^2 \)
每亚层最多电子数 \( 2(2l+1) \)
高频题型
  1. 从 \(n = 4\) 激发态最多能发出几条谱线：直接 \(n(n-1)/2\)。
  2. 求可见光中最短波长：通常优先考虑跃迁到 \(n = 2\) 的巴耳末系。
  3. 由 \(n, m_l\) 反推 \(l\)：用 \(|m_l| \le l \le n - 1\) 夹出唯一值。
  4. 角动量大小和分量：看到“大小”就写 \(L\)，看到 “z 分量”就写 \(L_z\)。
典型题骨架
若一群氢原子处于 \(n = 4\)：

最多谱线数：
\(N_{\max} = 4 \times 3 / 2 = 6\)

可见光中最短波长：
优先看 \(4 \to 2\)
\(\Delta E = 13.6 \left( 1/4 - 1/16 \right)\,\text{eV} = 2.55\,\text{eV}\)

若某态 \(n = 4,\, m_l = -3\)：
因为 \(|m_l| \le l \le n - 1 = 3\)
\(l = 3\)
\(m_s = \pm 1/2\)
\(L = \sqrt{3(4)}\,\hbar = 2\sqrt{3}\,\hbar\)
最容易错的点
  • 不会从激发态直接写出最多谱线条数公式。
  • 可见光题不知道要优先看跃迁到 \(n = 2\)。
  • \(m_l\) 和 \(l\) 的约束关系记反。
  • \(L\) 和 \(L_z\) 混掉，导致答案单位和数值都错。

这一章放在低优先级补充区，只有在老师明确强调时再重点投入。

最省时间的抓法： 先记住金属、绝缘体、半导体的能带差异；如果再往下考，通常才会碰到费米能、自由电子模型。
\(E = p^2 / (2m)\)

\(p_F = \hbar (3\pi^2 n)^{1/3}\)

\(E_F = p_F^2 / (2m)\)
\(= \hbar^2 (3\pi^2 n)^{2/3} / (2m)\)
  • 金属：能带部分填满，或价带和导带重叠，电子容易迁移。
  • 绝缘体：禁带宽，电子难跃迁。
  • 半导体：禁带较小，受温度和掺杂影响明显。

大题模板

考场上最重要的是“先下得去笔”。下面这些模板就是你每类题的起手式。

模板 1：双缝干涉

先写：
\(\delta \approx dx / D\)

再写：
亮纹 \(\delta = k\lambda\)
暗纹 \(\delta = (k + 1/2)\lambda\)

若插片：
附加光程 \((n - 1)t\)

模板 2：光栅

先由主极大：
\(d \sin \theta = k\lambda\)

若缺级：
\(a \sin \theta = m\lambda\)
\(\Rightarrow k = md / a\)

分辨本领：
\(R = kN\)

模板 3：偏振

自然光先过第一片：
\(I_1 = I_0 / 2\)

之后逐片乘：
\(\cos^2 \theta\)

模板 4：光电效应

\(h\nu = W + eU_s\)

先算光子能量
再求逸出功 / 截止电压 / 红限频率

模板 5：无限深势阱

先把初态拆成本征态
再乘时间因子 \(\exp(-iE_n t / \hbar)\)
概率 \(= |c_n|^2\)
平均能量 \(= \sum |c_n|^2 E_n\)

模板 6：氢原子

\(E_n = -13.6 / n^2\)
\(\Delta E = |E_i - E_f|\)
\(N_{\max} = n(n - 1) / 2\)
\(L = \sqrt{l(l + 1)}\,\hbar\)
\(L_z = m_l \hbar\)

考前速记

临进考场前最后五分钟，只需要把这些提醒重新过一遍。

最后五分钟提醒自己这些：
  1. 双缝、单缝、光栅三套条件一定分开。
  2. 迈克耳孙题别漏 \(2\)。
  3. 自然光先减半，再马吕斯。
  4. 黑体题先求温度比，再四次方。
  5. \(eU_s\) 是光电子最大动能，不是逸出功。
  6. 康普顿比值题里 \(\lambda_C\) 会约掉。
  7. 势阱题里的概率来自系数模方。
  8. 氢原子可见光通常优先看 \(n \to 2\)。
  9. 量子数题先用 \(|m_l| \le l \le n - 1\) 夹住 \(l\)。
