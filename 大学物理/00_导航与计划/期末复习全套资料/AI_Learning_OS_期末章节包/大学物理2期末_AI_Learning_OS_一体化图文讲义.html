公式血缘摘要

不是孤立公式表，而是“来源图像 → 公式 → 条件/误用”。

22 干涉 Delta phi = 2pi delta/lambda

来源：同频光矢量叠加。条件：相干、振动方向有平行分量、光程差含介质和半波损失。

23 衍射 a sinθ = kλ
d sinθ = kλ

单缝暗纹来自半波带相消；光栅主极大来自多缝同相加强。a 与 d 不能互换。

24 偏振 I = I0/2
I = Iin cos²α

自然光第一片先减半；线偏振光过检偏器用马吕斯。强度是振幅投影的平方。

26 波粒 hν = W + eUs
λ = h/p

单位/边界：hν、W、eUs 都是能量；实物粒子不能用光子的 E=hc/λ 代替总能量。

27 势阱 Hhat Phi = E Phi
P(E_n)=|c_n|²

本征方程给可能测量值；叠加态中概率来自展开系数模方。

28 原子 E_n=-13.6/n² eV
L=sqrt[l(l+1)]ℏ

单位/边界：氢原子能级用 eV；多电子原子不能直接套氢原子公式。量子数是相容力学量本征值标签。

29 固体 hν > Eg
λmax = hc/Eg

单位/边界：若 Eg 用 eV，可用 hc≈1240 eV·nm；自由电子/费米公式是否深考需核对原课件/历年题。

典型题型入口

光学三章混排

先标模型：双缝/薄膜/迈克耳孙、单缝/光栅/圆孔、偏振片/布儒斯特/波片。模型错比计算错更危险。

量子实验链

黑体看量子化，光电看单光子能量，康普顿看动量守恒，电子衍射看物质波，不确定关系看数量级结论。

势阱测量题

先找本征基底，再看初态系数；时间演化逐项乘对应相位，概率和平均值不要混。

原子谱线题

能级差取正，波长用第 26 章光子能量公式；可见光优先检查巴耳末系。

量子数合法性

按 n、l、m_l、m_s 层层筛；大小用 L，z 分量用 L_z。

固体概念题

导体/半导体/绝缘体看能带和禁带；n/p 型看多数载流子；p-n 结先判正反偏。

易错点防线

半波损失

薄膜反射先看低到高还是高到低；不判断半波损失就套明暗条件，最容易整题反写。

往返系数 2

迈克耳孙动镜和插介质都是往返光路；双缝插片通常是单程，不要混用。

a 与 d

单缝暗纹用缝宽 a；光栅主极大用光栅常数 d；缺级才联立两者。

自然光第一片

自然光过第一片偏振片先 I0/2；之后已经是线偏振光，逐片乘 cos²。

概率幅与概率

双缝概率波、势阱展开都先叠加概率幅，再取模方；c_n 不是概率，|c_n|² 才是。

能级差正负

发射/吸收光子能量写正值；第 28 章只写能级差图像，光子能量公式调用第 26 章。

错题回写接口

错题不只改答案，必须挂回“物理图像、公式血缘、题型动作、复测时间”。

1. 定位章节

先标记章节与模型：双缝/薄膜、单缝/光栅、偏振片/波片、光电/康普顿/布拉格、势阱、谱线、能带。

2. 切出错因

按概念误解、公式误用、条件漏看、单位量纲、题型识别、时间压力六类记录，不写“粗心”这种空标签。

3. 回写位置

错在图像就回写到物理图像走廊；错在公式条件就回写到公式血缘摘要；错在模型判断就回写到典型题型入口。

4. 对抗复测

生成同类型变式：换角度定义、换坐标区间、换已知未知、加入干扰条件。第 29 章只做概念/阈值复测。

记录模板 题目触发词 -> 真实考点 -> 错误步骤 -> 错因类别 -> 回写区块 -> 对抗变式 -> 下次复测日期

三遍法入口

第一遍：物理图像

  1. 画装置或能级图，不写公式。
  2. 说清相干光束、次波源、光矢量、本征态或能带。
  3. 用一句话解释现象为什么发生。

第二遍：公式血缘

  1. 从关键光程差、边界条件或能量守恒推公式。
  2. 标出符号含义、单位、适用条件。
  3. 写一个最常见误用。

第三遍：题目应用

  1. 先圈关键词并贴题型标签。
  2. 列主公式和检查点。
  3. 错题回写到图像、公式或条件漏看。
