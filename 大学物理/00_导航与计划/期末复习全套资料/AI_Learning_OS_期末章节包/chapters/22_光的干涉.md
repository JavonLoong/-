# 第 22 章 光的干涉

## 1. 章节一句话总纲

光的干涉不是先背一堆条纹公式，而是先找出“哪两束相干光在同一点相遇”，再把它们的光程差 `delta` 换成相位差 `Delta phi = 2 pi delta / lambda`，最后用明暗条件判断条纹的位置、形状、级次、移动和衬比度。

本章从相干叠加出发，依次处理双缝等分波面干涉、时间相干性、空间相干性、光程、薄膜等厚条纹、薄膜等倾条纹和迈克耳孙干涉仪。考试时的统一抓手是：

```text
确定干涉光束
-> 画出两条光路
-> 写光程差
-> 判断是否有半波损失
-> 套明暗条件
-> 解释条纹形状、级次和移动
```

## 2. 核心问题

1. 什么样的两束光能稳定干涉？
   - 光矢量有平行分量。
   - 频率相同。
   - 相位差恒定。

2. 光强为什么不是简单相加？
   - 非相干叠加时，干涉项时间平均为零。
   - 相干叠加时，干涉项保留，亮暗由 `cos Delta phi` 决定。

3. 为什么双缝条纹等间距？
   - 远屏、小角度下，几何光程差 `delta ≈ d x / D` 与屏上坐标 `x` 成正比。
   - 相邻亮纹光程差相差一个 `lambda`，所以相邻位置差恒定。

4. 为什么光源“不够单色”会让高级次条纹看不清？
   - 不同波长的条纹间距不同。
   - 级次越高，不同波长的明暗位置错开越明显，衬比度下降。

5. 为什么光源“太宽”也会让条纹消失？
   - 宽光源可看成许多互不相干的点光源。
   - 各点光源产生的条纹彼此平移，非相干叠加后互相洗掉。

6. 为什么要引入光程？
   - 光在介质中走路程 `r`，相位效果等价于在真空中走 `nr`。
   - 用光程后，经过不同介质的光路可以统一用真空波长计算相位差。

7. 薄膜干涉为什么总要先判断半波损失？
   - 反射时是否相位反转会把明暗条件整体交换。
   - 不先判断相位反转，薄膜题很容易把反射亮纹和暗纹写反。

8. 迈克耳孙干涉仪为什么总有一个系数 `2`？
   - 动镜移动 `Delta d` 时，该臂往返光程改变 `2 Delta d`。
   - 插入介质片时，若光往返通过厚度 `l`，附加光程是 `2(n - 1)l`。

## 3. 前置知识

| 前置知识 | 本章用法 | 不稳时的补救 |
|---|---|---|
| 波的叠加与相位 | 用相位差判断加强、减弱 | 先复习 `cos` 叠加和相位差 |
| 电磁波强度 | 光强正比于光矢量振幅平方的时间平均 | 只需会用结论，不要求完整电磁推导 |
| 小角近似 | `sin theta ≈ tan theta ≈ x / D` | 双缝、远屏题必须先确认小角条件 |
| 几何光路 | 画出 `r1`、`r2`、薄膜内外两束反射光 | 不会画光路就先不要套公式 |
| 折射率 | 光程 `nr`，介质中波长 `lambda/n` | 统一用真空波长 `lambda` 写相位差 |
| 反射相位反转 | 低折射率到高折射率反射有半波损失 | 薄膜题先数两束反射光的相位反转次数 |
| 三角几何 | 劈尖厚度、牛顿环半径、等倾环半径 | 会用 `e ≈ x theta`、`r^2 ≈ 2Re` |
| 级次 `k` | 明暗纹编号、相干级次、条纹移动数 | 注意不同教材对 `k` 起点不同，需核对原课件/历年题 |

## 5. 公式血缘表

| 公式名称 | 公式表达 | 来源图像 | 适用条件 | 常见误用 |
|---|---|---|---|---|
| 相干叠加强度 | `I = I1 + I2 + 2 sqrt(I1 I2) cos Delta phi` | 两个同频、相位差恒定的光矢量叠加 | 两束光相干，且取平行光矢量分量 | 把非相干光也代入干涉项 |
| 相位差与光程差 | `Delta phi = 2 pi delta / lambda` | 光程差对应相位落后或超前 | `lambda` 用真空波长；`delta` 是光程差 | 把几何路程差直接当光程差，忘了折射率 |
| 明纹条件 | `delta = k lambda` | 相位差为 `2k pi`，同相加强 | 未额外计入半波损失，或 `delta` 已包含全部相位修正 | 薄膜反射中不判断半波损失就用 |
| 暗纹条件 | `delta = (k + 1/2) lambda` | 相位差为奇数倍 `pi`，反相抵消 | 未额外计入半波损失，或 `delta` 已包含全部相位修正 | 把双缝暗纹和单缝衍射暗纹混用 |
| 条纹衬比度 | `V = (Imax - Imin) / (Imax + Imin)` | 明暗对比程度 | 判断条纹可见性 | 只看条纹位置，不看光源宽度和单色性 |
| 双缝光程差 | `delta = r2 - r1 ≈ d sin theta ≈ d x / D` | 两缝到屏上点的路程差 | `lambda << d << D`，远屏小角 | 忘了小角近似条件；把 `D` 和光源距离 `R` 混用 |
| 双缝明纹位置 | `x_k = k lambda D / d` | `d x / D = k lambda` | 双缝、远屏、小角、单色光 | 级次从 `k = 0` 起；左右对称位置有正负号 |
| 双缝暗纹位置 | `x_k = (k + 1/2) lambda D / d` | `d x / D = (k + 1/2) lambda` | 同上 | 用成单缝暗纹条件 |
| 双缝条纹间距 | `Delta x = lambda D / d` | 相邻明纹光程差相差 `lambda` | 明暗相间且小角度 | 白光高级次重叠问题需核对历年题 |
| 插片附加光程 | `Delta L = (n - 1)t` | 一路光经过厚度 `t` 的介质片 | 双缝一路单程插片 | 和迈克耳孙往返插片的 `2(n - 1)l` 混淆 |
| 双缝插片平移量 | `Delta x_shift = D (n - 1)t / d` | 附加光程使整套图样平移 | 薄片只插入一路，小角双缝 | 只移动某一条纹，忘了整套图样平移 |
| 最大相干级次 | `k_M = lambda / Delta lambda` | 不同波长的相邻级次开始重叠 | 准单色光，`lambda >> Delta lambda` | 把 `Delta lambda` 写成频率宽度 |
| 相干长度 | `delta_M = k_M lambda = lambda^2 / Delta lambda` | 能发生干涉的最大光程差 | 时间相干性估算 | 只背公式，不知道是最大光程差 |
| 相干时间 | `tau = delta_M / c` | 光通过相干长度的时间 | 真空或空气中近似 | 忽略它反映单色性 |
| 空间相干极限宽度 | `b0 = lambda R / d` | 宽光源不同点产生条纹平移并相互洗掉 | 光源到双缝距离 `R`，缝距 `d` | 和屏距 `D` 混用；总复习中常写 `L` 表示光源距离 |
| 相干间隔 | `d0 = lambda R / b` | 给定宽度光源下，仍可相干的最大缝距 | 正对光源的平面 | 忘了 `d < d0` 才能观察条纹 |
| 相干孔径角 | `theta0 = d0 / R = lambda / b` | 光源对相干孔径张角的限制 | 小角近似 | 把它当屏上条纹角宽 |
| 光程 | `Lopt = nr` 或 `sum n_i r_i` | 介质中路程 `r` 等效真空路程 `nr` | 同一频率光在介质中传播 | 用介质中波长后又再乘 `n`，重复计算 |
| 光程差相位 | `Delta phi = 2 pi Delta(Lopt) / lambda` | 不同介质光路统一比较 | `lambda` 为真空波长 | 把透镜当作必然产生附加光程差；课件强调理想成像等光程 |
| 薄膜反射一般光程差 | `delta = 2e sqrt(n^2 - n'^2 sin^2 i) + lambda/2` | 薄膜上下表面反射两束光，含一次半波损失 | 课件情形：薄膜折射率 `n`，外侧 `n'`，且反射光相差一次半波损失 | 泛化到任意折射率次序前必须重新判断半波损失 |
| 正入射薄膜光程差 | `delta ≈ 2ne + lambda/2` | 薄膜内往返一次，正入射 | 反射光相差一次半波损失 | 直接套 `2ne`，漏掉半波损失 |
| 劈尖相邻亮纹厚度差 | `Delta e = lambda / (2n)` | 相邻同类条纹光程差差一个 `lambda` | 单色平行光正入射，等厚条纹 | 忽略介质折射率 `n` |
| 劈尖条纹间距 | `L ≈ lambda / (2n theta)` | `Delta e = L theta` | 劈尖角 `theta` 很小 | 把 `theta` 当角度数值代入，不换弧度 |
| 牛顿环暗环半径 | `r_k = sqrt(k R lambda)` | 空气劈尖，`e = r^2/(2R)`，反射暗纹 | 反射光、空气膜、中央暗，`k = 0,1,2,...` | 透射光条纹与反射互补；非空气膜需改 `n` |
| 等倾光程差 | `delta = 2e sqrt(n^2 - n'^2 sin^2 i) + lambda/2` | 膜厚均匀，光程差随倾角变 | 反射光相差一次半波损失 | 和等厚条纹混淆 |
| 等倾环半径 | `R_k = f tan i_k` | 倾角相同的光在透镜焦平面会聚 | 焦平面观察 | 把等倾条纹画在薄膜表面 |
| 等倾膜厚变化 | `Delta e = lambda / (2n)` | 中心每冒出或缩进一个亮纹，膜厚改变半波光程 | 正入射中心附近 | 不区分“向外扩展”和“向内缩进”的物理原因 |
| 迈克耳孙动镜位移 | `Delta d = N lambda / 2` | 动镜移动导致往返光程改变 `2 Delta d` | 空气中，两臂干涉 | 漏掉系数 `2` |
| 迈克耳孙测折射率 | `2(n - 1)l = N lambda` | 介质片在一臂中往返通过 | 介质长度 `l` 被光两次通过 | 和双缝插片的单程附加光程混淆 |

| 公式组 | 来源定律/图像 | 推导前提 | 不适用条件 | 符号含义 | 单位检查 | 典型题 |
| --- | --- | --- | --- | --- | --- | --- |
| 相干叠加强度与条纹衬比度 | 同频光矢量叠加，强度正比振幅平方 | 两束光频率相同、振动方向可叠加、相位差稳定 | 非相干普通光源、偏振方向互相垂直时不能直接保留干涉项 | `I1,I2` 为两束光强，`Delta phi` 相位差，`V` 衬比度 | 强度单位一致；`cos Delta phi` 与 `V` 无量纲 | 已知最大/最小光强求衬比度、判断是否能见稳定条纹 |
| 相位差、光程和明暗条件 | 光程差对应相位差，整波长同相、半波长反相 | `delta` 已包含全部光程差和额外相位修正；半波损失另判或已并入 | 薄膜反射若未判断半波损失，不能直接套双缝明暗条件 | `delta` 光程差，`lambda` 真空波长，`k` 整数级次 | `delta` 与 `lambda` 都是长度；`Delta phi` 无量纲 | 给光程差判明暗、反射薄膜判半波损失 |
| 双缝位置、间距、插片平移 | 两缝到屏上点的几何路程差在小角下近似线性 | 远屏、小角、单色光，`lambda << d << D`；插片只在一路单程通过 | 大角度、近场、白光高级次重叠不能无条件套线性位置公式 | `d` 缝距，`D` 屏距，`x` 屏上坐标，`t` 薄片厚度，`n` 折射率 | `lambda D/d`、`D(n-1)t/d` 都是长度 | 求第 `k` 级条纹位置、条纹间距、插片导致整套图样平移 |
| 时间相干性与空间相干性 | 有限谱宽导致波列长度有限；有限光源宽度导致不同点光源条纹相互洗掉 | 准单色、窄谱宽、小角；光源宽度和光源到双缝距离定义清楚 | 多谱线、复杂光源或题目给频率宽度时需先换算；不能把屏距当光源距 | `Delta lambda` 谱宽，`R` 光源到双缝距离，`b` 光源宽度，`d0` 相干间隔 | `lambda^2/Delta lambda`、`lambda R/b` 都是长度；`tau` 为秒 | 最大干涉级次、相干长度、光源极限宽度 |
| 薄膜、劈尖、牛顿环、等倾 | 上下表面反射光干涉，膜内往返光程加半波损失；等厚或等倾决定条纹位置 | 折射率顺序已判断；正入射或题给角度满足公式条件；劈尖角小 | 半波损失次数变化、透射观察、非空气膜时必须重建光程差 | `e` 膜厚，`n` 膜折射率，`R` 牛顿环透镜曲率半径，`r_k` 环半径，`theta` 劈尖角 | `2ne`、`lambda`、`r_k^2/R` 都是长度；角度用弧度 | 增透膜、劈尖测厚、牛顿环半径、等倾环随膜厚变化 |
| 迈克耳孙干涉仪 | 动镜移动改变往返光程，介质片在一臂中往返通过 | 两臂相干，空气近似，动镜位移沿光路方向 | 把一臂插片当单程、或把动镜位移当光程改变本身会错 | `Delta d` 动镜位移，`N` 条纹数，`l` 介质长度 | `2Delta d=Nlambda`、`2(n-1)l=Nlambda` 两边都是长度 | 测波长、测折射率、数条纹求动镜位移 |

## 6. 典型题型表

| 题型 | 识别关键词 | 物理图像先行 | 标准动作 | 高风险点 |
|---|---|---|---|---|
| 相干条件判断 | 相干光、普通光源、激光、干涉项 | 两束光的频率、偏振方向、相位关系是否稳定 | 写出相干条件，再判断是否有稳定条纹 | 只说“频率相同”而漏掉相位差恒定 |
| 光强与衬比度 | `I1`、`I2`、最大光强、最小光强、对比度 | 两束相干光振幅叠加，不是强度直接叠加 | 用 `I = I1 + I2 + 2 sqrt(I1 I2) cos Delta phi` | `I1 != I2` 时误以为暗纹一定全黑 |
| 双缝位置与间距 | 双缝、相邻亮纹、中央明纹、第 `k` 级 | 两缝到屏上点的光程差随 `x` 线性变化 | `delta ≈ d x / D`，再套明暗条件 | 忘记左右对称和 `k = 0` 中央明纹 |
| 双缝白光条纹 | 白光、彩色条纹、级次重叠 | 不同波长有不同条纹间距 | 先说明中央白，外侧彩色；重叠计算需列不同 `lambda` 的位置条件 | 具体重叠级次需核对原课件/历年题 |
| 插入薄片平移 | 一路插片、折射率 `n`、厚度 `t`、图样移动 | 插片使一路光程变长，整套条纹平移 | 加 `Delta L = (n - 1)t`，求 `Delta x_shift` | 平移方向向插片所在一路，正负号由坐标约定 |
| 时间相干性 | 谱线宽度、相干长度、最大级次、高级次看不清 | 波列长度有限，不同波长条纹错位 | 用 `k_M`、`delta_M`、`tau` | 把时间相干性和空间相干性混在一起 |
| 空间相干性 | 光源宽度、极限宽度、相干间隔、条纹消失 | 宽光源等效为许多互不相干点光源的条纹叠加 | 用 `b0 = lambda R / d` 或 `d0 = lambda R / b` | 把光源到双缝距离 `R` 写成屏距 `D` |
| 光程差计算 | 介质、折射率、厚度、透镜、等光程 | 介质内实际路程要折算成等效真空路程 | 写 `Lopt = sum nr` 后相减 | 透镜题中误加附加光程差；理想成像光线等光程 |
| 劈尖等厚条纹 | 劈形膜、棱边、条纹间距、平晶检测 | 同一厚度对应同一光程差，所以条纹平行于棱边 | `Delta e = lambda/(2n)`，`L = Delta e / theta` | 棱边反射常为暗纹，需先确认半波损失 |
| 牛顿环 | 平凸透镜、平晶、空气膜、暗环半径 | 圆形等厚空气膜，厚度随半径增大 | `e = r^2/(2R)`，反射暗环 `r_k = sqrt(kRlambda)` | 中央暗/亮取决于反射或透射及半波损失 |
| 等倾条纹 | 均匀薄膜、焦平面、同心圆环、倾角 | 膜厚固定，光程差只随倾角变 | 用等倾光程差和 `R_k = f tan i_k` | 和牛顿环同心圆混淆；等倾定域在无穷远 |
| 增透膜/增反膜 | 反射最弱、反射最强、膜厚设计 | 让反射光相消或相长 | 先判断两束反射光半波损失差，再选反射暗/亮条件 | 课件只作应用引入，具体设计公式需核对原课件/历年题 |
| 迈克耳孙动镜 | 动镜、条纹吞吐、移动 `N` 条、波长测量 | 一臂来回走，镜面移动一次影响两倍光程 | `2 Delta d = N lambda` | 漏掉 `2` 是本章高频失分点 |
| 迈克耳孙插介质 | 测折射率、气流、介质片长度 `l`、条纹移动 | 介质片在一臂被往返穿过 | `2(n - 1)l = N lambda` | 和双缝单程插片公式混用 |

## 7. 易错点清单

1. 把“几何路程差”和“光程差”混为一谈。  
   避坑句：经过介质时先写 `sum nr`，再谈相位差。

2. 只记明暗公式，不先判断相干条件。  
   避坑句：没有稳定相位差，就没有稳定干涉条纹。

3. 双缝题忘记小角近似的来源。  
   避坑句：先写 `delta = d sin theta`，再在远屏下写 `≈ d x / D`。

4. 把双缝干涉暗纹和单缝衍射暗纹混用。  
   避坑句：双缝暗纹是两束光反相，单缝暗纹是缝内多点配对抵消。

5. 插片题只移动某一条纹，忘记是整套图样平移。  
   避坑句：插片改变一路光程，所有级次一起挪。

6. 时间相干性和空间相干性分不清。  
   避坑句：谱线宽度管“波列有多长”；光源宽度管“不同光源点条纹会不会洗掉”。

7. 空间相干公式中把 `R`、`D` 混用。  
   避坑句：`R` 是光源到双缝距离，`D` 是双缝到屏距离。

8. 薄膜题不数半波损失。  
   避坑句：先看每次反射是低到高还是高到低，再决定明暗条件是否交换。

9. 反射光和透射光条纹互补关系忘记。  
   避坑句：同一薄膜下，反射相消通常对应透射相长，具体条件仍需看相位。

10. 牛顿环中心条纹判断反了。  
    避坑句：空气膜反射观察通常中央暗；若题目改成透射或介质膜，必须重判。

11. 等厚和等倾只看条纹形状，不看成因。  
    避坑句：等厚是厚度变，条纹定域在膜表面附近；等倾是倾角变，条纹定域在无穷远或焦平面。

12. 等倾条纹级次分布写反。  
    避坑句：课件给出等倾条纹内高外低，环半径小的级次高。

13. 迈克耳孙动镜漏掉往返系数 `2`。  
    避坑句：镜子动多少，光程差变两倍。

14. 迈克耳孙插介质和双缝插片混淆。  
    避坑句：双缝插片通常单程 `(n - 1)t`；迈克耳孙一臂插片通常往返 `2(n - 1)l`。

15. 看到“增透膜”就直接背 `lambda/4`。  
    避坑句：`lambda/4` 是常见特殊折射率排列下的最小厚度结论，考试若改变介质次序必须重新判断；需核对原课件/历年题。

## 8. 最低掌握标准

达到最低掌握，意味着本章基础题不应空题：

1. 能用自然语言说清：干涉条纹来自两束相干光的相位差，而相位差由光程差决定。
2. 能背出并解释相干条件：光矢量有平行分量、频率相同、相位差恒定。
3. 能写出相干光强公式，并知道 `I1 = I2` 时明暗对比最好。
4. 能在双缝题中从 `delta ≈ d x / D` 推出亮纹、暗纹位置和条纹间距。
5. 能处理双缝一路插入薄片导致的附加光程和平移量。
6. 能用 `k_M = lambda / Delta lambda` 和 `delta_M = lambda^2 / Delta lambda` 做时间相干性基础计算。
7. 能用 `b0 = lambda R / d` 或 `d0 = lambda R / b` 判断空间相干性。
8. 能用光程 `sum nr` 计算含介质的光程差。
9. 能在薄膜反射题中先判断半波损失，再写明暗条件。
10. 能做劈尖条纹间距、牛顿环暗环半径、迈克耳孙动镜位移三类标准题。
11. 能说清等厚条纹和等倾条纹的区别。
12. 能在计算题中至少写出“物理图像、关键光程差、明暗条件、结果单位”四层步骤。

## 9. 高分掌握标准

达到高分掌握，意味着遇到变式题也能稳住：

1. 能从陌生光路图中主动找出参与干涉的两束光，而不是套题型标签。
2. 能根据折射率次序重新判断半波损失，不依赖死记薄膜明暗表。
3. 能解释条纹形状、条纹间距、级次分布和移动方向，而不只是算一个数。
4. 能区分三种“看不清条纹”的原因：振幅不等、非单色性、光源宽度。
5. 能把双缝、薄膜、迈克耳孙统一到“光程差改变一个 `lambda`，条纹移动一条”的图像。
6. 能处理白光、多波长、条纹重叠的判断题；具体命题频率需核对历年题。
7. 能处理反射观察和透射观察互补的薄膜变式。
8. 能说明等倾条纹为什么可以用面光源照明而不严重降低衬比度。
9. 能用迈克耳孙干涉仪同时解释等厚条纹、等倾条纹、测微小位移、测折射率。
10. 能在考试中对不会完整计算的题写出正确模型和主公式，拿到步骤分。

## 10. 三遍法学习入口

| 模块 | 第一遍：物理图像 | 第二遍：公式推导 | 第三遍：题目应用 |
|---|---|---|---|
| 相干叠加 | 两束光像两根同频“指针”，相位差稳定才有固定明暗 | 从 `E1 + E2` 推到 `I = I1 + I2 + 2 sqrt(I1 I2) cos Delta phi` | 给 `I1`、`I2`、相位差，求最大最小光强和衬比度 |
| 双缝干涉 | 两缝到屏上点路程差越大，级次越高 | 从 `delta = d sin theta` 推到 `x_k` 和 `Delta x` | 求第 `k` 级亮/暗纹位置、相邻条纹间距、插片平移 |
| 时间相干性 | 波列有限长，高级次处不同波长条纹错开 | 从两端波长条纹重叠条件推 `k_M = lambda / Delta lambda` | 求最大相干级次、相干长度、条纹是否可见 |
| 空间相干性 | 宽光源许多点光源条纹错开后叠加洗掉 | 从极限宽度几何关系推 `b0 = lambda R / d` | 给光源宽度和缝距，判断是否还能观察干涉 |
| 光程 | 介质中走一段路等效于真空中走 `n` 倍路 | 从介质中波长 `lambda/n` 推 `nr` | 含介质片、透镜等光程、光程差计算 |
| 等厚薄膜 | 厚度相同的点光程差相同，所以形成等厚条纹 | 从薄膜反射两束光推 `delta ≈ 2ne + lambda/2` | 劈尖间距、牛顿环暗环半径、表面平整度判断 |
| 等倾薄膜 | 膜厚固定，倾角相同的光束同相干涉 | 用一般光程差说明 `delta` 随 `i` 变 | 同心圆环、中心冒出/缩进条纹、膜厚变化 |
| 迈克耳孙 | 两臂光程差由动镜、介质、虚镜夹角控制 | 推 `2 Delta d = N lambda` 和 `2(n - 1)l = N lambda` | 动镜位移、波长测量、折射率测量 |

建议的学习口令：

```text
第一遍：不用公式，解释本题是哪两束光在干涉，以及条纹为什么会亮暗。
第二遍：从光程差开始推到最终公式，每一步说明适用条件。
第三遍：给一道同型题，先识别题型，再列已知量、未知量、主公式和检查方法。
```

本章最值得反复练的三条推导线：

```text
双缝：delta = d sin theta ≈ d x / D -> x_k -> Delta x
薄膜：反射相位反转 -> delta -> 明暗条件 -> 条纹形状
迈克耳孙：光程差变化 -> 条纹数 N -> 位移或折射率
```


调度方注意：
