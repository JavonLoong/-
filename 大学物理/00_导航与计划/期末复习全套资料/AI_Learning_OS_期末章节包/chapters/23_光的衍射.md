# 第 23 章 光的衍射

## 1. 章节一句话总纲

光的衍射本质上是同一波面上大量次波源的相干叠加：先用“光能绕进几何阴影区、并重新分布”的物理图像判断模型，再用单缝、光栅、圆孔、晶面这几套条件把条纹位置、宽度、缺级、分辨率和晶体间距算出来。

本章的考试入口不是“见到 `lambda` 就套公式”，而是先问：

- 是一条缝内部各部分互相干涉，还是多条缝之间互相干涉？
- 是在算条纹位置，还是条纹宽度？
- 是普通光学孔径限制分辨率，还是光栅光谱分辨率？
- 是可见光衍射，还是 X 射线在晶体晶面上的布拉格衍射？

## 2. 核心问题

1. 衍射为什么会出现：光不是只沿几何直线传播，波前上每一点都可看作次波源，子波相干叠加后会进入几何阴影区，并造成光强重新分布。
2. 菲涅耳衍射和夫琅禾费衍射怎么分：近场衍射中光源或屏距至少一个有限；远场衍射用平行光，常由透镜把远场图样搬到焦平面。
3. 单缝暗纹为什么满足 `a sin(theta) = k lambda`：缝宽方向上总光程差刚好可分成偶数个半波带，相邻半波带两两相消。
4. 中央明纹为什么最宽最亮：`theta = 0` 时各次波同相，中央明纹宽度约为其他次级明纹的 2 倍。
5. 光栅为什么比单缝条纹更细锐：`N` 条缝的多光束干涉把能量集中到主极大，`N` 越大主极大越窄越亮。
6. 光栅缺级为什么发生：光栅主极大位置恰好落在单缝暗纹包络上，主极大被单缝包络压没。
7. 分辨本领为什么受衍射限制：几何光学中的“物点成像为象点”在波动光学中变成“物点成像为艾里斑”，斑太近就不可分辨。
8. X 射线为什么能测晶体：X 射线波长与晶面间距同量级，平行晶面反射波的光程差可满足 `2d sin(phi) = k lambda`。

## 7. 易错点清单

- 把衍射理解成“光被挡住后边缘漏了一点”，而不是波面次波源相干叠加后的光强重新分布。
- 把单缝暗纹 `a sin(theta)=k lambda` 和光栅主极大 `d sin(theta)=k lambda` 写成同一物理条件。
- 中央明纹宽度漏掉系数 2：`2f lambda/a` 是全宽，`f lambda/a` 是半宽。
- 单缝暗纹级次从 `k=1` 开始，不存在 `k=0` 暗纹；`theta=0` 是中央明纹中心。
- 半波带法给单缝暗纹位置准确，但给次极大中心只是近似，精确极值来自 `tan(alpha)=alpha`。
- 用 `f` 和 `L` 时不看装置：透镜焦平面用 `f`，远屏近似用屏距 `L`。
- 光栅题只判断 `|sin(theta)|<=1`，忘记题目可能限制角域。
- 缺级题只写 `d sin(theta)=k lambda`，忘了联立 `a sin(theta)=k' lambda`。
- 把 `d` 的变化和 `a` 的变化混在一起：`d` 管干涉主极大间距，`a` 管单缝包络宽度。
- N 缝暗纹数记错：相邻主极大之间有 `N-1` 个暗纹，不是 `N` 个。
- 主极大半角宽和相邻主极大间距混淆；半角宽随 `N` 增大而变小。
- 光栅角色散本领与色分辨本领混淆：前者与 `N` 无关，后者 `R=Nk` 与 `N` 有关。
- 圆孔分辨极限漏掉 `1.22`，或者把圆孔公式误用于单缝中央明纹。
- 瑞利判据的图像说不清，只会背 `1.22 lambda/D`，遇到车灯/双星距离题就不会转成 `theta≈l/L`。
- 布拉格公式中的 `phi` 是掠射角；若题目给入射线与晶面法线夹角，要先取互余。
- 第 23 章的 X 射线衍射与第 26 章波粒二象性中的 X 射线/布拉格公式有重叠，复习调度时需要合并去重。

## 8. 最低掌握标准

达到最低掌握，要求能在闭卷中稳定拿下基础题和公式直给题：

1. 能用一句话区分干涉和衍射：干涉偏“有限分立光束”，衍射偏“波面连续次波源”。
2. 能说出菲涅耳衍射和夫琅禾费衍射的差别，并知道课件计算主线是夫琅禾费衍射。
3. 能写出单缝暗纹条件 `a sin(theta)=k lambda`，并会用小角近似求暗纹位置。
4. 能写出中央明纹线宽 `Delta x0=2f lambda/a`，知道中央明纹宽度是其他明纹的 2 倍。
5. 能写出光栅主极大条件 `d sin(theta)=k lambda`，并用 `|sin(theta)|<=1` 判断最高级次。
6. 能写出缺级条件 `k=k' d/a`，至少知道缺级必须联立单缝暗纹和光栅主极大。
7. 能写出圆孔分辨极限 `theta_min≈1.22lambda/D`，并用于望远镜最小角距题。
8. 能写出光栅色分辨本领 `R=lambda/delta lambda=Nk`，会求分辨双线所需缝数。
9. 知道 X 射线晶体衍射的物理图像是晶面反射波相干加强，并能把布拉格公式交叉索引到第 26 章。
10. 能对每道题先标出模型：单缝、光栅、圆孔分辨、光栅光谱、X 射线晶体衍射接口。

最低标准自测题：

```text
1. 单缝宽 a，透镜焦距 f，波长 lambda，求中央明纹宽度。
2. 光栅常数 d，波长 lambda，问能看到最高几级主极大。
3. 已知 d/a=4，问哪些主极大缺级。
4. 望远镜口径 D，波长 lambda，求最小分辨角。
5. X 射线波长 lambda，晶面间距 d，求第 k 级布拉格掠射角。
```

## 9. 高分掌握标准

达到高分掌握，要求不只会套公式，还能处理课件中的推导、图像和变式：

1. 能用半波带法解释单缝暗纹，并说明为什么次极大中心用半波带法只是近似。
2. 能从旋转矢量和理解 `I=I0(sin alpha/alpha)^2`，知道 `alpha=pi a sin(theta)/lambda`。
3. 能解释 `a` 变大、`lambda` 变小会让衍射变弱，并把几何光学看作 `a >> lambda` 的极限。
4. 能处理 N 缝问题：相邻主极大之间 `N-1` 个暗纹、`N-2` 个弱次极大，主极大半角宽 `lambda/(Nd cos theta_k)`。
5. 能看懂光栅光强公式中“单缝衍射因子 × 多光束干涉因子”的结构，解释包络和缺级。
6. 能在缺级题中从最小正整数 `k'` 或 `m` 开始试，求“最小缝宽”“可能的 d/a”。
7. 能处理斜入射光栅方程，并按题图自行重建光程差符号。
8. 能区分光栅角色散本领 `Dtheta` 与色分辨本领 `R`：一个看角分开，一个看能否分开。
9. 能用瑞利判据解释“刚可分辨”的物理图像，而不是只背公式。
10. 能说明劳厄法和粉末法的基本目的：劳厄法常用于定晶轴方向，粉末法常用于定晶格常数。
11. 能把本章与第 22 章、第 24 章、第 26 章串起来：干涉负责相干叠加基础，偏振与衍射同属光学，布拉格公式在光学衍射和 X 射线/波粒二象性中会重复出现。

高分自测题：

```text
1. 从单缝光强公式推出暗纹条件，并说明 alpha -> 0 的极限。

3. 已知 d=4a，说明哪些光栅主极大缺级，并估计中央单缝包络内有多少条主极大。
```

## 10. 三遍法学习入口

第一遍：物理图像先行，目标是“看到图就知道是什么模型”。

| 模块 | 要复述的问题 | 合格输出 |
| --- | --- | --- |
| 衍射现象 | 为什么光会进入几何阴影区？ | 波前上每点发子波，子波叠加造成光强重新分布 |
| 单缝 | 为什么中央最亮、两侧有暗纹？ | `theta=0` 全体同相；暗纹来自半波带两两相消 |
| 光栅 | 为什么主极大细锐？ | N 缝多光束干涉把能量集中到主极大 |
| 缺级 | 为什么某级主极大没了？ | 它落在单缝暗纹位置，被包络压没 |
| 分辨 | 为什么仪器不能无限清楚？ | 点像实际是艾里斑，太近时斑重叠 |
| X 射线 | 为什么晶体能衍射 X 射线？ | 波长和晶面间距同量级，晶面反射波干涉加强 |

第二遍：公式血缘，目标是“知道公式从哪来、什么时候能用”。

```text
单缝：
delta = a sin(theta)
-> 半波带相消
-> a sin(theta) = k lambda
-> 第一暗纹给中央明纹宽 Delta x0 = 2f lambda/a

光栅：
相邻缝光程差 delta = d sin(theta)
-> 主极大 d sin(theta)=k lambda
-> N 缝暗纹与主极大半角宽
-> 单缝包络调制
-> 缺级 k=k' d/a

分辨：
圆孔衍射第一暗环 D sin(theta1)=1.22lambda
-> 瑞利判据
-> theta_min=1.22lambda/D

X 射线：
相邻晶面反射光程差 delta=2d sin(phi)
-> 布拉格公式 2d sin(phi)=k lambda
```

第三遍：题型应用，目标是“看到关键词就能选择模型”。

| 关键词 | 立即进入 |
| --- | --- |
| 单缝、中央明纹、焦距 | `Delta x0=2f lambda/a` |
| 单缝、暗纹、第 k 级 | `a sin(theta)=k lambda` |
| 光栅、主极大、第 k 级 | `d sin(theta)=k lambda` |
| 缺级、不出现 | 联立 `d sin(theta)=k lambda` 与 `a sin(theta)=k' lambda` |
| N 条缝、暗纹个数 | `N-1` 暗纹、`N-2` 弱次极大 |
| 双线、谱线刚分辨 | `R=lambda/delta lambda=Nk` |
| 望远镜、口径、最小角距 | `theta_min=1.22lambda/D` |
| 晶面、掠射角、X 射线 | `2d sin(phi)=k lambda` |

建议每遍结束后都做一次 3 分钟复述：

```text
不看资料，说出：
1. 本章的一个总图像。
2. 单缝、光栅、圆孔、晶体各自的关键光程差。
3. 本章最容易混的三对量：a/d、色散/分辨、掠射角/法线角。
```
