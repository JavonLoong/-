# 第28章 原子中的电子

## 1. 章节一句话总纲

原子中的电子不是沿经典轨道“绕圈的小球”，而是由离散能级、量子数、自旋和泡利不相容原理共同标记的量子态；光谱线是能级跃迁留下的能量差记录，核外电子排布是“量子态标签 + 泡利限制 + 能量最低”共同决定的结构。

本章考试主线按优先级分为：

| 优先级 | 内容块 | 复习定位 |
| --- | --- | --- |
| A | 氢原子能级、跃迁谱线、里德伯公式、量子数、角动量 | 期末总复习明确高频，必须熟练 |
| B | 玻尔模型、类氢离子、电子自旋、泡利不相容、壳层容量 | 课件主线，常用于概念题和填空题 |
| C | X 射线、分子光谱、激光简介 | 原课件包含，但期末总复习未列为第28章高频核心，考试优先级需核对原课件/历年题 |

## 2. 核心问题

1. 为什么氢原子光谱是分立线，而不是连续谱？
2. 玻尔理论中哪些图像可以保留，哪些经典轨道图像不能当真？
3. 氢原子能级 `E_n = -13.6/n^2 eV` 如何推出跃迁频率和里德伯公式？
4. 一群处于激发态 `n` 的氢原子最多能产生多少条谱线？
5. 为什么可见光谱线通常优先看巴耳末系，也就是跃迁到 `n = 2`？
6. 四个量子数 `n, l, m_l, m_s` 分别描述电子态的什么信息？
7. `L` 和 `L_z` 为什么不是同一个量？题目问“大小”和“z 分量”时该分别用什么公式？
8. 电子自旋为什么不能理解为经典小球自转？斯特恩-盖拉赫实验提供了什么证据？
9. 泡利不相容原理如何推出每个亚层、每个壳层最多容纳多少电子？
10. X 射线、分子光谱、激光是否在本次考试中高频出现，需核对原课件/历年题。

## 4. 概念依赖图

```text
实验光谱是线状
  -> 光子能量 E = hν = hc/λ
  -> 谱线对应两个能级的能量差
  -> 氢原子能级 E_n = -13.6/n^2 eV
  -> 里德伯公式与谱线系
  -> 谱线条数、最短波长、类氢离子

氢原子是中心力场
  -> 守恒量完全集 H, L^2, L_z, S_z
  -> 主量子数 n 决定能级
  -> 角量子数 l 决定轨道角动量大小
  -> 磁量子数 m_l 决定 L_z 与空间取向
  -> 自旋磁量子数 m_s 决定电子自旋取向

电子有自旋且全同
  -> 斯特恩-盖拉赫实验给出两种自旋取向
  -> 电子是费米子
  -> 泡利不相容原理
  -> 亚层容量 2(2l+1)
  -> 壳层容量 2n^2
  -> 核外电子排布

能级跃迁图像的拓展
  -> X 射线：内壳层跃迁与轫致辐射
  -> 分子光谱：电子、振动、转动能级共同跃迁
  -> 激光：受激辐射 + 粒子数反转 + 谐振腔
```

| 谱线条数 | 一群氢原子、处于激发态 `n`、最多 | 多个原子可级联形成所有向下跃迁 | 用 `N=n(n-1)/2` | 若题目强调选择定则，需核对原课件/历年题 |
| 里德伯公式 | 赖曼、巴耳末、帕邢、波数 | 下能级固定成某谱线系 | 写 `1/λ=R(1/n_l^2-1/n_u^2)` | `n_u > n_l` |

| 角动量大小与分量 | 角动量大小、z 分量、空间取向 | 大小来自 `L^2`，分量来自 `L_z` | 大小用 `sqrt[l(l+1)]ℏ`，z 分量用 `m_lℏ` | 不要把 `L` 写成 `m_lℏ` |

## 7. 易错点清单

- 把玻尔轨道当成真实电子运动轨迹。考试中可用玻尔模型算能级和半径，但量子力学图像中电子没有经典轨道。
- 发射光子时把 `ΔE` 写成负值。正确做法是 `hν = |E_i - E_f|`。
- 可见光谱线题没有先想到巴耳末系 `n -> 2`。
- 最短波长和最大能量关系弄反。`λ` 越短，`hc/λ` 越大。
- 里德伯公式上下能级写反，导致波数为负。
- 谱线条数 `n(n-1)/2` 的前提是“一群原子”并且允许级联跃迁；若题目要求选择定则，需核对原课件/历年题。
- `l` 的范围写错。必须是 `0,1,...,n-1`，不是到 `n`。
- `m_l` 的范围写错。必须满足 `-l <= m_l <= l`。
- 把 `L = sqrt[l(l+1)]ℏ` 和 `L_z = m_lℏ` 混成一个公式。
- 把电子自旋量子数 `s=1/2` 和自旋磁量子数 `m_s=±1/2` 混掉。
- 壳层容量忘记自旋简并，写成 `n^2` 而不是 `2n^2`。
- X 射线连续谱和特征谱混淆：连续谱截止波长由加速电压决定，特征谱由元素决定。
- 激光题只写“高能级跃迁”，忘记粒子数反转和受激辐射。

## 8. 最低掌握标准

达到最低掌握，必须能在闭卷状态下完成以下动作：

1. 用一句话解释：氢光谱线来自离散能级之间的跃迁。
2. 默写并使用 `E_n=-13.6/n^2 eV`、`hν=|ΔE|`、`1/λ=R(1/n_l^2-1/n_u^2)`。
3. 看到“可见光”能立刻想到巴耳末系 `n -> 2`。
4. 会算从 `n=4` 激发态最多产生 `6` 条谱线。
5. 会写四个量子数允许范围：`n`、`l`、`m_l`、`m_s`。
6. 会由 `|m_l| <= l <= n-1` 反推可能的 `l`。
7. 会区分并使用 `L=sqrt[l(l+1)]ℏ` 与 `L_z=m_lℏ`。
8. 会说清泡利不相容原理，并用它得到亚层容量 `2(2l+1)` 和壳层容量 `2n^2`。
9. 若考试覆盖课件拓展部分，至少能区分：X 射线连续谱/特征谱、分子电子/振动/转动光谱、激光的粒子数反转；覆盖程度需核对原课件/历年题。

## 9. 高分掌握标准

达到高分掌握，需要在最低标准上再做到：

1. 能从 `E_n=-13.6/n^2 eV` 推出跃迁频率条件和里德伯公式，而不是只背公式。
2. 能评价玻尔理论：定态、能级、跃迁频率、角动量量子化的结论仍有价值；经典轨道和硬加量子化条件有局限。
3. 能解释氢原子中 `H, L^2, L_z, S_z` 可作为标记电子态的一组量，并对应到四个量子数。
4. 能说明氢原子能量只依赖 `n`，所以同一 `n` 下不同 `l,m_l` 的态能量简并；忽略自旋时简并度为 `n^2`，计入自旋时为 `2n^2`。
5. 能处理类氢离子，把能级和谱线中的 `Z^2` 因子带进去。
6. 能用斯特恩-盖拉赫实验解释自旋两取向，并知道自旋不是经典小球自转。
7. 能把自旋-轨道耦合和精细结构联系起来：`J=L+S`，`j=l±1/2`，但具体光谱项考法需核对原课件/历年题。
8. 能把“全同性 -> 费米子反对称波函数 -> 泡利不相容 -> 壳层排布”连成一条逻辑链。
9. 若拓展部分入卷，能说明 X 射线连续谱、特征谱、分子光谱和激光都仍然是在使用“能级差与辐射”这一总图像。

## 10. 三遍法学习入口

### 第一遍：物理图像

目标：先把本章想象成一张能级楼梯图，而不是一堆公式。

自问清单：

1. 为什么氢原子只能发出某些特定颜色的光？
2. 电子从高能级掉到低能级，光子能量从哪里来？
3. `n,l,m_l,m_s` 分别像是在给电子状态贴什么标签？
4. 泡利不相容为什么会限制核外电子排布？
5. X 射线、分子光谱、激光和氢原子光谱有什么共同图像？

产出要求：能不用公式讲清“能级、跃迁、量子数、泡利”的关系。

### 第二遍：公式血缘

目标：把公式按来源串起来。

推荐顺序：

```text
E = hν = hc/λ
-> E_n = -13.6/n^2 eV
-> hν = |E_i - E_f|
-> 1/λ = R(1/n_l^2 - 1/n_u^2)
-> Nmax = n(n-1)/2
-> n,l,m_l,m_s 范围
-> L = sqrt[l(l+1)]ℏ, L_z = m_lℏ
-> 2(2l+1), 2n^2
```

产出要求：每个公式至少能说出“它从哪个物理图像来、什么时候不能用”。

### 第三遍：题目应用

目标：把知识变成考试动作。

训练顺序：

1. `n=4` 氢原子谱线条数。
2. `n=4` 到巴耳末系的最短可见波长。
3. 给 `n,m_l` 反推 `l,m_s,L,L_z`。
4. 给 `n,l` 求亚层态数和最多电子数。
5. 类氢离子 `He+` 的能级或跃迁波长。
6. X 射线或激光概念判断题，是否训练取决于历年题覆盖情况。

产出要求：每题写出“题型识别 -> 物理图像 -> 公式 -> 条件检查 -> 结果”五步。
