# 第 24 章 光的偏振

## 1. 章节一句话总纲

光的偏振不是先背公式，而是先盯住一个图像：光是横波，电场矢量只能在垂直传播方向的平面内振动；第 24 章所有题，基本都是在问“振动方向如何被筛选、分解、延迟、反射选择，最后形成怎样的偏振态和光强”。

本章主线：

```text
横波光矢量
=> 自然光 / 线偏振 / 圆偏振 / 椭圆偏振 / 部分偏振
=> 偏振片筛方向，马吕斯定律算光强
=> 反射界面按 s、p 分量选择，布儒斯特角给完全偏振
=> 各向异性晶体按 o、e 本征态分解，波片只改相位差
=> 振幅关系 + 相位差决定最终偏振态
```

期末优先级建议：

| 优先级 | 内容 | 调度备注 |
| --- | --- | --- |
| A | 自然光过偏振片、马吕斯定律、多片偏振片 | 高频、计算最直接 |
| A | 布儒斯特角、反射光完全偏振 | 高频、容易把角度混掉 |
| A | 四分之一波片、二分之一波片、偏振态判断 | 高频或中高频，必须会拆分量 |
| B | 自然光、部分偏振光、偏振度、检偏 | 概念题和选择题友好 |
| B | 双折射、o 光/e 光、晶体光轴 | 课件明确讲，题型需核对历年题 |
| C | 偏振光干涉、人工双折射、旋光、散射光偏振 | 课件覆盖，但期末权重需核对原课件/历年题 |

## 2. 核心问题

1. 什么是“偏振态”？
   - 光矢量方向固定：线偏振。
   - 光矢量端点转成圆：圆偏振。
   - 光矢量端点转成椭圆：椭圆偏振。
   - 光矢量方向随机、没有优势方向：自然光。
   - 自然光叠加一部分完全偏振光：部分偏振光。

2. 偏振片到底做了什么？
   - 偏振片不是“让光变强/变弱”的普通滤光片，而是只让沿通光方向的电场分量通过。
   - 自然光过第一片理想偏振片：两个等强正交分量只保留一个，所以 `I = I0 / 2`。
   - 线偏振光过检偏器：只取投影，振幅乘 `cos alpha`，强度乘 `cos^2 alpha`。

3. 为什么布儒斯特角会出现完全偏振反射光？
   - 界面把光矢量分成 `s` 分量和 `p` 分量。
   - 在布儒斯特角，反射光只剩 `s` 分量，所以反射光是线偏振光。
   - 此时反射线与折射线互相垂直，即 `iB + r = 90°`。

4. 波片为什么能改变偏振态？
   - 波片本身不靠吸收来起偏，而是让沿 o 光/e 光方向的两个正交分量走出不同相位。
   - 最后偏振态由两个量共同决定：分量振幅比 + 相位差。

5. 本章最小闭环是什么？
   - 先画“传播方向 + 光矢量方向”。
   - 再选本征分量：偏振片用通光方向，界面用 `s/p`，晶片用 `o/e`，旋光用左/右圆偏振。
   - 最后用光强或相位差判断结果。

| 前置知识 | 本章用法 | 不会时的补救入口 |

## 4. 概念依赖图

```text
电磁波是横波
  -> 光矢量只能在垂直传播方向的平面内振动
    -> 偏振态
      -> 自然光：方向随机、无优势方向
      -> 线偏振光：方向固定
      -> 圆/椭圆偏振光：光矢量端点作圆/椭圆
      -> 部分偏振光：自然光 + 完全偏振光

偏振态
  -> 起偏与检偏
    -> 偏振片通光方向
    -> 自然光先减半
    -> 线偏振光投影
    -> 马吕斯定律

界面反射/折射
  -> s 分量：光矢量垂直入射面
  -> p 分量：光矢量平行入射面
  -> 布儒斯特角
    -> 反射光完全偏振
    -> iB + r = 90°

各向异性晶体
  -> 光轴
  -> o 光 / e 光
  -> 双折射
  -> 晶片/波片
    -> 四分之一波片
    -> 二分之一波片
    -> 偏振态转换

本征偏振态
  -> 界面：s / p
  -> 晶片：o / e
  -> 旋光：左旋 / 右旋圆偏振
```

最低复习路径：

```text
偏振态分类
-> 偏振片和马吕斯定律
-> 布儒斯特角
-> 波片相位延迟
-> 偏振态判断
```

## 7. 易错点清单

1. 自然光过第一片偏振片忘记先除以 `2`。
2. 已经通过第一片后的光是线偏振光，后面不要再反复除以 `2`。
3. 多片偏振片必须按相邻偏振化方向夹角连乘，不是只看第一片和最后一片。
4. 马吕斯定律的 `alpha` 是“入射线偏振方向”和“检偏器通光方向”的夹角。
5. 强度用 `cos^2 alpha`，振幅才用 `cos alpha`。
6. 两片正交偏振片中间插入偏振片后可能有光透过，这正是投影链条的结果。
7. 布儒斯特角题中，入射角、反射角、折射角不能混：`iB + r = 90°`，不是 `2iB = 90°`。
8. `tan iB = n` 只适合空气到折射率为 `n` 的介质；一般式是 `tan iB = n2/n1`。
9. 反射光完全偏振时，反射光主要是 `s` 分量；折射光通常是部分偏振，不是完全偏振。
10. 判断偏振方向时，必须先确定“入射面”，再说 `s` 垂直入射面、`p` 平行入射面。
11. 四分之一波片不一定把线偏振变成圆偏振；只有入射偏振方向与光轴成 `45°` 时，两个分量振幅相等。
12. 波片不靠吸收削弱光强，核心作用是改变 o/e 两个分量的相位差。
13. 判断线、圆、椭圆偏振时不能只看相位差，还要看两个正交分量振幅是否相等。
14. 半波片让线偏振方向关于光轴对称翻折，总旋转角是 `2alpha`。
15. 两块光轴相同的四分之一波片串联，等效于半波片。
16. o 光、e 光都是线偏振光，且振动方向互相垂直。
17. o 光遵从折射定律；e 光一般不遵从折射定律，也不一定在入射面内。
18. 晶体光轴是晶体中的特殊方向，不是某条实物刻线；沿光轴传播不发生双折射。
19. 自然光和圆偏振光只用一片转动偏振片都可能表现为光强不变，区分时要加四分之一波片。
20. 偏振光干涉、人工双折射、旋光的计算深度需核对原课件/历年题，不建议无依据拔高。

## 8. 最低掌握标准

达到最低标准，期末遇到基础偏振题不应空题：

1. 能说清本章一句话图像：偏振研究光矢量的振动方向。
2. 能区分自然光、线偏振光、圆偏振光、椭圆偏振光、部分偏振光。
3. 会写自然光过第一片偏振片：`I = I0 / 2`。
4. 会写马吕斯定律：`I = Iin cos^2 alpha`。
5. 会做自然光通过两片、三片理想偏振片的强度连乘。
6. 会写布儒斯特定律：`tan iB = n2/n1`。
7. 知道布儒斯特角时 `iB + r = 90°`。
8. 知道四分之一波片的相位差是 `pi/2`，二分之一波片是 `pi`。
9. 会用三条快判规则：
   - `Delta phi = 0` 或 `pi`：线偏振。
   - `Ax = Ay` 且 `Delta phi = ±pi/2`：圆偏振。
   - 其他一般是椭圆偏振。
10. 知道线偏振光与四分之一波片光轴成 `45°` 入射，出射为圆偏振。
11. 知道半波片使线偏振方向相对光轴翻折，转角为 `2alpha`。
12. 对双折射至少知道：o 光/e 光都是线偏振光，方向互相垂直；o 光遵从折射定律，e 光一般不遵从。

最低做题模板：

```text
1. 先判断入射光是什么偏振态。
2. 如果是自然光遇第一片偏振片，先 I0/2。
3. 如果是线偏振光遇偏振片，用 cos^2。
4. 如果题目说反射光完全偏振，用布儒斯特角。
5. 如果题目说波片，先分解到 o/e 或快慢轴，再看相位差。
```

## 9. 高分掌握标准

高分不是多背几个公式，而是能在题目换说法时自动切换“本征分量”：

1. 看到偏振片，自动切换到“通光方向/垂直通光方向”分解。
2. 看到界面反射折射，自动切换到 `s/p` 分解。
3. 看到晶体或波片，自动切换到 `o/e` 分解。
4. 看到旋光，自动切换到左旋/右旋圆偏振分解。
5. 能解释马吕斯定律来自振幅投影，而不是死记强度公式。
6. 能用“是否消光”判断自然光、线偏振光、部分偏振光。
7. 能说明为什么一片偏振片不能区分自然光和圆偏振光，必须借助四分之一波片。
8. 能处理“最大/最小光强”类检偏问题，并知道偏振度公式是否需要核对原课件/历年题。
9. 能在布儒斯特角题中同时求 `iB`、`r`、偏振方向。
10. 能从 `Ax, Ay, Delta phi` 系统判断线/圆/椭圆，而不是只背特殊结论。
11. 能解释半波片的“镜像翻折”图像，并正确画出输出偏振方向。
12. 能把两块四分之一波片光轴相同时等效为半波片。
13. 能说出偏振光干涉的关键：P2 把 o/e 两束重新投到同一方向，才可能干涉。
14. 能把自然旋光与磁致旋光的往返效果区分开：自然旋光往返抵消，磁致旋光往返加倍。
15. 能根据考试时间决定取舍：A 级内容必须秒做，C 级内容只做课件原型题，深挖需核对历年题。

高分题的通用答题骨架：

```text
物理图像：
本题按哪两个本征偏振态分解？

关键方程：
偏振片：I = Iin cos^2 alpha
布儒斯特：tan iB = n2/n1, iB + r = 90°
波片：Delta phi = 2pi(no-ne)d/lambda

结果判断：
强度题给 I/I0；
角度题给 iB 或 r；
偏振态题同时说明振幅关系和相位差。
```

## 10. 三遍法学习入口

### 第一遍：直觉图像遍

目标：不看公式也能知道每个器件在“处理光矢量的什么东西”。

1. 画一束光沿 `z` 方向传播，电场只能在 `x-y` 平面内振动。
2. 画自然光：很多随机方向。
3. 画线偏振光：一个固定方向。
4. 画圆/椭圆偏振光：光矢量端点转圈/椭圆。
5. 画偏振片：只放行通光方向投影。
6. 画界面：分成 `s` 和 `p` 两个方向。
7. 画晶片：分成 `o` 和 `e` 两个方向，出来时相位差改变。

第一遍自测：

```text
偏振片改变什么？筛方向。
波片改变什么？改相位差。
布儒斯特角选择什么？反射光只剩 s 分量。
```

### 第二遍：公式血缘遍

目标：每个公式都知道从哪个图像来。

1. 从自然光分解推出 `I = I0/2`。
2. 从振幅投影推出 `I = Iin cos^2 alpha`。
3. 从反射线与折射线垂直推出 `tan iB = n2/n1`。
4. 从 o/e 光折射率不同推出 `Delta phi = 2pi(no-ne)d/lambda`。
5. 从 `Delta phi` 得到四分之一波片、二分之一波片条件。
6. 用 `Ax, Ay, Delta phi` 做偏振态快判。

第二遍自测：

```text
为什么马吕斯是 cos^2 而不是 cos？
为什么四分之一波片不是一定出圆偏振？
为什么半波片转角是 2alpha？
```

### 第三遍：题型应用遍

目标：看到题目关键词能马上落到模板。

1. “自然光 + 多片偏振片”：先减半，再连乘。
2. “反射光完全偏振”：立刻写布儒斯特角。
3. “四分之一波片 + 45°”：线偏振变圆偏振。
4. “半波片 + 夹角 alpha”：输出方向翻折，转 `2alpha`。
5. “检偏有无消光”：判断自然光、线偏振、部分偏振。
6. “方解石/双折射”：先判断 o/e 光与光轴关系。

第三遍限时训练建议：

```text
10 分钟：多片偏振片 3 题
10 分钟：布儒斯特角 3 题
10 分钟：波片偏振态判断 3 题
5 分钟：易错点口述复盘
```

### 考前最后 20 分钟

只看这 8 条：

```text
1. 自然光过第一片偏振片：I = I0/2。
2. 后续偏振片逐片乘 cos^2(相邻夹角)。
3. 马吕斯定律适用于线偏振光入射检偏器。
4. 布儒斯特角：tan iB = n2/n1，且 iB + r = 90°。
5. 波片相位差：Delta phi = 2pi(no-ne)d/lambda。
6. 四分之一波片：pi/2；半波片：pi。
7. 圆偏振条件：Ax = Ay 且 Delta phi = ±pi/2。
8. 波片题先拆 o/e 分量，别只背结论。
```
