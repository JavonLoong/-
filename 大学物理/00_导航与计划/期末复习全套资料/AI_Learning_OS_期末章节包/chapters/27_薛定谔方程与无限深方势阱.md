# 第27章 薛定谔方程与无限深方势阱

## 1. 章节一句话总纲

第27章的主线是：用薛定谔方程描述微观粒子的“态”如何随时间演化，再用无限深方势阱这个最简单模型看清楚能量量子化、本征态、态叠加、测量概率和平均值。

学习顺序必须是：

```text
物理图像
-> 波函数表示状态
-> 哈密顿量决定演化
-> 定态方程给出本征值和本征函数
-> 无限深方势阱用边界条件筛出允许能量
-> 任意初态展开成本征态
-> 测量结果由展开系数模方决定
```

本章不是单纯背微分方程，而是训练一个动作：看到波函数，先判断它是不是某个力学量的本征态；如果不是，就把它展开到本征态基底上，再读出概率、平均值和时间演化。

## 2. 核心问题

1. 薛定谔方程在量子力学里扮演什么角色？
   - 它不是从牛顿定律推出来的，而是非相对论量子力学的基本动力学假设；地位类似经典力学中的牛顿方程。

2. 波函数 `Psi` 到底可不可以直接测量？
   - `Psi` 通常是复函数，本身不可直接测量；可测的是概率密度 `|Psi|^2`。

3. 为什么势能不显含时间时要转向定态薛定谔方程？
   - 因为此时可以分离变量，把“随时间变化”拆成相位因子，把核心问题化成能量本征值问题。

4. 无限深方势阱为什么会出现离散能级？
   - 粒子被限制在有限空间内，波函数在阱壁处必须为零；只有能在阱内形成驻波的波长被允许，因此能量只能取一串离散值。

5. 为什么最低能量不是零？
   - `n = 0` 会导致波函数处处为零，不是物理态；最低可取 `n = 1`，对应零点能。这是量子限制和不确定关系的体现。

6. 已知初态后，怎样写出任意时刻的波函数？
   - 先把初态写成本征函数线性组合 `sum c_n psi_n(x)`，再给每一项乘自己的时间因子 `exp(-i E_n t / hbar)`。

7. 测量能量时，为什么答案通常不是一个确定值？
   - 只有处在单一能量本征态时，能量测量才确定；叠加态中测得 `E_n` 的概率是 `|c_n|^2`。

8. 课件后半的势垒穿透、一维谐振子、力学量算符怎么处理？
   - 力学量算符与“测量/本征态”直接相关，应纳入主线；势垒穿透和一维谐振子是课件覆盖内容，但期末权重需核对原课件/历年题。

## 3. 前置知识

| 前置知识 | 本章用法 | 不稳时的表现 |
| --- | --- | --- |
| 复指数 `exp(i theta)` | 表示平面波、时间相位和动量本征态 | 看见虚数 `i` 就以为没有物理意义 |
| 德布罗意关系 `lambda = h / p` | 建立“粒子也是波”的直觉 | 只把势阱当成经典小球反弹 |

| 二阶常系数微分方程 | 求无限深方势阱本征函数 | 不会从 `Phi'' + k^2 Phi = 0` 写出正弦/余弦 |
| 边界条件 | 在阱壁处要求波函数为零 | 得到连续能量，或保留 `n = 0` |
| 归一化 | 确定波函数前的系数 | 把概率算成大于 1 |
| 正交归一 | 用投影求展开系数 `c_n` | 把不同本征态的系数混在一起 |
| 三角恒等式 | 把初态拆成本征态 | 看见 `sin x cos x` 或 `sin x(1-cos x)` 就卡住 |
| 概率平均 | 求平均能量、平均动量 | 把平均值当成本征值，或把概率幅当概率 |

## 4. 概念依赖图

```text
波粒二象性
  -> 微观粒子需要用波函数 Psi 描述
  -> |Psi|^2 是概率密度
  -> 薛定谔方程给出 Psi 的时间演化

薛定谔方程
  -> 哈密顿算符 Hhat
  -> 势能 U 不显含时间
  -> 分离变量 Psi(r,t)=Phi(r)T(t)
  -> 定态薛定谔方程 Hhat Phi = E Phi
  -> 能量本征值 E 与本征函数 Phi

一维定态方程
  -> 给定 U(x)
  -> 无限深方势阱 U=0(阱内), U=无穷(阱外)
  -> 阱外 Phi=0, 阱壁 Phi=0
  -> 阱内驻波
  -> k = n pi / a
  -> En = n^2 pi^2 hbar^2 / (2 m a^2)
  -> psi_n(x) = sqrt(2/a) sin(n pi x/a)  [0<x<a 版本]

本征函数完备性
  -> 任意合理初态 Psi(x,0)=sum c_n psi_n(x)
  -> 时间演化 Psi(x,t)=sum c_n psi_n(x) exp(-i En t/hbar)
  -> 能量测量 P(En)=|c_n|^2
  -> 平均能量 <E>=sum |c_n|^2 En

力学量算符
  -> p_hat = -i hbar d/dx
  -> Hhat = -hbar^2/(2m) d^2/dx^2 + U(x)
  -> 算符本征方程 Ahat psi_n = A_n psi_n
  -> 测量结果只能是本征值

课件补充分支
  -> 势垒穿透：有限势垒中 E<U 仍有非零概率透入/穿过
  -> 一维谐振子：能级 En=(n+1/2)hbar omega
  -> 上述两项作为低优先补充分支处理
```

## 7. 易错点清单

1. 把 `Psi` 当成可直接测量的物理量。正确说法是 `|Psi|^2` 是概率密度。
2. 忘记薛定谔方程是非相对论形式，不能拿它处理相对论高速粒子。
3. 势能不显含时间时，定态解的时间部分是 `exp(-iEt/hbar)`，不是 `exp(+iEt/hbar)`；不同教材符号约定可能影响整体相位写法，需按课件约定。
4. 把“定态”误解成波函数不随时间变化。定态的波函数有时间相位，概率密度不随时间变化。
5. 无限深势阱的区间没看清。`0<x<a` 用正弦基底最方便；课件推导的中心阱 `-a/2<x<a/2` 会出现奇偶函数分类。
6. 在无限深阱外还保留非零波函数。无限深势阱外 `Phi=0`，边界处也必须为零。
7. 保留 `n=0`。`n=0` 给出零波函数，不是物理态。
8. 把 `E_n=n^2 h^2/(8ma^2)` 写成与 `n` 成正比。
9. 混用 `h` 和 `hbar`：`hbar=h/(2pi)`；`n^2 h^2/(8ma^2)` 等价于 `n^2 pi^2 hbar^2/(2ma^2)`。
10. 忘记归一化系数 `sqrt(2/a)`。
11. 初态展开时先慌，没先用三角恒等式化简。
12. 把概率幅 `c_n` 当成概率；真正概率是 `|c_n|^2`。
13. 计算区间概率时积分 `Psi`，而不是积分 `|Psi|^2`。
14. 叠加态的 `|Psi|^2` 忘记交叉项，因此漏掉随时间振荡的项。
15. 求平均能量时把 `E1`、`E2` 简单平均，没用概率加权。
16. 箱中单一能量本征态里，能量确定，但动量一般不是单一确定值。
17. 把“可能测得的动量值”和“平均动量”混为一谈；对称的 `+p`、`-p` 可以使平均动量为零，但可能值不是零。
18. 把有限势垒的隧穿图像套到无限深势阱壁上。无限深势垒是理想极限，阱外波函数为零。
19. 一维谐振子的能级是等间隔，势阱能级间隔随 `n` 增大；两者不要互换。
20. 对课件后半的势垒穿透、谐振子、算符内容不做区分：算符和测量属于本章主线，势垒/谐振子先按补充内容处理。

## 8. 最低掌握标准

达到最低标准，应该能在闭卷中稳拿基础分：

1. 能写出含时薛定谔方程、哈密顿算符、定态薛定谔方程。
2. 能说明 `Psi` 与 `|Psi|^2` 的区别。
3. 能说清楚定态、本征值、本征函数的含义。
4. 能从无限深势阱边界条件写出 `psi_n(x)` 和 `E_n`。
5. 能判断 `n=0` 不允许，最低能量 `E1>0`。
6. 能对已给出的叠加态写时间演化。
7. 能由 `c_n` 写出测得 `E_n` 的概率和平均能量。
8. 能计算简单区间概率，知道必须积分 `|Psi|^2`。
9. 能识别中心阱和 `0<x<a` 势阱只是坐标选取不同。
10. 能写出动量算符 `p_hat=-i hbar d/dx`，知道平面波是动量本征态。

最低限度的闭卷模板：

```text
1. 先看 U(x) 和边界。
2. 写定态方程。
3. 阱内解成正弦/余弦。
4. 用边界条件筛出 k=n pi/a。
5. 写 En。
6. 初态若是叠加，找 c_n。
7. 概率用 |c_n|^2，平均值用 sum |c_n|^2 A_n。
```

## 9. 高分掌握标准

达到高分标准，应该不仅会套公式，还能处理变形题：

1. 能从含时薛定谔方程完整分离变量，解释为什么出现能量本征方程。
2. 能区分单一本征态和叠加态的时间行为：前者概率密度不变，后者可能有干涉振荡。
3. 能在 `0<x<a` 与 `-a/2<x<a/2` 两种坐标约定之间转换，不被奇偶函数形式干扰。
4. 能用三角恒等式快速把初态拆成少数几个势阱本征态。
5. 能用投影积分 `c_n=integral psi_n^* Psi dx` 处理不能直接看出的展开。
6. 能规范写出测量语言：可能值是本征值，概率是展开系数模方，平均值是概率加权。
7. 能处理两态叠加的区间概率，保留交叉项和相位差 `cos[(E_m-E_n)t/hbar]`。
8. 能说明为什么无限深势阱有零点能，并用不确定关系给出物理解释。
9. 能辨析“能量确定不等于位置确定，也不等于动量确定”。
10. 能把力学量算符、本征方程、态叠加和测量原理连成一条逻辑链。
11. 若考试覆盖课件后半，能定性说明势垒穿透，并记住穿透概率随势垒宽度和高度指数下降。
12. 若考试覆盖谐振子，能说出 `E_n=(n+1/2)hbar omega`、等间隔能级和零点能。

高分答题习惯：

```text
先写物理模型和边界条件，再写公式；
先说明本征态/叠加态，再谈测量概率；
先检查归一化，再计算平均值；
最后用量纲、概率和极限情况检查答案。
```

## 10. 三遍法学习入口

### 第一遍：直觉图像

目标：不用推导，先能说清楚“这章在解决什么问题”。

自测问题：

1. 为什么微观粒子不能只用轨迹描述？
2. 为什么 `|Psi|^2` 是概率密度，而不是 `Psi`？
3. 无限深势阱为什么像一根两端固定的弦？
4. 为什么只有某些驻波能存在？
5. 为什么零点能不是计算误差？

本遍输出：

```text
用 3 句话解释：
薛定谔方程决定波函数演化；
无限深势阱用边界条件筛出驻波；
测量概率来自本征态展开系数模方。
```

### 第二遍：公式推导

目标：把核心公式从一条线推出来。

推导路线：

```text
i hbar partial_t Psi = Hhat Psi
-> U 不显含 t，令 Psi=Phi T
-> Hhat Phi = E Phi
-> 一维形式 Phi'' + (2m/hbar^2)(E-U)Phi=0
-> 无限深阱内 U=0
-> Phi'' + k^2 Phi=0
-> 边界条件
-> k=n pi/a
-> E_n=hbar^2 k^2/(2m)
-> psi_n 归一化
```

本遍必须能独立写出：

```text
psi_n(x)=sqrt(2/a)sin(n pi x/a)
E_n=n^2 pi^2 hbar^2/(2ma^2)
Psi(x,t)=sum c_n psi_n(x)exp(-iE_n t/hbar)
P(E_n)=|c_n|^2
<E>=sum |c_n|^2 E_n
```

### 第三遍：题目应用

目标：把公式变成考场动作。

训练题入口：

1. 给 `Psi(x,0)=c1 psi1+c2 psi2`，写 `Psi(x,t)`、能量概率、平均能量。
2. 给 `Psi(x,0)=sqrt(8/5a)(1-cos(pi x/a))sin(pi x/a)`，拆成本征态。
3. 对两态叠加，求 `0~a/2` 内找到粒子的概率随时间表达式。
4. 判断某波函数是不是能量本征态、动量本征态。
5. 给平面波叠加，求可能动量、概率和平均动量。

第三遍检查句：

```text
我是否先找了本征基底？
我是否区分了概率幅和概率？
我是否检查了归一化？
我是否看清了势阱区间？
我是否把时间因子逐项加在对应能级上？
```
