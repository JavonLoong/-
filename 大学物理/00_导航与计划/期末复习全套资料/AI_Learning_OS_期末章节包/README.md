## 调度规则

- 第 22-24 章是光学主线：干涉、衍射、偏振分别训练“相位叠加”“波面连续次波源”“光矢量方向”。
- 第 26-28 章是量子主线：光子/物质波、势阱与测量、原子量子数和谱线。
- 第 29 章保持低优先度，只做概念维护与 `Eg` 阈值公式，不挤占第 22-28 章主线复习时间。
- 布拉格公式采用“第 23 章概念接口 + 第 26 章冲刺记忆和主要刷题归属”，避免重复刷卡。
- 第 28 章调用第 26 章的 `E = hν = hc/λ` 处理跃迁，不另立重复背诵卡。
- 第 28 章的量子数应和第 27 章本征态/本征值语言联动：量子数是一组可兼容力学量本征值的标签。
