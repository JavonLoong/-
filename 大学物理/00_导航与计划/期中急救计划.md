## 必背公式区

### 第 9 章 热学基础

1. 理想气体状态方程：`pV = nRT`
2. 理想气体内能：`U = i/2 · nRT`
3. 压强的统计意义：`p = (1/3) n m <v^2>`
4. 平均自由程：`λ = kT / (sqrt(2) π d^2 p)`
5. 黏度关系：`η = (1/3) ρ v̄ λ`
6. 范德瓦尔斯方程：`(p + a n^2 / V^2)(V - nb) = nRT`

### 第 10 章 热力学第一定律

1. 第一定律：`Q = ΔU + W`
2. 功：`W = ∫ p dV`
3. 理想气体内能改变量：`ΔU = n C_V ΔT`
4. 绝热过程：
   `pV^γ = 常量`
   `TV^(γ-1) = 常量`
   `T^γ p^(1-γ) = 常量`
5. 多方过程做功（会认就行）：`W = (p_2 V_2 - p_1 V_1) / (1 - n)`
6. 卡诺热机效率：`η = 1 - T_2 / T_1`

### 第 11 章 熵与第二定律

1. 可逆过程熵变：`ΔS = ∫ dQ_rev / T`
2. 等温过程：`ΔS = Q_rev / T`
3. 理想气体熵变：
   `ΔS = n C_V ln(T_2/T_1) + nR ln(V_2/V_1)`
   `ΔS = n C_P ln(T_2/T_1) - nR ln(p_2/p_1)`
4. 孤立系统：`ΔS_total >= 0`
5. 可逆过程：`ΔS_total = 0`
6. 不可逆过程：`ΔS_total > 0`

## 每章最常见题型模板

### 第 9 章

#### 1. 混合气体求末温

思路：

1. 先用 `pV = nRT` 找两边物质的量关系。
2. 再用能量守恒，写混合前后总内能相等。
3. 单原子、双原子的自由度别写错。

代表题：`9.17`

#### 2. 速率分布函数

思路：

1. 先用归一化 `∫f(v)dv = 1` 求常数。
2. 再算区间粒子比例。
3. 平均速率就是 `∫v f(v) dv`。

代表题：`9.18`

#### 3. 范德瓦尔斯方程

思路：

1. 先求 `n = m/M`
2. 再代入 `p = nRT/(V-nb) - an^2/V^2`
3. 别把单位搞混，L 要换 m^3

代表题：`9.26`

#### 4. 平均自由程和分子直径

思路：

1. 若给黏度，先用 `η = (1/3)ρv̄λ`
2. 再用 `λ = 1 / (sqrt(2) π d^2 n_0)` 求直径

代表题：`9.28`

### 第 10 章

#### 1. 已知路径求热量、功、内能

思路：

1. 先抓住 `ΔU` 是状态量，只和初末态有关。
2. 用一条已知路径先算出 `ΔU`
3. 换路径后继续用 `Q = ΔU + W`

代表题：`10.2`

#### 2. 绝热容器、活塞、隔板

思路：

1. 先判断谁是绝热过程
2. 压强平衡条件别漏
3. 最后用总能量守恒或 `ΔU = nC_VΔT`

代表题：`10.16`

#### 3. 循环效率

思路：

1. 先认清每一段是等温、等压、等容还是绝热
2. 效率永远先写 `η = W/Q_in = 1 - Q_out/Q_in`
3. 再把各过程关系代进去

代表题：`10.18`

### 第 11 章

#### 1. 和热库接触的熵变

思路：

1. 物体熵变用自身可逆路径算
2. 热库熵变直接用 `ΔS = -Q/T`
3. 最后求系统总熵变

代表题：`11.7`

#### 2. 可逆循环

思路：

1. 先判断每段过程类型
2. 分别算 `W`、`Q`、`ΔS`
3. 一个完整可逆循环总熵变应为 0

代表题：`11.9`

## 考前最后背这个

### 必会判断

1. 状态量：`U`、`S`
2. 过程量：`Q`、`W`
3. 理想气体内能只和温度有关
4. 绝热不代表温度不变，等温才是温度不变
5. 可逆绝热过程熵不变
6. 孤立系统总熵不会减小

### 最容易丢分的点

1. 双原子气体 `C_V = 5R/2`，单原子气体 `C_V = 3R/2`
2. `W` 的正负号要统一，资料里默认是“系统对外做功为正”
3. 温度进热力学公式要用 K，不要直接用摄氏度
4. 体积单位 L 要换成 `m^3`
5. 熵变要分“系统”“热库”“总熵变”
