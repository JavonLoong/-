## 历年题反复出现的高频考点

### 光学部分

- 双缝干涉：条纹间距、某级亮纹位置、插入介质片后的条纹移动
- 空间相干性：由光源宽度或距离判断条纹何时消失
- 迈克耳孙干涉仪：反射镜移动距离、条纹吞吐数、等倾/等厚条纹
- 单缝衍射：暗纹条件、中央明纹宽度
- 光栅衍射：主极大条件、缺级、可见级次、分辨本领
- 薄膜干涉与等厚干涉：增透膜、牛顿环/表面凹凸判断
- 偏振：马吕斯定律、布儒斯特角、偏振片串联、波片、圆偏振/椭圆偏振
- 光学仪器分辨本领

### 量子部分

- 黑体辐射：维恩位移定律、斯特藩-玻尔兹曼定律
- 光电效应：逸出功、截止电压、红限频率
- 康普顿散射：波长改变量、反冲电子动能
- 德布罗意波与不确定关系
- 无限深方势阱：本征函数、能量本征值、态展开、时间演化、概率计算
- 势垒穿透：定性判断穿透系数变化
- 氢原子：能级、谱线条数、可见谱、跃迁能量
- 量子数：`n, l, m_l, m_s`
- 角动量大小与分量
- X 射线衍射、布拉格公式

## 闭卷最该背熟的关系式

### 光学

- 双缝干涉条纹间距：`\Delta x = \lambda D / d`
- 亮纹条件：`\delta = k\lambda`
- 暗纹条件：`\delta = (k + 1/2)\lambda`
- 迈克耳孙干涉仪：`2\Delta d = N\lambda`
- 单缝暗纹条件：`a \sin\theta = k\lambda`
- 单缝中央明纹宽度：`2f\lambda / a`
- 光栅主极大：`d \sin\theta = k\lambda`
- 光栅分辨本领：`R = \lambda / \Delta\lambda = kN`
- 马吕斯定律：`I = I_0 \cos^2\theta`
- 自然光过理想偏振片：`I = I_0 / 2`
- 布儒斯特角：`\tan i_B = n_2 / n_1`

### 量子

- 光子能量：`E = h\nu = hc/\lambda`
- 光电效应方程：`h\nu = W + E_{k,\max} = W + eU_s`
- 康普顿公式：`\Delta\lambda = \lambda_C (1 - \cos\varphi)`
- 德布罗意关系：`\lambda = h/p`
- 不确定关系：`\Delta x \Delta p_x \gtrsim \hbar/2`
- 无限深势阱本征态：`\psi_n(x) = \sqrt{2/a}\sin(n\pi x / a)`
- 无限深势阱能级：`E_n = n^2 h^2 / (8ma^2)`
- 时间演化因子：`e^{-iE_n t/\hbar}`
- 氢原子能级：`E_n = -13.6/n^2 \text{ eV}`
- 从第 `n` 激发态最多产生谱线数：`n(n-1)/2`
- 轨道角动量大小：`L = \sqrt{l(l+1)}\,\hbar`
- 角动量 z 分量：`L_z = m_l \hbar`

## 复习时要特别警惕的失分点

- 只会背公式，但不会判断是亮纹还是暗纹
- 把双缝干涉、单缝衍射、光栅衍射三个条件混在一起
- 迈克耳孙干涉仪漏掉了光程差中的系数 `2`
- 偏振题没有先判断入射光是自然光、线偏振光还是椭圆偏振光
- 量子题里把“测量概率”和“平均值”混写
- 氢原子题不会由 `n, l, m_l, m_s` 反推允许取值
- 计算题只写结果，不写关键物理依据和中间关系式

## 最后建议

- 这门期末卷的填空分很多，所以**公式熟练度和题型识别速度**非常关键。
- 真正拉分的部分通常不是最难的大题，而是你能不能在短时间内认出这是哪一类干涉、衍射、偏振或量子题。
- 如果开始复习时感觉乱，优先把历年题按专题拆开，比从头死看课件更有效。
