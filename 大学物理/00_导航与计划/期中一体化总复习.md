## 二、你现在必须先背下来的总公式

### 热学总公式

1. 理想气体状态方程：`pV = nRT`
2. 理想气体内能：`U = i/2 · nRT`
3. 热力学第一定律：`Q = ΔU + W`
4. 理想气体内能变化：`ΔU = nC_VΔT`
5. 功：`W = ∫ p dV`
6. 绝热过程：`pV^γ = 常量`
7. 绝热过程：`TV^(γ-1) = 常量`
8. 熵变定义：`ΔS = ∫ dQ_rev / T`
9. 理想气体熵变：`ΔS = nC_V ln(T_2/T_1) + nR ln(V_2/V_1)`
10. 理想气体熵变：`ΔS = nC_P ln(T_2/T_1) - nR ln(p_2/p_1)`

### 振动波动总公式

1. 振动表达式：`x = A cos(ωt + φ)`
2. 角频率和周期：`ω = 2π/T`
3. 弹簧振子总能量：`E = 1/2 kA^2`
4. 波函数：`y = A cos(ωt ∓ kx + φ)`
5. 波速关系：`v = λf = ω/k`
6. 驻波波节：`x = nλ/2`
7. 驻波波腹：`x = (2n+1)λ/4`

### 光学开头总公式

1. 光强和电场平方平均有关：`I ∝ <E^2>_t`
2. 两列相干光叠加：`I = I_1 + I_2 + 2√(I_1 I_2) cosΔφ`
3. 双缝波程差：`δ ≈ d x / D`
4. 相位差：`Δφ = 2πδ/λ`
5. 明纹条件：`δ = kλ`
6. 暗纹条件：`δ = (2k+1) λ/2`
7. 条纹间距：`Δx = λD/d`

## 三、第 9 章热学基础

### 这一章到底在讲什么

这章本质上在讲三件事：

1. 温度和理想气体状态方程
2. 气体分子的统计规律
3. 真实气体、平均自由程、输运过程

### 你必须会的概念

1. 理想气体内能只和温度有关
2. 单原子气体自由度通常取 `3`
3. 双原子气体常温下自由度通常取 `5`
4. 平均自由程越大，分子碰撞越少
5. 范德瓦尔斯方程是理想气体方程的修正

### 必背公式

1. `pV = nRT`
2. `U = i/2 · nRT`
3. 平均自由程：`λ = kT / (sqrt(2) π d^2 p)`
4. 黏度关系：`η = (1/3) ρ v̄ λ`
5. 范德瓦尔斯方程：`(p + a n^2 / V^2)(V - nb) = nRT`

### 这一章最常考的 4 类题

#### 1. 混合气体求末温

思路：

1. 先用 `pV = nRT` 求两边物质的量关系
2. 再用总内能守恒
3. 自由度写对

代表结果你可以直接记一个例子：

已知氦气 `250 K`、氧气 `310 K`、体积相等压强相等，混合后温度：

`T = 8T_1T_2 / (3T_2 + 5T_1)`

代入得：

`T ≈ 284.4 K`

#### 2. 速率分布函数

思路：

1. 先归一化：`∫f(v)dv = 1`
2. 再求区间比例
3. 最后求平均速率：`v̄ = ∫v f(v)dv`

例题结论：

如果

`f(v) = av/v_0`，`0 ≤ v ≤ v_0`

`f(v) = a`，`v_0 ≤ v ≤ 2v_0`

那么

1. `a = 2/(3v_0)`
2. `v < v_0` 的粒子数是 `N/3`
3. `v > v_0` 的粒子数是 `2N/3`
4. 平均速率 `v̄ = 11v_0/9`

#### 3. 范德瓦尔斯方程

思路：

1. 先算 `n = m/M`
2. 再代入 `p = nRT/(V-nb) - an^2/V^2`
3. 单位要全换成 SI

例题结论：

CO₂ 在 `V = 20 L`、`m = 1.1 kg`、`T = 286 K` 时：

1. `n = 25 mol`
2. 理想气体压强 `≈ 2.972 × 10^6 Pa`
3. 范德瓦尔斯压强 `≈ 2.571 × 10^6 Pa`
4. 内压强 `≈ 5.69 × 10^5 Pa`

#### 4. 平均自由程和分子直径

思路：

1. 若已知黏度，先用 `η = (1/3)ρv̄λ`
2. 再用 `λ = 1/(sqrt(2) π d^2 n_0)` 求 `d`

例题结论：

氦气标准状态：

1. `λ ≈ 2.65 × 10^-7 m`
2. `d ≈ 1.78 × 10^-10 m`

### 第 9 章最容易错的点

1. 摄氏度不能直接带公式，要换 K
2. `L` 要换 `m^3`
3. 自由度别写错
4. 平均自由程公式里有 `sqrt(2) π d^2`

## 四、第 10 章热力学第一定律

### 这一章到底在讲什么

这章本质上讲的是：

1. 热量、功、内能之间怎么互相换
2. 不同热力学过程怎么算
3. 循环和效率怎么看

### 你必须分清的东西

1. `U` 是状态量
2. `Q` 是过程量
3. `W` 是过程量
4. 同样初末态，`ΔU` 一样，但 `Q` 和 `W` 可以不一样

### 必背公式

1. `Q = ΔU + W`
2. `ΔU = nC_VΔT`
3. `W = ∫ p dV`
4. 等温理想气体：`ΔU = 0`
5. 等容过程：`W = 0`
6. 绝热过程：`Q = 0`
7. 绝热：`pV^γ = 常量`
8. 绝热：`TV^(γ-1) = 常量`
9. 多方过程做功：`W = (p_2V_2 - p_1V_1)/(1-n)`
10. 热机效率：`η = W/Q_in = 1 - Q_out/Q_in`

### 这一章最常考的 3 类题

#### 1. 已知路径求 `Q`、`W`、`ΔU`

核心思路：

1. 先用一条已知路径算出 `ΔU`
2. 因为 `ΔU` 只和初末态有关
3. 再换路径继续用 `Q = ΔU + W`

例题结论：

已知沿 acb：

1. `Q = 560 J`
2. `W = 356 J`

所以

`ΔU = 204 J`

若沿 adb 做功 `220 J`，则

`Q = 204 + 220 = 424 J`

#### 2. 绝热容器和隔板问题

核心思路：

1. 找谁是绝热过程
2. 找压强平衡
3. 找体积约束
4. 最后用 `ΔU = nC_VΔT`

代表结论：

绝热容器中两边各 `1 mol` 氮气，B 侧加热，A 被压缩到体积减半：

1. `T_A ≈ 321.5 K`
2. `T_B ≈ 964.5 K`
3. `Q_B ≈ 16.6 kJ`

#### 3. 热机循环效率

核心思路：

1. 先认清过程类型
2. 先写 `η = 1 - Q_out/Q_in`
3. 再把几何关系、绝热关系代进去

狄赛尔循环记住结果形式即可：

`η = 1 − (σ^γ − 1) / [γ r^(γ−1)(σ − 1)]`

其中：

1. `r = V_1/V_2`
2. `σ = V'_1/V_2`

### 第 10 章最容易错的点

1. `Q = ΔU + W` 里默认“系统对外做功为正”
2. 等温不等于绝热
3. 绝热过程 `Q = 0`，不是 `ΔU = 0`
4. 循环中 `ΔU_total = 0`

## 五、第 11 章热力学第二定律和熵

### 这一章到底在讲什么

这章讲的是：

1. 自然过程为什么有方向
2. 熵到底是什么
3. 可逆和不可逆怎么判断

### 你必须会的概念

1. 孤立系统总熵不会减小
2. 可逆过程总熵变为 0
3. 不可逆过程总熵变大于 0
4. 熵是状态量

### 必背公式

1. `ΔS = ∫ dQ_rev/T`
2. 等温过程：`ΔS = Q_rev/T`
3. 理想气体：`ΔS = nC_V ln(T_2/T_1) + nR ln(V_2/V_1)`
4. 理想气体：`ΔS = nC_P ln(T_2/T_1) - nR ln(p_2/p_1)`

### 这一章最常考的 3 类题

#### 1. 物体和热库的熵变

核心思路：

1. 物体熵变沿可逆路径算
2. 热库熵变直接用 `-Q/T`
3. 两者相加看总熵变

例题结论：

`1 kg`、`0°C` 的水直接放到 `100°C` 热库上：

1. 水的熵变 `≈ 1305 J/K`
2. 热库熵变 `≈ -1122 J/K`
3. 总熵变 `≈ 183 J/K`

如果先放 `50°C` 再放 `100°C`：

总熵变变成 `≈ 97.4 J/K`

说明：

温差越小，越接近可逆，熵增越小。

#### 2. 理想气体经历不同过程的熵变

核心思路：

1. 等容：多用 `nC_V ln(T_2/T_1)`
2. 等温：多用 `nR ln(V_2/V_1)`
3. 可逆绝热：`ΔS = 0`

#### 3. 可逆循环

核心思路：

1. 分段判断过程
2. 分别求 `W`、`Q`、`ΔS`
3. 完整可逆循环总熵变为 0

例题结论：

某理想气体循环中：

1. `1→2` 等温膨胀：`W ≈ 1685 kJ`，`ΔS ≈ 5616 J/K`
2. `2→3` 等容降温：`W = 0`，`ΔS ≈ -5616 J/K`
3. `3→1` 绝热压缩：`W ≈ -1294 kJ`，`ΔS = 0`
4. 循环总功 `≈ 391 kJ`

### 第 11 章最容易错的点

1. 熵变公式中的温度必须用 K
2. 热库熵变别忘了负号
3. 物体熵变和总熵变不是一回事
4. 可逆绝热过程熵不变

## 六、第 6 章振动

### 这章到底在讲什么

核心只有三件事：

1. 振动表达式怎么写
2. 相位和初相怎么判断
3. 能量和周期怎么求

### 必背公式

1. `x = A cos(ωt + φ)`
2. `v = -Aω sin(ωt + φ)`
3. `a = -ω^2 x`
4. `ω = 2π/T`
5. `E = 1/2 kA^2`
6. 弹簧振子周期：`T = 2π sqrt(m/k)`

### 最常考题型

#### 1. 由图像求振动表达式

核心动作：

1. 先读振幅 `A`
2. 再由两次过平衡位置求周期 `T`
3. 再用 `x(0)` 和运动方向求初相 `φ`

例题结论：

若：

1. `A = 5 cm`
2. `x(0) = 2.5 cm`
3. `t = 1.0 s` 过平衡位置向负方向
4. `t = 2.2 s` 过平衡位置向正方向

则：

1. `T = 2.4 s`
2. `ω = 5π/6`
3. `φ = -π/3`
4. `x = 5 cos(5πt/6 - π/3) cm`

#### 2. 相差问题

若一个振子过平衡位置向负方向，另一个恰在正向端点，则后者相位比前者落后 `π/2`。

直接记：

`x_2 = A cos(ωt + φ - π/2)`

#### 3. 能量题

记住：

1. 总能量 `E = 1/2 kA^2`
2. 势能 `E_p = 1/2 kx^2`
3. 动能 `E_k = E - E_p`

例题：

若 `k = 25 N/m`，初始 `E_k = 0.2 J`，`E_p = 0.6 J`

则：

1. `E = 0.8 J`
2. `A ≈ 0.253 m`
3. `E_p = E_k` 时 `x ≈ ±0.179 m`

#### 4. 竖直弹簧振子

关键结论：

虽然有重力，但如果以平衡位置为原点，小振动方程还是：

`m d^2x/dt^2 + kx = 0`

周期仍是：

`T = 2π sqrt(m/k)`

总能量仍是：

`E = 1/2 kA^2`

### 第 6 章最容易错的点

1. 判断初相时，不能只看 `cosφ`，还要看速度方向
2. 竖直弹簧振子周期不因重力改变
3. 相位差和初相差不要混

## 七、第 7 章波动

### 这章到底在讲什么

这章主要就五件事：

1. 波函数怎么写
2. 波长、频率、波速关系
3. 驻波
4. 波峰位置
5. 多普勒效应

### 必背公式

1. `y = A cos(ωt - kx + φ)` 表示沿 `+x` 传播
2. `y = A cos(ωt + kx + φ)` 表示沿 `-x` 传播
3. `k = 2π/λ`
4. `ω = 2πf`
5. `v = λf = ω/k`
6. 驻波波节：`x = nλ/2`
7. 驻波波腹：`x = (2n+1)λ/4`

### 最常考题型

#### 1. 已知某点振动写波函数

例题：

若在 `x = 0.1 m` 处有

`y = 0.05 sin(1.0 - 4.0t)`

并且波速 `u = 0.8 m/s`

则：

1. `ω = 4`
2. `k = 5`
3. 波函数：

`y(x,t) = 0.05 sin(5x - 4t + 0.5)`

#### 2. 由图像写波函数

核心动作：

1. 读振幅 `A`
2. 读波长 `λ`
3. 用 `ω = kv`
4. 用原点初态求初相

#### 3. 波峰位置

若：

`y = A cos π(4t + 2x)`

波峰条件：

`4t + 2x = 2n`

所以：

`x = n - 2t`

当 `t = 4.2 s`：

`x = n - 8.4`

离原点最近的波峰在：

`x = -0.4 m`

#### 4. 驻波

入射波：

`y_1 = A sin(ωt - kx)`

反射波：

`y_2 = -A sin(ωt + kx)`

合成驻波：

`y = -2A sin(kx) cos(ωt)`

所以：

1. 波节：`sin(kx)=0`
2. 即 `x = nλ/2`

#### 5. 多普勒效应

这类题第一步不是列式，而是先判断：

1. 谁是声源
2. 谁是接收者
3. 谁在靠近谁

例题结论：

驱逐舰探潜问题中：

`f_2 = f_0 (v+v_s)/(v-v_s)`

最终潜艇速度：

`v_s ≈ 9.4 m/s`

### 第 7 章最容易错的点

1. `ωt - kx` 和 `ωt + kx` 方向别写反
2. 驻波不是行波
3. 波节位置和波腹位置别背反
4. 多普勒效应不要一上来硬套错公式

## 八、光学开头：相干、光强、双缝

### 这一块到底在讲什么

这部分其实只要你先搞懂四件事就够了：

1. 普通光源为什么不相干
2. 什么叫相干条件
3. 光强为什么不是简单振幅相加
4. 双缝干涉条纹怎么算

### 你必须会的概念

#### 1. 普通光源为什么通常不相干

因为：

1. 不同原子发光彼此独立
2. 同一个原子不同次发光也彼此独立

所以相位关系不稳定，不能长期保持固定相差。

#### 2. 激光为什么相干性好

因为受激辐射产生的是“全同光子”，相干性好，波列长也更长。

#### 3. 相干条件

两列光要发生稳定干涉，必须满足：

1. 频率相同
2. 振动有平行分量
3. 相差恒定

只要这三条里少一条，稳定干涉就没有。

#### 4. 光强公式

单色光平均光强和电场平方平均有关：

`I ∝ <E^2>_t`

两列相干光叠加：

`I = I_1 + I_2 + 2√(I_1 I_2) cosΔφ`

这里最后一项就是干涉项。

如果两列光不相干，则平均后干涉项为 0：

`I = I_1 + I_2`

#### 5. 单色光和准单色光

1. 理想单色光：只有单一频率
2. 准单色光：频率分布很窄，但不是绝对单频

考试里一般只要知道：

谱线越窄，越接近单色，越有利于稳定干涉。

### 双缝干涉必须会的结论

波程差：

`δ ≈ d x / D`

相位差：

`Δφ = 2πδ/λ`

明纹：

`δ = kλ`

暗纹：

`δ = (2k+1)λ/2`

条纹间距：

`Δx = λD/d`

### 光学这一块最容易错的点

1. 相干不只是“频率差不多”就够
2. 频率不同通常不形成稳定干涉条纹
3. 干涉条纹公式里 `D` 在上，`d` 在下
