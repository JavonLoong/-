# 布儒斯特角为什么会让反射光完全偏振

这份只解释一个问题：

```text
为什么反射光和折射光互相垂直时，反射光会变成完全偏振光？
```

结论先放前面：

```text
不是“垂直”本身产生了偏振。
而是在这个角度下，平行于入射面的那一部分电场反射不出来。
所以反射光只剩下垂直于入射面的偏振分量。
```

---

## 1. 先分清两个偏振方向

光打到界面时，要把电场分成两类：

```text
s 偏振：电场垂直于入射面
p 偏振：电场平行于入射面
```

入射面就是：

```text
入射光线 + 法线 共同组成的平面
```

自然光里这两种分量都有。

普通角度反射时：

```text
s 分量会反射
p 分量也会反射
```

所以反射光只是部分偏振。

但在布儒斯特角时：

```text
p 分量的反射变成 0
```

于是反射光只剩 `s` 分量，所以变成完全线偏振光。

---

## 2. 为什么 p 分量在布儒斯特角反射不出来

可以用一个很直观的模型理解：

```text
光进入介质后，会驱动介质中的电荷振动。
这些振动的电荷会重新辐射电磁波。
反射光可以看成这些电荷重新辐射出来的一部分。
```

关键规律：

```text
振动电偶极子不会沿着自己的振动方向辐射电磁波。
```

也就是：

```text
电荷沿某个方向来回振动
在这个振动方向的正前方和正后方，辐射强度为 0
```

这不是大学物理考试通常要求推的内容，但它能解释为什么布儒斯特角会发生。

---

## 3. 把这个规律放到布儒斯特角里

在布儒斯特角时：

```text
反射光方向 ⟂ 折射光方向
```

也就是：

```text
i_B + r = 90°
```

对于 `p` 偏振分量来说，它的电场在入射面内，并且电场方向总是垂直于光的传播方向。

折射光进入介质后，介质中的电荷主要沿这个 `p` 分量的电场方向振动。

而在布儒斯特角的几何关系下，这个振动方向刚好指向反射光方向。

但是：

```text
振动电偶极子不会沿自己的振动方向辐射
```

所以：

```text
p 偏振分量不能沿反射光方向辐射出来
```

结果：

```text
p 分量的反射光消失
```

反射光就只剩下 `s` 偏振分量。

这就是“反射光完全偏振”的真正意思。

---

## 4. 公式上怎么体现

更正式一点，反射光里 `p` 偏振分量的反射系数是：

```text
r_p = (n2 cos i - n1 cos r) / (n2 cos i + n1 cos r)
```

当：

```text
r_p = 0
```

就表示：

```text
p 偏振分量没有反射
```

所以：

```text
n2 cos i = n1 cos r
```

再结合折射定律：

```text
n1 sin i = n2 sin r
```

可以推出：

```text
i + r = 90°
```

也能推出：

```text
tan i_B = n2 / n1
```

所以公式上看：

```text
布儒斯特角就是 p 偏振反射系数为 0 的角度。
```

---

## 5. 最容易误会的地方

### 误会 1：是不是因为两条光线垂直，所以光自己变偏振？

不是。

更准确地说：

```text
两条光线垂直，是 p 分量反射为 0 时对应的几何结果。
```

### 误会 2：是不是所有偏振都消失了？

不是。

消失的是：

```text
p 偏振反射分量
```

还剩下：

```text
s 偏振反射分量
```

所以反射光不是没了，而是变成了单一偏振方向。

### 误会 3：完全偏振是不是指传播方向改变？

不是。

偏振说的是：

```text
电场振动方向
```

不是：

```text
光线传播方向
```

---

## 6. 考试怎么写

看到“反射光完全偏振”“布儒斯特角”，直接写：

```text
tan i_B = n2 / n1
```

并且：

```text
i_B + r = 90°
```

如果问原因，写：

```text
在布儒斯特角，p 偏振分量的反射系数为 0，
因此反射光只剩 s 偏振分量，所以反射光为完全线偏振光。
```

更直观一点可以写：

```text
介质中被激发的电偶极子不沿自身振动方向辐射。
布儒斯特角时，p 分量对应的振动方向与反射方向一致，
所以 p 分量不能形成反射光。
```

---

## 7. 一句话背法

```text
布儒斯特角不是把光“偏开”，而是让 p 偏振反射为零；
反射光只剩 s 偏振，所以完全偏振。
```
