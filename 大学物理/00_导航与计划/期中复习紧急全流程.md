## 六、第 3 轮怎么背

这一步只盯你那张总纸。

### 必须默写的热学核心

1. `pV = nRT`
2. `U = i/2 · nRT`
3. `Q = ΔU + W`
4. `ΔU = nC_VΔT`
5. `pV^γ = 常量`
6. `TV^(γ-1) = 常量`
7. `ΔS = ∫ dQ_rev/T`
8. `ΔS = nC_V ln(T_2/T_1) + nR ln(V_2/V_1)`

### 必须默写的振动波动核心

1. `x = A cos(ωt + φ)`
2. `ω = 2π/T`
3. `E = 1/2 kA^2`
4. `y = A cos(ωt ∓ kx + φ)`
5. `v = λf`
6. 驻波波节：`x = nλ/2`

### 必须说得出口的光学概念

1. 相干条件
2. 普通光源不相干的原因
3. 单色与准单色
4. 光强叠加不是振幅简单叠加

## 七、第 4 轮只回看这些高危点

### 热学高危点

1. 双原子气体 `C_V = 5R/2`
2. 单原子气体 `C_V = 3R/2`
3. 摄氏度不能直接代热力学公式
4. `L` 要换成 `m^3`
5. 系统对外做功为正还是负，必须统一

### 振动波动高危点

1. 初相正负容易错
2. 波函数中 `ωt - kx` 和 `ωt + kx` 方向别写反
3. 驻波和行波不要混
4. 多普勒效应先判断谁是声源谁是观察者

### 光学高危点

1. 相干不等于频率接近就一定行
2. 频率不同通常不会形成稳定干涉条纹
3. 光强和电场平方平均有关
