## 6.3 已知振动曲线求相位、振动表达式和相量图

题图可直接读出：

- 振幅 `A = 5.0 cm`
- 在 `t = 0` 时，`x(0) = 2.5 cm`
- 在 `t = 1.0 s` 时，振子过平衡位置且向负方向运动
- 在 `t = 2.2 s` 时，再次过平衡位置且向正方向运动

后两次过平衡位置的方向相反，因此它们相隔半个周期：

`T / 2 = 2.2 - 1.0 = 1.2 s`

所以

`T = 2.4 s`

`ω = 2π / T = 2π / 2.4 = 5π / 6 rad/s`

设振动表达式为

`x = A cos(ωt + φ)`

代入 `t = 0` 时的条件：

`2.5 = 5 cos φ`

`cos φ = 1 / 2`

又因为图像在 `t = 0` 时向上运动，所以 `v(0) > 0`。

而

`v = dx/dt = -Aω sin(ωt + φ)`

故 `v(0) > 0` 等价于 `sin φ < 0`。

满足 `cos φ = 1/2` 且 `sin φ < 0` 的相位为

`φ = -π / 3`

因此振动表达式为

`x = 5 cos(5πt / 6 - π / 3) cm`

### （1）各状态对应的相

这里“相”取瞬时相位 `θ = ωt + φ`。

- 点 `a`：位于正最大位移处，`θ_a = 0`
- 点 `b`：`x = 2.5 cm` 且向下运动，`cos θ = 1/2`、`sin θ > 0`，故 `θ_b = π/3`
- 点 `c`：过平衡位置且向下运动，`θ_c = π/2`
- 点 `d`：`x = -2.5 cm` 且向下运动，`cos θ = -1/2`、`sin θ > 0`，故 `θ_d = 2π/3`
- 点 `e`：`x = -2.5 cm` 且向上运动，`cos θ = -1/2`、`sin θ < 0`，故 `θ_e = 4π/3`

以上相位都可再加上 `2kπ`。

### （2）振动表达式

`x = 5 cos(5πt / 6 - π / 3) cm`

### （3）相量图

按余弦表示法，相量长度为 `5 cm`，相量以角速度 `ω = 5π/6 rad/s` 逆时针转动。

对应各点的相量方位：

- `a`：与 `+x` 轴重合
- `b`：与 `+x` 轴夹角 `π/3`
- `c`：与 `+x` 轴夹角 `π/2`
- `d`：与 `+x` 轴夹角 `2π/3`
- `e`：与 `+x` 轴夹角 `4π/3`

## 6.6 两个同频率、同振幅简谐振动的相差

已知第一个振子的表达式为

`x_1 = A cos(ωt + φ)`

题意说：当第一个振子“从振动的正方向回到平衡位置”时，第二个振子恰在正方向位移端点。

第一个振子回到平衡位置且向负方向运动时：

- `x_1 = 0`
- `v_1 < 0`

因此它的相位应为

`ωt + φ = π/2 + 2kπ`

同一时刻，第二个振子在正向端点，即

`x_2 = +A`

所以第二个振子的相位应为

`ωt + φ_2 = 2kπ`

于是有

`φ_2 - φ = -π/2`

即第二个振子比第一个振子落后 `π/2`，或者说第一个振子超前第二个振子 `π/2`。

故第二个振子的振动表达式可写为

`x_2 = A cos(ωt + φ - π/2)`

也可写成

`x_2 = A sin(ωt + φ)`

### （1）答案

- 第二个振子的表达式：`x_2 = A cos(ωt + φ - π/2)`
- 相差：`Δφ = φ_2 - φ_1 = -π/2`

### （2）若 `t = 0` 时，`x_1 = -A/2`，并向 `x` 负方向运动

先求第一个振子的初相。

由

`x_1(0) = A cos φ = -A/2`

得

`cos φ = -1/2`

又因为 `v_1(0) < 0`，即

`-Aω sin φ < 0`

所以 `sin φ > 0`

于是

`φ = 2π/3`

因此

`x_1 = A cos(ωt + 2π/3)`

第二个振子相位比它小 `π/2`，故

`x_2 = A cos(ωt + 2π/3 - π/2) = A cos(ωt + π/6)`

所以本问最终结果为：

- `x_1 = A cos(ωt + 2π/3)`
- `x_2 = A cos(ωt + π/6)`

相量图上两相量长度相同，`x_1` 的相量比 `x_2` 超前 `π/2`。

## 6.8 弹簧振子的振幅、能量关系

已知：

- `k = 25 N/m`
- 初动能 `E_k = 0.2 J`
- 初势能 `E_p = 0.6 J`

总机械能

`E = E_k + E_p = 0.8 J`

而简谐振动总能量为

`E = 1/2 kA^2`

### （1）振幅

`1/2 × 25 × A^2 = 0.8`

`A^2 = 1.6 / 25 = 0.064`

`A = sqrt(0.064) ≈ 0.253 m`

所以

`A ≈ 0.253 m`

### （2）位移多大时，势能和动能相等

当势能和动能相等时，各占总能量的一半，即

`E_p = E_k = 0.4 J`

而

`E_p = 1/2 kx^2`

故

`1/2 × 25 × x^2 = 0.4`

`x^2 = 0.8 / 25 = 0.032`

`x = ± sqrt(0.032) ≈ ± 0.179 m`

所以

`x = ± 0.179 m`

### （3）位移是振幅的一半时，势能多大

当 `x = A/2` 时，

`E_p = 1/2 k(A/2)^2 = 1/8 kA^2`

而 `1/2 kA^2 = 0.8 J`

所以

`E_p = 0.8 / 4 = 0.2 J`

故

`E_p = 0.2 J`

## 6.9 竖直弹簧振子的动力学方程、周期与总能量

设小球平衡时弹簧伸长为 `b`，并以平衡位置为原点，取竖直向下为正方向，设小球相对平衡位置的位移为 `x`。

则任意时刻弹簧的总伸长量为

`b + x`

小球受力：

- 重力 `mg`，方向向下
- 弹簧弹力 `k(b + x)`，方向向上

由牛顿第二定律：

`m d^2x/dt^2 = mg - k(b + x)`

平衡时有

`kb = mg`

代入上式得

`m d^2x/dt^2 = kb - k(b + x) = -kx`

所以动力学方程为

`m d^2x/dt^2 + kx = 0`

这正是简谐振动方程。

故角频率为

`ω = sqrt(k/m)`

周期为

`T = 2π sqrt(m/k)`

### 总能量是否还是 `1/2 kA^2`

题目要求总能量包括：

- 动能
- 重力势能
- 弹簧弹性势能

且都以平衡位置为势能零点。

重力势能取平衡位置为零点时：

`U_g = -mgx`

弹簧弹性势能相对平衡位置的变化量为

`U_s = 1/2 k(b + x)^2 - 1/2 kb^2`

展开：

`U_s = kbx + 1/2 kx^2`

又因为 `kb = mg`

所以总势能

`U = U_g + U_s = -mgx + mgx + 1/2 kx^2 = 1/2 kx^2`

于是总机械能为

`E = 1/2 m(dx/dt)^2 + 1/2 kx^2`

当振幅为 `A` 时，在端点速度为零，故

`E = 1/2 kA^2`

因此答案是：**仍然是**

`E = 1/2 kA^2`

## 6.11 两根轻弹簧串联后的振动周期

两弹簧劲度系数分别为 `k_1`、`k_2`，串联后等效劲度系数 `k_eq` 满足

`1 / k_eq = 1 / k_1 + 1 / k_2`

所以

`k_eq = k_1 k_2 / (k_1 + k_2)`

该系统在竖直方向的小振动，仍可视为以平衡位置为中心的简谐振动，其周期为

`T = 2π sqrt(m / k_eq)`

代入 `k_eq`：

`T = 2π sqrt(m (k_1 + k_2) / (k_1 k_2))`

故答案为

`T = 2π sqrt(m (k_1 + k_2) / (k_1 k_2))`

## 6.25 三个同方向、同频率简谐振动的合成

已知

`x_1 = 0.08 cos(314t + π/6)`

`x_2 = 0.08 cos(314t + π/2)`

`x_3 = 0.08 cos(314t + 5π/6)`

### （1）求合振动的角频率、振幅、初相及表达式

三个振动同频率，所以合振动角频率不变：

`ω = 314 rad/s`

用相量法。三个相量分别为

- `0.08 ∠ π/6`
- `0.08 ∠ π/2`
- `0.08 ∠ 5π/6`

先求横、纵分量之和：

横分量：

`0.08(cos π/6 + cos π/2 + cos 5π/6)`

`= 0.08(√3/2 + 0 - √3/2) = 0`

纵分量：

`0.08(sin π/6 + sin π/2 + sin 5π/6)`

`= 0.08(1/2 + 1 + 1/2) = 0.16`

所以合相量大小为

`A = 0.16`

方向沿正虚轴，即初相为

`φ = π/2`

故合振动表达式为

`x = 0.16 cos(314t + π/2)`

也可写成

`x = -0.16 sin(314t)`

### （2）由初始位置运动到 `x = (√2 / 2) A` 所需最短时间

由合振动表达式

`x = 0.16 cos(314t + π/2)`

当 `t = 0` 时，

`x(0) = 0`

这就是题目说的“初始位置”。

要求到达

`x = (√2 / 2)A`

即

`cos(314t + π/2) = √2 / 2`

更直接地写：

`x = -A sin(314t)`

所以

`-sin(314t) = √2 / 2`

`sin(314t) = -√2 / 2`

最小正解为

`314t = 5π/4`

因此

`t = 5π / (4 × 314) s`

数值约为

`t ≈ 0.0125 s`

> 注：若题目要求的是第一次达到 `|x| = (√2/2)A`，那时间应为 `T/8`。但题面写的是 `x = +(√2/2)A`，所以应取上面的正位移结果。

## 6.26 两个相互垂直简谐振动的合成轨迹

已知

`x = 0.06 cos(20πt)`

`y = 0.04 cos(20πt + π/2)`

由

`cos(θ + π/2) = -sin θ`

得

`y = -0.04 sin(20πt)`

于是

`x / 0.06 = cos(20πt)`

`y / 0.04 = -sin(20πt)`

两边平方相加：

`(x / 0.06)^2 + (y / 0.04)^2 = cos^2(20πt) + sin^2(20πt) = 1`

故轨迹方程为

`x^2 / 0.06^2 + y^2 / 0.04^2 = 1`

这是一条正椭圆，且长短轴分别沿坐标轴。

### 半轴长度

- 长半轴：`a = 0.06`
- 短半轴：`b = 0.04`

### 绕行周期

两个分运动角频率相同，均为

`ω = 20π rad/s`

所以轨迹重复的周期为

`T = 2π / ω = 2π / 20π = 0.1 s`

### 绕行方向

当 `t = 0` 时，

`x = 0.06`

`y = 0`

质点位于椭圆最右端。

再看初速度方向：

`dx/dt = -0.06 × 20π sin(20πt)`

故 `t = 0` 时 `dx/dt = 0`

`dy/dt = -0.04 × 20π cos(20πt)`

故 `t = 0` 时

`dy/dt = -0.8π < 0`

说明质点从最右端开始向下运动，因此绕行方向为**顺时针**，即**右旋**。

## 最终答案汇总

- `6.3`：`x = 5 cos(5πt/6 - π/3) cm`
- `6.6`：`x_2 = A cos(ωt + φ - π/2)`；若给定初态，则 `x_1 = A cos(ωt + 2π/3)`，`x_2 = A cos(ωt + π/6)`
- `6.8`：`A ≈ 0.253 m`，`x = ±0.179 m`，`x = A/2` 时 `E_p = 0.2 J`
- `6.9`：动力学方程 `m d^2x/dt^2 + kx = 0`，周期 `T = 2π sqrt(m/k)`，总能量仍为 `1/2 kA^2`
- `6.11`：`T = 2π sqrt(m (k_1 + k_2) / (k_1 k_2))`
- `6.25`：合振动 `x = 0.16 cos(314t + π/2)`，到达 `x = (√2/2)A` 的最短时间 `t = 5π/(4×314) s ≈ 0.0125 s`
- `6.26`：轨迹 `x^2/0.06^2 + y^2/0.04^2 = 1`，长半轴 `0.06`，短半轴 `0.04`，周期 `0.1 s`，顺时针
