## 7.2 由已知质点振动写波函数

已知：简谐横波沿一长弦线以 `u = 0.8 m/s` 传播。在 `x = 0.1 m` 处，弦线质点的位移随时间的变化关系为

`y = 0.05 sin(1.0 − 4.0t)`

即

`y(x = 0.1, t) = 0.05 sin(1.0 − 4.0t)`

### 求波函数

先确定角频率和波数。由已知振动表达式可读出：

`ω = 4.0 rad/s`

波速 `u = 0.8 m/s`，故波数为

`k = ω / u = 4.0 / 0.8 = 5.0 rad/m`

波函数的一般形式为

`y(x, t) = A sin(ωt − kx + φ₀)` （沿 +x 方向传播）

或

`y(x, t) = A sin(−ωt + kx + φ₀')`

注意题目给的形式是 `sin(1.0 − 4.0t) = sin(−4.0t + 1.0)`，所以用

`y(x, t) = 0.05 sin(−4.0t + kx + C)`

代入 `x = 0.1`：

`y = 0.05 sin(−4.0t + 5.0 × 0.1 + C) = 0.05 sin(−4.0t + 0.5 + C)`

与已知 `y = 0.05 sin(−4.0t + 1.0)` 对比：

`0.5 + C = 1.0`

`C = 0.5`

因此波函数为

**`y(x, t) = 0.05 sin(5.0x − 4.0t + 0.5)` (m)**

也可以写成

`y(x, t) = 0.05 sin(1.0 − 4.0t + 5.0(x − 0.1))` (m)

> 注：需要判断波的传播方向。由表达式 `sin(kx − ωt + C)` 可知波沿 +x 方向传播。

## 7.5 由波形图写波函数并画 t = T/8 时的波形

已知：平面简谐波在 `t = 0` 时的波形曲线如图 7.44 所示，波沿 +x 方向传播，`u = 0.08 m/s`。

从图中读出：

- 振幅 `A = 0.04 m`
- 波长 `λ = 0.4 m`（一个完整周期对应 0.4 m）

由此得

`k = 2π / λ = 2π / 0.4 = 5π rad/m`

`ω = ku = 5π × 0.08 = 0.4π rad/s`

周期 `T = 2π / ω = 2π / (0.4π) = 5 s`

### （1）写出波函数

在 `t = 0` 时，原点处 `y(0, 0) = 0`。

从图看，波沿 +x 方向传播，在 `t = 0` 时原点处 `y = 0`。

观察波形图：在 `x = 0` 处 `y = 0`，紧右方 `y < 0`，这说明在 `t = 0` 时原点右侧的质点位移为负。

对于沿 +x 方向传播的波，`y = A cos(ωt − kx + φ₀)`

在 `x = 0, t = 0`：`y = A cos φ₀ = 0`

又因为波沿 +x 传播时，`x = 0` 处的质点将要向 +y 方向运动（图中原点左侧的波形正半部分将移过来），即 `∂y/∂t|₍ₜ₌₀, ₓ₌₀₎ > 0`。

`∂y/∂t = −Aω sin(ωt − kx + φ₀)`

在 `t = 0, x = 0`：`−Aω sin φ₀ > 0`，即 `sin φ₀ < 0`

`cos φ₀ = 0` 且 `sin φ₀ < 0`，故 `φ₀ = −π/2`

因此波函数为

**`y = 0.04 cos(0.4πt − 5πx − π/2)` (m)**

即

**`y = 0.04 sin(0.4πt − 5πx)` (m)**

### （2）画 t = T/8 时的波形

` = T/8 = 5/8 s` 时，波形整体沿 +x 方向移动 `uT/8 = 0.08 × 5/8 = 0.05 m = λ/8`

即波形图向右平移 `λ/8 = 0.05 m`。

或直接代入：

`y(x, T/8) = 0.04 sin(0.4π × 5/8 − 5πx) = 0.04 sin(π/4 − 5πx)`

各特征点：
- `x = 0`：`y = 0.04 sin(π/4) = 0.04 × √2/2 ≈ 0.0283 m`
- `x = 0.05`：`y = 0.04 sin(π/4 − π/4) = 0`
- `x = 0.15`：`y = 0.04 sin(π/4 − 3π/4) = 0.04 sin(−π/2) = −0.04 m`
- `x = 0.35`：`y = 0.04 sin(π/4 − 7π/4) = 0.04 sin(−3π/2) = 0.04 m`

波形图整体右移 0.05 m。

## 7.6 由波函数求波峰位置

已知波函数为

`y = A cos π(4t + 2x)`

### （1）t = 4.2 s 时各波峰位置，距原点最近的波峰，何时通过原点？

波峰条件：`cos π(4t + 2x) = 1`，即

`π(4t + 2x) = 2nπ` （n 为整数）

`4t + 2x = 2n`

`x = n − 2t`

当 `t = 4.2 s` 时：

`x = n − 2 × 4.2 = n − 8.4`

各波峰坐标为

**`x = n − 8.4` (m)，n = 0, ±1, ±2, ...`**

即 `x = ..., −10.4, −9.4, −8.4, −7.4, −6.4, ...`

距原点最近的波峰：`n = 8` 时 `x = −0.4 m`，`n = 9` 时 `x = 0.6 m`。

两者中 `|−0.4| = 0.4 < 0.6`，故**距原点最近的波峰在 `x = −0.4 m`**。

该波峰通过原点时 `x = 0`：

`0 = n − 2t`

对于 `n = 8` 这个波峰：`t = 8/2 = 4.0 s`

所以该波峰已在 `t = 4.0 s` 时通过原点。

> 注：波函数 `cos π(4t + 2x)` 表示波沿 −x 方向传播（因为 `ωt` 和 `kx` 符号相同）。
>
> `ω = 4π rad/s`，`k = 2π rad/m`
>
> `u = ω/k = 2 m/s`（沿 −x 方向）
>
> `λ = 2π/k = 1 m`，`T = 2π/ω = 0.5 s`

### （2）画 t = 4.2 s 时的波形曲线

`y(x, 4.2) = A cos π(16.8 + 2x) = A cos(16.8π + 2πx)`

由于 `16.8π = 8 × 2π + 0.8π`，所以

`y = A cos(2πx + 0.8π)`

这是一条余弦曲线，在 `x = −0.4` 处取最大值，波长 `λ = 1 m`。

## 7.10 钢轨中声波的强度和功率

已知：
- 钢轨中声速 `v = 5.1 × 10³ m/s`
- 某处振幅 `A = 1 × 10⁻⁹ m`
- 频率 `ν = 1 × 10³ Hz`
- 钢的密度 `ρ = 7.9 × 10³ kg/m³`
- 钢轨截面积 `S = 15 cm² = 15 × 10⁻⁴ m²`

### （1）声波强度

声波强度公式：

`I = ½ ρ v ω² A²`

其中 `ω = 2πν = 2π × 10³ rad/s`

`I = ½ × 7.9 × 10³ × 5.1 × 10³ × (2π × 10³)² × (10⁻⁹)²`

`= ½ × 7.9 × 10³ × 5.1 × 10³ × 4π² × 10⁶ × 10⁻¹⁸`

`= ½ × 7.9 × 5.1 × 4π² × 10³⁺³⁺⁶⁻¹⁸`

`= ½ × 7.9 × 5.1 × 39.48 × 10⁻⁶`

`= ½ × 1590.5 × 10⁻⁶`

`= 7.95 × 10⁻⁴ W/m²`

**`I ≈ 7.95 × 10⁻⁴ W/m²`**

### （2）通过钢轨输送的功率

`P = I × S = 7.95 × 10⁻⁴ × 15 × 10⁻⁴`

**`P ≈ 1.19 × 10⁻⁶ W ≈ 1.2 μW`**

## 7.14 平面简谐波的波函数与驻波

已知：平面简谐波沿 +x 方向传播，振幅 `A`，频率 `ν`，波速 `u`。

`t = 0` 时，原点 O 处的质元由平衡位置向 +x 轴正方向运动。反射面在 `x = 3λ/4` 处（P 点右方）。

### （1）写出入射波的波函数

`ω = 2πν`，`k = 2π/λ = ω/u`

在 `t = 0, x = 0`：质元在平衡位置且向正方向运动。

设入射波 `y₁ = A cos(ωt − kx + φ₀)`

`y₁(0, 0) = A cos φ₀ = 0`

速度：`∂y₁/∂t = −Aω sin(ωt − kx + φ₀)`

在 `t = 0, x = 0`：`−Aω sin φ₀ > 0`，即 `sin φ₀ < 0`

`cos φ₀ = 0, sin φ₀ < 0` → `φ₀ = −π/2`

**入射波：`y₁ = A cos(ωt − kx − π/2) = A sin(ωt − kx)`**

即 **`y₁ = A sin(2πνt − 2πx/λ)`**

### （2）反射波与驻波中的波节位置

在反射面（波密介质，即固定端反射）处有半波损失。设反射面在 `x = 3λ/4`。

反射波沿 −x 方向传播，且在反射面处与入射波的位移之和为零（固定端）。

反射波可写为

`y₂ = −A sin(ωt + kx − 2k × 3λ/4)`

`= −A sin(ωt + kx − 3π)`

`= −A sin(ωt + kx − 3π)`

`= −A × [−sin(ωt + kx − π)]`

`= A sin(ωt + kx − π)`

`= −A sin(ωt + kx)`

> 更严谨的推导：在 `x = 3λ/4` 处，`y₁ + y₂ = 0` 对所有 `t` 成立。
>
> `y₁(3λ/4, t) = A sin(ωt − k × 3λ/4) = A sin(ωt − 3π/2)`
>
> 要使 `y₂(3λ/4, t) = −y₁(3λ/4, t) = −A sin(ωt − 3π/2)`
>
> 反射波形式 `y₂ = B sin(ωt + kx + φ)`，且 `|B| = A`
>
> 在反射面：`A sin(ωt + 3π/2 + φ) = −A sin(ωt − 3π/2)`
>
> `sin(ωt + 3π/2 + φ) = −sin(ωt − 3π/2) = sin(−ωt + 3π/2)`
>
> 这不是恒等关系，重新整理。
>
> `−sin(ωt − 3π/2) = −[−cos ωt] = cos ωt`
>
> `sin(ωt + 3π/2 + φ) = cos ωt = sin(ωt + π/2)`
>
> 所以 `3π/2 + φ = π/2 + 2nπ`，即 `φ = −π + 2nπ`
>
> 取 `φ = −π`

**反射波：`y₂ = A sin(ωt + kx − π) = −A sin(ωt + kx)`**

合成驻波：

`y = y₁ + y₂ = A sin(ωt − kx) − A sin(ωt + kx)`

利用和差化积：

`y = A [sin(ωt − kx) − sin(ωt + kx)]`

`= A × 2 cos ωt × (−sin kx)`

`= −2A sin(kx) cos(ωt)`

驻波的**波节**（振幅为零）位置：

`sin(kx) = 0`

`kx = nπ`

`x = nλ/2`，n = 0, 1, 2, ...

在 `0 ≤ x ≤ 3λ/4` 范围内：

**波节位于 `x = 0, λ/2`**

驻波的**波腹**位置：

`sin(kx) = ±1`

`kx = (n + ½)π`

`x = (2n + 1)λ/4`

在范围内：**波腹位于 `x = λ/4, 3λ/4`**

## 7.15 压电石英晶片的驻波振动

已知：
- 晶片厚度 `d = 2.00 mm = 2.00 × 10⁻³ m`
- 石英片中声速 `v = 5.74 × 10³ m/s`
- 两表面自由 → 波腹
- 要求基频振动（两个波腹之间半个波长）

基频驻波条件：两个自由端都是波腹，基频时中间恰好一个波节。

`d = λ/2`

`λ = 2d = 4.00 × 10⁻³ m`

基频频率

`f = v / λ = 5.74 × 10³ / (4.00 × 10⁻³)`

**`f = 1.435 × 10⁶ Hz ≈ 1.44 MHz`**

## 7.22 多普勒效应——驱逐舰探潜

已知：
- 驱逐舰静止
- 发射超声波频率 `f₀ = 1.8 × 10⁴ Hz`
- 反射回来的频率与发射频率之差 `Δf = 220 Hz`
- 海水中声速 `v = 1.54 × 10³ m/s`
- 潜艇驶向驱逐舰

设潜艇速度为 `v_s`（朝向驱逐舰方向为正）。

超声波过程分两步：
1. 驱逐舰（静止声源）发射，潜艇（运动接收者）接收，接收频率为 `f₁`
2. 潜艇相当于运动声源将该频率反射回去，驱逐舰（静止接收者）接收频率为 `f₂`

第一步（潜艇迎向声波）：

`f₁ = f₀ × (v + v_s) / v`

第二步（潜艇作为运动声源，驱逐舰接收）：

`f₂ = f₁ × v / (v − v_s) = f₀ × (v + v_s) / (v − v_s)`

频差：

`Δf = f₂ − f₀ = f₀ [(v + v_s)/(v − v_s) − 1] = f₀ × 2v_s / (v − v_s)`

代入数值：

`220 = 1.8 × 10⁴ × 2v_s / (1.54 × 10³ − v_s)`

由于 `v_s ≪ v`，先近似估计：

`220 ≈ 1.8 × 10⁴ × 2v_s / 1540`

`v_s ≈ 220 × 1540 / (2 × 1.8 × 10⁴) = 338800 / 36000 ≈ 9.41 m/s`

精确求解：

`220(1540 − v_s) = 36000 v_s`

`220 × 1540 = 36000 v_s + 220 v_s = 36220 v_s`

`v_s = 338800 / 36220 ≈ 9.354 m/s`

**`v_s ≈ 9.35 m/s ≈ 9.4 m/s`**
