# 学习成果归档

这个仓库用于整理学习过程中后期加工出来的成果，而不是上传原始教材或课程资料。

## 核心内容

- `大学物理/`：大学物理复习讲义、AI Learning OS 章节包、HTML 图文页面与配套示意图。
- `固体力学/`：材料力学/固体力学期末复习页面、速查表、冲刺清单与生成脚本。
- `流体力学/`：流体力学期末复习 AI Learning OS 页面、复习材料目录与整理成果。
- `AI_Learning_OS/`：用于生成和组织学习材料的提示词、模板与课程适配器。
- 其他目录：保留实验整理、探究性小项目、生成图表、脚本和说明文档等明显属于后期加工的学习产物。

## 未纳入内容

仓库通过 `.gitignore` 排除了原始教材 PDF、Word/PPT 课件、压缩包、可执行程序、虚拟环境、依赖缓存和批量 PDF 页面截图。

这样做是为了让公开仓库保持清爽、可浏览，也避免把不适合公开发布的原始参考资料推上去。
