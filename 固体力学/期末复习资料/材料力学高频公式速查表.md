# 材料力学高频公式速查表

适用方式：考前最后两天快翻，或者做题卡壳时迅速定位。

## 1. 内力与内力图

- 截面法：先取分离体，再列平衡，求 `N / Q / M / T`
- 梁的微分关系：`dQ/dx = -q(x)`，`dM/dx = Q(x)`
- 集中力：`Q` 跳跃
- 集中力偶：`M` 跳跃
- `Q = 0` 常对应 `M` 极值点

## 2. 拉压、温度、薄壁圆筒

- 正应力：`σ = N / A`
- 正应变：`ε = Δl / l`
- 轴向变形：`Δl = ∫ N/(EA) dx`
- 温度应变：`ε_T = αΔT`
- 自由热伸长：`Δl_T = αΔTl`
- 闭口薄壁圆筒：
  - 环向应力：`σ_θ = pD/(2t)`
  - 轴向应力：`σ_z = pD/(4t)`

## 3. 扭转、剪切、连接件

- 圆轴扭转剪应力：`τ = Tr/J_p`
- 最大扭转剪应力：`τ_max = TR/J_p`
- 扭转角：`φ = ∫ T/(GJ_p) dx`
- 实心圆：`J_p = πd^4/32`
- 空心圆：`J_p = π(D^4-d^4)/32`
- 梁弯曲剪应力：`τ = VQ/(Ib)`
- 连接件平均剪应力：`τ_avg = F/A_shear`
- 挤压应力：`σ_bs = F/A_bearing`

## 4. 弯曲、截面、组合应力

- 纯弯曲正应力：`σ = My/I`
- 组合受力：`σ = N/A ± My/I`
- 平行轴定理：`I = I_c + Ad^2`
- 弯曲强度条件：`|σ|max ≤ [σ]`

## 5. 应力状态、莫尔圆、强度理论

- 平均应力：`σ̄ = (σx + σy)/2`
- 莫尔圆半径：`R = sqrt(((σx-σy)/2)^2 + τxy^2)`
- 主应力：`σ1,2 = σ̄ ± R`
- 最大剪应力：`τmax = R`
- 广义胡克定律：
  - `εx = [σx - ν(σy + σz)] / E`
  - `γxy = τxy / G`
- 模量关系：`G = E / [2(1 + ν)]`
- Tresca：`σ_eq^T = max(|σ1-σ2|, |σ2-σ3|, |σ3-σ1|)`
- von Mises：`σ_eq^M = sqrt(((σ1-σ2)^2 + (σ2-σ3)^2 + (σ3-σ1)^2)/2)`

## 6. 位移、能量法、卡氏定理

- 梁挠曲微分方程：`EI v''(x) = M(x)`
- 总应变能：
  - `U = ∫ N^2/(2EA) dx + ∫ T^2/(2GJ) dx + ∫ M^2/(2EI) dx`
- 卡氏第二定理：
  - 位移：`δ_i = ∂U/∂P_i`
  - 转角：`θ_i = ∂U/∂M_i`

## 7. 静不定题骨架

- 静不定次数 = 未知力个数 - 独立平衡方程数
- 标准流程：
  1. 判静不定次数
  2. 选冗余未知量
  3. 建基本静定系
  4. 写变形协调
  5. 写位移或应变能表达式
  6. 联立求冗余量

## 8. 稳定

- 欧拉临界力：`P_cr = π^2EI/(μl)^2`
- 柔度：`λ = μl/i`
- 欧拉临界应力：`σ_cr = π^2E/λ^2`
- 常见 `μ`：
  - 两端铰支：`1`
  - 一端固定一端自由：`2`
  - 一端固定一端铰支：`约 0.7`
  - 两端固定：`0.5`

## 9. 复合梁 / 温差题最该先想的事

- 完美粘合：界面无滑移，截面转角与曲率协调
- 无粘合：各层可相对滑移，不能直接把整体当作一个完整截面
- 温差题：先判断是否允许自由热胀冷缩
- 装配题：先画装配前后几何关系，再列协调

## 10. 考场提醒

- 先图后式：先画受力图、变形图、应力状态图
- 先结构后公式：先判断题目是哪一类，再选公式
- 先约束后结果：静不定、温差、弹簧、组合梁都要先看约束
