1. 课程主线图：这门课到底在干什么

你可以把整门材料力学理解成一句话：已知外力和约束，研究杆件内部怎么传力、怎么变形、会不会失效，并据此做强度、刚度、稳定性判断。

整门课的逻辑骨架

你最先要稳住的三个词

强度：会不会坏。

刚度：会不会变形过大，虽然没坏但已经不能正常工作。

稳定：会不会突然失稳，明明应力没超限却先“歪掉”。

全课最常见的三条链

平衡：外力与内力关系。

几何：变形协调、位移关系。

物理：应力应变关系、胡克定律、材料本构。

考试里最容易忘的一点

题目不是在考你“记不记得公式”，而是在考你能不能把 平衡 + 协调 + 本构 拼起来。尤其静不定、温度、组合材料、弹簧梁这些题，单靠背公式基本拿不到满分。

最常见的失分方式

1. 只列平衡，不写变形协调。

2. 中性轴、主惯性轴、危险点位置判断错误。

3. 公式会背，但不知道变量对应的物理对象。

3. 核心知识框架：按“会考什么”而不是按“课件页码”整理

下面这部分是你真正要反复看的主干内容。我按复习效率分了七组，每组都给你：必会概念、必背公式、做题流程、常错点。

3.1 内力、内力图与最基本的平衡思维

这一组是整门课的地基。后面所有强度、位移、稳定问题，都是先从内力开始。

截面法核心先取分离体，再列平衡，求截面上的 N / Q / M / T。
梁上微分关系dQ/dx = -q(x)，dM/dx = Q(x)
图形判断集中力使 Q 突变；集中力偶使 M 突变；Q = 0 处常对应 M 极值点。
先统一符号约定，再画受力图。不要脑子里默认“向上为正”，必须和你后面图上的符号一致。
梁题先求支反力，再按区段列内力表达式。每跨过一个集中力或力偶，都要检查函数是否发生跳跃。
如果题目给的是内力图，让你反推载荷或支座，记住：斜率来自分布载荷，跳跃来自集中力，边界值反推支座。

常错点：把“弯矩最大”机械理解成“端点最大”或“荷载最大处最大”。真正判断要看弯矩图。

3.2 拉压、位移、温度与装配问题

这类题看似简单，实际上特别容易和静不定、热应力、复合材料题串联。

正应力σ = N / A
轴向变形Δl = ∫ N / (EA) dx
温度自由变形Δl_T = α ΔT l，ε_T = α ΔT
受约束温度应力思路总应变 = 机械应变 + 温度应变；若总位移被限制，则必须加协调条件。
薄壁圆筒（闭口）σ_θ = pD/(2t)，σ_z = pD/(4t)

一看到“先安装、后升温、预拉伸、两端固定、不同材料并联或串联”这些字眼，你就不要急着套公式，先写：平衡 + 变形协调 + 材料关系。

会考的隐藏点：不同材料组合拉压时，若两部分完美连接，则轴向应变相同，不是应力相同。

3.3 扭转、连接件剪切与弯曲剪应力

这一块在课件里通常放在“剪应力分析”里，但考试很爱把它拆开：圆轴扭转是一类，梁腹板剪应力流是一类，连接件剪切和挤压又是一类。

圆轴扭转剪应力τ = Tr / J_p，τ_max = TR / J_p
扭转角φ = ∫ T / (GJ_p) dx
极惯性矩实心圆：J_p = πd⁴/32；空心圆：J_p = π(D⁴-d⁴)/32
梁的弯曲剪应力τ = VQ / (Ib)
连接件τ_avg = F/A_shear，σ_bs = F/A_bearing

如果题目问切应力流，核心不是背图，而是要知道：剪力想沿截面传递出去，材料里会形成一条连续的剪力流路径。

常错点：把弯曲剪应力最大点和弯曲正应力最大点混为一谈。矩形截面里，正应力最大在上下边缘，剪应力最大在中性轴附近。

3.4 弯曲、截面几何性质与强度

这是期末最值钱的一组。大题里只要出现梁、T 形截面、工字梁、组合梁，几乎都绕不开它。

纯弯正应力σ = My / I
轴力 + 弯矩组合σ = N/A ± My/I
平行轴定理I = I_c + Ad²
弯曲强度判据|σ|max ≤ [σ]
先求形心位置，再求形心惯性矩，再定中性轴，再找最危险纤维。
如果是组合截面或分层梁，先判断是否完美粘合；不同判断会直接改变应变分布和中性轴位置。
如果题目同时给了剪力和弯矩，不要只算弯曲正应力，要想一想是否还需要验算剪应力或强度理论。

实战记忆法：遇到截面题，脑子里按“形心 → 惯性矩 → 中性轴 → 危险点 → 应力”五连走，基本不容易乱。

3.5 应力状态、应变状态、莫尔圆与强度理论

这块是许多人期末最怕的内容，但只要把公式和几何意义连起来，其实非常稳定。

平面应力主应力σ̄ = (σx + σy)/2，R = √[((σx - σy)/2)² + τxy²]
主应力与最大剪应力σ1,2 = σ̄ ± R，τmax = R
广义胡克定律εx = [σx - ν(σy + σz)]/E，γxy = τxy/G
模量关系G = E / [2(1 + ν)]
Trescaσeq^T = max(|σ1-σ2|, |σ2-σ3|, |σ3-σ1|)
von Misesσeq^M = √{[(σ1-σ2)² + (σ2-σ3)² + (σ3-σ1)²] / 2}

题目如果出现薄壁圆筒、应变片、45°方向、主应力、最大剪应力、强度准则这些词，几乎一定要走“先求该点应力状态，再变换，再判断强度”这条路线。

常错点：主平面角度和最大剪应力平面角度差 45°，但应力圆上转角是实际物理平面转角的 2 倍。

3.6 位移、能量法、卡氏定理、静不定

这是压轴区。你只要吃透这部分，最后的大题会轻松很多。

应变能总式U = ∫ N²/(2EA) dx + ∫ T²/(2GJ) dx + ∫ M²/(2EI) dx
卡氏第二定理δ_i = ∂U/∂P_i，θ_i = ∂U/∂M_i
梁挠曲微分方程EI v''(x) = M(x)
力法骨架δ^0 + Σδij Xj = 已知约束位移
先判静不定次数，选冗余未知量。
去掉冗余约束，建立基本静定系。
写变形协调：被去掉的约束方向上，总位移必须满足题目给定条件，通常是零。
用积分法、叠加法或卡氏定理求相应位移。
联立解出冗余量，再回到原结构求内力、应力、位移。

2011 回忆题里的“悬臂梁 + 中点弹簧”本质就是：弹簧力未知，但中点位移必须等于弹簧伸长，典型变形协调。

3.7 压杆稳定、复合梁、温差变形与综合设计

这类题往往最后一问才最重要，因为它通常要求你比较多个限制条件后给出“允许值”。

欧拉临界力Pcr = π²EI / (μl)²
柔度与临界应力λ = μl / i，σcr = π²E / λ²
常见等效长度系数两端铰支 μ=1；一端固一端自由 μ=2；一端固一端铰 μ≈0.7；两端固定 μ=0.5
复合梁完美粘合同一截面转角相同、曲率相同、界面无相对滑移。
复合梁无粘合各层可相对滑移，应按各层独立受弯来分担内力。

2024 回忆题已经说明：老师不满足于只考“标准欧拉杆”，而是要你会证明等效长度，会区分分层梁粘合/不粘合，会处理双材料热弯曲。

稳定题最容易出错的地方不是公式，而是先没判清边界条件，直接把 μ 写错，后面全部白做。

4. 六大高频题型：你上考场时应该怎么启动

这一部分我特意写成“拿到题就能照着走”的格式。你做题时先识别题型，再套步骤，不要每次从零开始想。

题型 A：由内力图反推载荷、支座与弯矩图

观察 Q 图斜率：常斜率说明有均布载荷，斜率变化说明 q(x) 变了。
观察 Q 图跳跃：跳跃量就是集中力大小。
观察 M 图折线或曲线：曲线二阶特征来自载荷形式，尖跳来自集中力偶。
由 M 图边界值判断支座类型：简支端弯矩常为零，固端弯矩一般不为零。

一旦题目让你“指出支座约束类型”，说明光算数值不够，还要用边界条件反推结构。

题型 B：拉压 + 温度 + 装配 + 多材料

先画系统变形后的相对位移关系，想清楚谁跟谁一起动。
列平衡方程：各杆件轴力和外力关系。
列协调方程：总伸长、总缩短、间隙闭合、预拉伸释放等。
列本构：每一段的 Δ = Nl/(EA)，温度项加 αΔTl。

会考的关键不是“算出 N”，而是“能不能先把约束关系看对”。

题型 C：梁的弯曲应力、剪应力与危险点

先求截面形心和惯性矩。
由内力图确定危险截面：通常看 |M| 最大处，若要算剪应力再看 |Q| 大处。
根据 σ = My/I 判边缘应力，根据 τ = VQ/Ib 判中性轴附近剪应力。
如果是组合截面或胶合梁，先判断层间是否完美粘合。

题里如果出现 T 形、工字形、薄壁开口截面，危险点几乎从来不是“凭感觉就能看出来”的，必须算。

题型 D：薄壁圆筒、平面应力状态、莫尔圆

先写出该点的原始应力状态：例如圆筒就是环向 + 轴向。
再根据方向变换公式或莫尔圆求任意方向应力或应变。
如果题目给应变片角度，就把应变变换也写出来。
最后按主应力、最大剪应力、强度准则做判断。

2011 回忆题的“45°方向应变与内压关系”，本质就是应力状态变换。

题型 E：位移、叠加法、卡氏定理与静不定

判断是直接积分更快，还是叠加更快，还是卡氏定理更快。
如果有弹簧、额外支撑或固定端，优先考虑力法和协调。
对卡氏定理题，先写总应变能，再对相应外力求偏导。
最后不要忘记把求出的冗余力重新带回去求目标量。

最常见失分：只写“中点位移 = 弹簧伸长”，但没有把位移表示成荷载和未知弹簧力的函数。

题型 F：压杆稳定与综合设计

先看杆件边界条件，写对 μ。
算柔度 λ = μl/i，确认是否用欧拉公式。
若题目还有限用应力、挠度或其他材料条件，要把所有条件都列出。
求“最大允许载荷”时，最后取所有条件中最小的那个。

综合设计题真正拉开分差的地方在最后一句：你能不能把强度、刚度、稳定性统一比较。

6. 实验与开放题：如果老师不只考纯计算，这里最值得看

你下载的资料里实验材料也很多，虽然期末未必大考实验，但实验背景题、开放题或者简答题经常从这里出发。

拉伸实验

会读应力-应变曲线：比例极限、屈服、强化、颈缩、极限强度、断后伸长。

知道韧性材料和脆性材料的失效差别。

应变片 / 电测

会把 0° / 45° / 90° 应变片结果和主应变、主应力联系起来；知道桥路是在把微小应变转换成可测信号。

组合梁 / 弯扭实验

会用理论解释：为什么界面是否滑移会影响刚度；为什么弯曲和扭转叠加后，危险点位置会变化。

开放题最常见的问法

1. 解释某实验现象对应的理论原因。

2. 对比两种材料或两种结构形式的力学响应。

3. 问“为什么这个公式或假定成立”，本质是让你说清楚平面假定、线弹性、薄壁假定、细长杆假定等适用条件。

7. 自测题：考前请至少过一遍

这些题不是为了折磨你，而是为了检验你是否真的把“公式 + 图像 + 步骤”串起来了。

7.1 快速问答

1. 为什么 Q = 0 常对应弯矩极值点？

2. 为什么梁的弯曲正应力在边缘最大，而弯曲剪应力在中性轴附近最大？

3. 完美粘合的复合梁与无粘合的分层梁，最核心的差别是什么？

4. 为什么莫尔圆上转过的角度是物理平面转角的 2 倍？

5. 为什么一端固定一端自由的压杆比两端铰支更容易失稳？

6. 静不定题里，多余约束为什么既让问题更难解，又让它最终可解？

7. 温度变化题里，什么时候会产生热应力，什么时候不会？

8. 薄壁圆筒题里，环向应力为什么比轴向应力大一倍？

展开答案提要

1. 因为 dM/dx = Q，导数为零对应极值候选点。

2. 因为弯曲正应力与离中性轴距离成正比，而矩形截面弯曲剪应力在边缘为零、中间最大。

3. 完美粘合意味着界面无滑移、各层协调变形；无粘合意味着各层可相对滑移，应变分布不再跨层连续。

4. 因为莫尔圆把平面法向角度映射成应力点在圆上的圆心角，几何推导会出现倍角关系。

5. 因为它的等效屈曲半波更长，等效长度系数更大，临界力更小。

6. 多余约束增加了未知量，但也带来了额外的位移限制；用变形协调补上平衡方程不足的部分后问题就可解。

7. 自由热胀冷缩不产生热应力；只有当变形被约束时，温度应变不能完全实现，才转化成机械应力。

8. 因为截取圆筒不同方向平衡时，抗内压的受力边界长度不同，导致环向平衡给出的膜力更大。

7.2 真正上考场前你要能口述出来的五件事

1. 任何静不定题的标准流程。

2. 莫尔圆如何从已知应力状态画到主应力和最大剪应力。

3. T 形或工字形截面如何找形心、惯性矩和危险点。

4. 压杆稳定四种常见边界条件的等效长度系数。

5. 组合梁“完美粘合”和“无粘合”的本质区别。

8. 最后冲刺建议：如果时间只够再拼一轮，优先级怎么排

第一优先级：先吃大题骨架

按顺序复习：弯曲应力与截面几何 → 应力状态与强度理论 → 位移/能量法/静不定 → 压杆稳定。

这四块能决定你大题能不能写满。

第二优先级：把组合问题练熟

重点看温度、装配、弹簧、复合梁、薄壁圆筒、组合受力。因为老师非常喜欢把基础题“组合化”。

第三优先级：用图带公式

复习时把图和公式绑在一起：看到中性轴就想到 σ = My/I；看到应力圆就想到主应力公式；看到细长杆就想到 μ 和欧拉力。

第四优先级：不要把时间全扔给细枝末节

动态载荷、疲劳、复合材料概述这些如果你时间紧，可以先抓概念和老师上课强调的结论，不要反客为主。

真正决定分数的不是“你看了多少页”，而是“你能不能在陌生综合题里快速写出平衡、协调、本构三板斧”。
