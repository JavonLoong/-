# 第07章 一般应力状态下的强度失效与强度设计

## 1. 章节一句话总纲

本章解决复杂应力状态下如何选择强度理论，把主应力或应力分量转化为相当应力，并与许用应力比较完成校核或尺寸设计。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 塑性材料选什么理论？ | 第三强度理论或第四强度理论 |
| 脆性材料选什么理论？ | 最大拉应力理论为主 |
| 弯扭组合怎么校核？ | `sigma_eq3=sqrt(sigma^2+4tau^2)`, `sigma_eq4=sqrt(sigma^2+3tau^2)` |
| 危险截面和危险点怎么找？ | 先看内力图，再看应力分布 |
| 第一/第二强度理论放在哪里？ | 第一理论偏脆性拉断，第二理论工程使用较少，期末一般作概念识别 |

## 3. 前置知识

- 内力图和危险截面。
- 弯曲正应力、扭转剪应力。
- 平面应力主应力。
- 屈服极限、强度极限、许用应力、安全系数。

## 4. 概念依赖图

```text
外载 -> 受力图 -> 内力 N/V/M/T -> 危险点应力分量 -> 主应力/相当应力 -> 强度理论 -> 校核/设计
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `sigma_eq3=sigma1-sigma3` | 最大剪应力理论 | 塑性材料，偏保守 | 主应力未排序 |
| `sigma_eq4=sqrt(s1^2+s2^2+s3^2-s1s2-s2s3-s3s1)` | 畸变能理论 | 塑性材料常用 | 漏掉 `sigma3=0` |
| 第一理论 `sigma_eq1=sigma1` | 最大拉应力理论 | 脆性材料拉断校核 | 用于塑性屈服主控题 |
| 第二理论 `sigma_eq2=epsilon1 E` 或按最大伸长线应变判据 | 最大伸长线应变理论 | 历史理论，工程考试多作概念了解 | 与第一理论混淆 |
| 平面应力 `sigma_eq4=sqrt(sigma_x^2+sigma_y^2-sigma_x sigma_y+3tau_xy^2)` | 第四理论展开 | 平面应力 | 符号项写错 |
| 弯扭第三 `sqrt(sigma^2+4tau^2)` | 单向正应力+剪应力特例 | 圆轴弯扭 | 与第四混淆 |
| 弯扭第四 `sqrt(sigma^2+3tau^2)` | 单向正应力+剪应力特例 | 圆轴弯扭 | 系数写成 4 |

### 公式血缘审计补充

强度理论的前提是先由内力得到同一点的应力状态，并按材料类型选判据；不适用于没有排序主应力、把不同点应力混合、或疲劳/稳定题替代静强度校核。`sigma1>=sigma2>=sigma3` 必须先排序；`[sigma]` 若已含安全系数不可重复除以 `n`；典型题入口是圆轴弯扭组合、薄壁容器、偏心拉压和截面设计。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 已知应力分量校核 | 求主应力或直接算相当应力 -> 比 `[sigma]` |
| 圆轴弯扭组合 | 求 `M,T` -> 外表面 `sigma=M/W`, `tau=T/Wp` -> 相当应力 |
| 薄壁圆筒 | 求环向、轴向应力 -> 强度理论校核 |
| 偏心拉压 | 叠加边缘正应力 -> 找最大拉/压 |
| 设计轴径 | 建立 `sigma_eq <= [sigma]` -> 代入 `W,Wp` -> 反解 `d` |

## 7. 易错点清单

- 危险截面和危险点混为一谈。
- 弯扭组合忘记圆截面 `Wp=2W`。
- 平面应力下 `sigma3=0` 忘记参与排序。
- 脆性材料机械套第四理论。
- 许用应力已含安全系数时又重复除以 `n`。

## 8. 最低掌握标准

- 能求平面应力主应力。
- 能区分塑性、脆性材料常用强度理论。
- 能做圆轴弯扭组合校核。

## 9. 高分掌握标准

- 能主动判断危险截面、危险点和材料类型。
- 能比较第三、第四理论的保守性。
- 能根据强度理论反求轴径或截面尺寸。

## 10. 三遍法学习任务

| 遍数 | 任务 |
|---|---|
| 第一遍 | 整理第一到第四强度理论的判断对象；计算题重点掌握第三、第四理论 |
| 第二遍 | 3 道应力单元题 + 3 道弯扭组合题 |
| 第三遍 | 每题标注“内力错/应力错/理论选错/单位错” |

## 11. 考试模拟与变式训练

实心圆轴受弯矩 `M` 和扭矩 `T`：  
`sigma=32M/(pi d^3)`, `tau=16T/(pi d^3)`。  
第四理论：`sqrt(sigma^2+3tau^2) <= [sigma]`。

变式：第三理论改为 `sqrt(sigma^2+4tau^2) <= [sigma]`。
