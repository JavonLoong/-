# 第02章 内力分析与内力图

## 1. 章节一句话总纲

内力分析用截面法把外载与支反力转化为截面上的 `N,V,M,T`，内力图则把内力沿杆件轴线的变化展示出来，是后续应力、变形和校核的入口。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 如何求指定截面内力？ | 保留一侧，列平衡 |
| 如何画剪力图和弯矩图？ | 先求反力，再分段 |
| 集中力/力偶/分布载荷如何影响图形？ | 集中力使剪力跳，集中力偶使弯矩跳，均布载荷使剪力斜变 |
| 最大弯矩在哪里？ | 常检查 `V=0` 点和端点 |

## 3. 前置知识

- 支座反力：固定端、铰支座、滚支座。
- 静力平衡方程。
- 力矩正负号。
- 分段函数和图像。

## 4. 概念依赖图

```text
外载与约束 -> 整体受力图 -> 支反力 -> 截面法 -> N/V/M/T -> 内力图 -> 最大内力位置
```

## 5. 公式血缘表

| 公式/关系 | 来源 | 用途 | 常见误用 |
|---|---|---|---|
| `sum Fx=0` | 截面一侧平衡 | 求轴力 `N` | 忽略中间集中力 |
| `sum Fy=0` | 截面一侧平衡 | 求剪力 `V` | 正负号前后不一 |
| `sum M=0` | 截面一侧取矩 | 求弯矩 `M` | 力臂取错 |
| `dM/dx=V` | 微段平衡 | 用剪力判断弯矩斜率 | 忘记符号约定 |
| `dV/dx=-q` | 微段平衡 | 用载荷判断剪力斜率 | 混淆 `q` 方向 |

### 公式血缘审计补充

内力公式的推导前提是“取出的杆段处于静力平衡”；不适用于动力学平衡或大变形几何非线性题。`N,V,M,T,q,x` 必须在图上标方向和单位；内力图单位分别为 `N`、`N`、`N*mm`、`N*mm`；典型题入口是指定截面内力、剪力图/弯矩图和由图反推载荷。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 指定截面内力 | 画受力图 -> 求反力 -> 截开 -> 保留受力少的一侧 -> 列平衡 |
| 轴力图 | 在轴向集中力处分段，拉正压负 |
| 剪力图 | 从左到右扫：集中力跳跃，均布载荷斜线 |
| 弯矩图 | 用 `dM/dx=V` 或剪力图面积；集中力偶处跳跃 |
| 反推载荷 | 看剪力图跳跃、斜率和弯矩图跳跃 |

## 7. 易错点清单

- 未求支反力就画内力图。
- 截开后把被切掉一侧的外力也带入方程。
- 集中力处弯矩通常连续，剪力跳跃。
- 集中力偶处弯矩跳跃，剪力不跳。
- 均布载荷下剪力不是水平线，弯矩不是直线。

## 8. 最低掌握标准

- 能求简单梁支反力。
- 能用截面法求 `N,V,M`。
- 能画集中力、均布载荷下的剪力图和弯矩图。

## 9. 高分掌握标准

- 能用斜率、跳跃、面积关系检查内力图。
- 能处理多段载荷、集中力偶组合题。
- 能把最大内力自然接到后续应力校核。

## 10. 三遍法学习任务

| 遍数 | 任务 |
|---|---|
| 第一遍 | 每天做 5 个指定截面内力小题 |
| 第二遍 | 画集中力、均布载荷、集中力偶各 3 张内力图 |
| 第三遍 | 每张图用“端点、突跳、斜率、面积”检查 |

## 11. 考试模拟与变式训练

简支梁长 `L`，跨中集中力 `P`：  
`RA=RB=P/2`，左半段 `V=P/2`，`M=Px/2`，最大弯矩 `Mmax=PL/4`。

变式：全跨均布载荷 `q`：  
`RA=RB=qL/2`，`V=qL/2-qx`，`M=qLx/2-qx^2/2`，`Mmax=qL^2/8`。

## 12. 复习卡片

| 正面 | 背面 |
|---|---|
| 求内力第一步？ | 画整体受力图并求支反力 |
| 集中力对剪力图影响？ | 剪力图突跳 |
| 集中力偶对弯矩图影响？ | 弯矩图突跳 |
| `V=0` 常提示什么？ | 弯矩可能取极值 |
