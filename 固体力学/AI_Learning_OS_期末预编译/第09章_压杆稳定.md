# 第09章 压杆的稳定性分析与稳定性设计

## 1. 章节一句话总纲

受压细长杆可能在压应力未达到强度极限前发生侧向失稳，所以压杆必须用临界力、柔度和稳定安全系数校核。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 稳定失效与强度失效区别？ | 不是材料压坏，而是整体弯曲失稳 |
| 欧拉临界力是什么？ | `Pcr=pi^2 EI/(mu L)^2` |
| 端部约束如何影响稳定？ | 通过长度系数 `mu` |
| 弱轴为什么重要？ | 稳定绕 `Imin` 方向发生 |
| 设计怎么判断？ | 同时满足强度和稳定 |
| 欧拉公式什么时候能用？ | 柔度足够大、临界应力未超过比例极限时使用 |

## 3. 前置知识

- 轴向压缩 `sigma=P/A`。
- 梁弯曲刚度 `EI`。
- 惯性矩 `I`、面积 `A`、惯性半径 `i=sqrt(I/A)`。

## 4. 概念依赖图

```text
轴向压力 -> 受力图 -> 端部约束 -> mu L -> Imin/A/i -> 柔度 lambda -> Pcr -> 稳定校核
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `Pcr=pi^2 EI/(mu L)^2` | 欧拉稳定理论 | 细长压杆、线弹性 | 忘记有效长度 |
| `sigma_cr=Pcr/A=pi^2 E/lambda^2` | 欧拉公式除以面积 | 细长杆 | 用于短粗杆 |
| `lambda=mu L/i` | 柔度定义 | 判断长细程度 | 把 `i` 写成 `I/A` |
| `i=sqrt(I/A)` | 截面几何 | 任意截面 | 不取弱轴 |
| `[P]st=Pcr/nst` | 稳定安全系数 | 设计校核 | 把 `Pcr` 当许用载荷 |

### 公式血缘审计补充

欧拉公式前提是细长压杆、轴向受压、线弹性、初始缺陷不作为主控量；不适用于短粗柱直接压溃、中等柔度需经验公式、偏心受压未处理弯曲的题。`mu,L,Imin,A,i,lambda` 必须取同一失稳平面；`Pcr` 为临界力，设计要除以稳定安全系数；典型题入口是端部约束识别、弱轴稳定校核和强度/稳定双校核。

### 欧拉适用范围

考试中可用这个判据：先算 `lambda=mu L/i`，再算 `sigma_cr=pi^2 E/lambda^2`。若 `sigma_cr` 不超过材料比例极限 `sigma_p`，或题目明确说明“细长压杆/适用欧拉公式”，可用欧拉公式；否则属于中小柔度杆或短柱，应改用题给经验公式或先按强度压缩校核。没有给 `sigma_p` 时，期末题通常会用“细长杆”提示。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 欧拉临界力 | 判断约束 -> 取 `mu` -> 取 `Imin` -> 算 `Pcr` |
| 安全系数 | `nst=Pcr/P` |
| 许用压力 | `[P]st=Pcr/nst` |
| 柔度判断 | 算 `A,Imin,i,lambda` |
| 综合校核 | 先强度 `P/A <= [sigma]`，再稳定 `P <= Pcr/nst` |

## 7. 易错点清单

- 只算 `sigma=P/A`，漏稳定。
- `mu` 取错。
- 不取 `Imin`。
- 欧拉公式无条件用于短粗杆。
- 没有检查 `sigma_cr` 是否仍在线弹性比例范围。
- 临界力没有除以稳定安全系数。

## 8. 最低掌握标准

- 能解释压杆稳定。
- 能记住欧拉公式和常见 `mu`。
- 能完成简单稳定校核。

## 9. 高分掌握标准

- 能判断弱轴和适用范围。
- 能同时做强度、稳定校核。
- 能反推截面尺寸或改善方案。

## 11. 考试模拟与变式训练

两端铰支压杆：`mu=1`，`Pcr=pi^2 E Imin/L^2`。  
一端固定一端自由：`mu=2`，`Pcr=pi^2 E Imin/(2L)^2`。  
两端固定：`mu=0.5`，`Pcr=pi^2 E Imin/(0.5L)^2`。

## 12. 复习卡片

| 正面 | 背面 |
|---|---|
| 欧拉公式？ | `Pcr=pi^2 EI/(mu L)^2` |
| 柔度？ | `lambda=mu L/i`, `i=sqrt(I/A)` |
| 最容易绕哪轴失稳？ | 弱轴，取 `Imin` |
| 稳定许用压力？ | `Pcr/nst` |
