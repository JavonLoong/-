# 第03章 轴向载荷作用下杆件的材料力学问题

## 1. 章节一句话总纲

轴向拉压问题以轴力为核心，按“受力图 -> 截面法求轴力 -> `sigma=N/A` 求应力 -> `Delta l=NL/EA` 求变形 -> 强度/刚度校核”的流程解决。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 轴向杆内力是什么？ | 轴力 `N`，拉正压负 |
| 如何求正应力？ | `sigma=N/A` |
| 如何求伸长/缩短？ | `Delta l=NL/EA`，多段求和 |
| 超静定轴向杆怎么做？ | 平衡 + 变形协调 + 胡克定律 |
| 温度变化如何处理？ | 自由热胀无热应力，受约束才有热应力 |

## 3. 前置知识

- 截面法。
- 胡克定律 `sigma=E epsilon`。
- 应变定义 `epsilon=Delta l/L`。
- 热膨胀 `Delta l_T=alpha L Delta T`。

## 4. 概念依赖图

```text
轴向外载 -> 受力图 -> 截面法 -> 轴力图 -> sigma=N/A -> Delta l=NL/EA -> 强度/刚度/协调
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `sigma=N/A` | 平均正应力定义 | 轴力通过形心，远离载荷突变处 | 把外力都当轴力 |
| `Delta l=NL/(EA)` | `sigma=N/A`, `sigma=E epsilon`, `epsilon=Delta l/L` | 等截面、线弹性、轴力常量 | 单位混乱 |
| `Delta l=sum Ni Li/(Ei Ai)` | 分段叠加 | 阶梯杆、多材料杆 | 漏掉符号 |
| `Delta l_T=alpha L Delta T` | 热膨胀规律 | 自由热变形 | 自由膨胀误算热应力 |
| `sigma_T=E alpha Delta T` | 完全约束协调 | 两端固定且线弹性 | 方向号错 |

### 公式血缘审计补充

轴向公式的推导前提是轴力沿杆轴、截面平均应力可用、材料线弹性；不适用于偏心受力、屈服后变形或局部应力集中主导的题。`N,L,E,A,alpha,Delta T` 必须逐段对应；使用 `N-mm-MPa` 时 `E` 用 `MPa`、`A` 用 `mm^2`；典型题入口是阶梯杆变形、温度应力和轴向超静定。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 等截面杆强度 | 求 `N` -> `sigma=N/A` -> 比 `[sigma]` |
| 设计面积 | `A >= Nmax/[sigma]` |
| 阶梯杆变形 | 画轴力图 -> 分段列表 -> 求和 |
| 超静定杆 | 设反力 -> 平衡 -> 协调 -> 本构 -> 联立 |
| 温度应力 | 判断约束 -> 写自由热变形和弹性变形协调 |

## 7. 易错点清单

- 忽略中间集中力导致轴力图错。
- 阶梯杆用错对应截面面积。
- 变形计算漏正负号。
- 超静定题只列平衡，漏协调。
- 自由升温误认为有热应力。

## 8. 最低掌握标准

- 会画轴力图。
- 会算 `sigma=N/A` 和 `Delta l=NL/EA`。
- 会做简单强度、刚度校核。
- 能判断自由热变形与约束热应力。

## 9. 高分掌握标准

- 能处理多段、多材料、温度、装配误差题。
- 能建立“平衡 + 协调 + 本构”方程组。
- 能根据最大轴力和最小面积判断危险截面。

## 10. 三遍法学习任务

| 遍数 | 任务 |
|---|---|
| 第一遍 | 6 道多集中力轴向杆，全部先画轴力图 |
| 第二遍 | 4 道阶梯杆，列表写 `N,L,A,E,Delta l` |
| 第三遍 | 3 道超静定/温度题，强制写协调方程 |

## 11. 考试模拟与变式训练

等截面杆：`L=2000 mm`, `A=400 mm^2`, `P=40 kN`, `E=200000 MPa`。  
`sigma=40000/400=100 MPa`。  
`Delta l=40000*2000/(200000*400)=1 mm`。

变式：若 `[sigma]=120 MPa` 强度满足；若 `[Delta]=0.8 mm` 刚度不满足。
