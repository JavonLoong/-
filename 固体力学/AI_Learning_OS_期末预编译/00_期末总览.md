# 固体力学期末总览

## 复习总原则

所有计算题统一按这条链路启动：

```text
受力图 -> 截面法 -> 平衡方程 -> 内力图/内力函数 -> 应力或变形 -> 强度/刚度/稳定/疲劳校核
```

不要跳过受力图直接套公式；不要把外载误当截面内力；不要只算结果而不写校核结论。

## 三遍复习路线

| 阶段 | 目标 | 任务 |
|---|---|---|
| 第一遍：预编译 | 建立章节地图 | 每章整理核心公式、适用条件、题型入口、易错点 |
| 第二遍：三遍法 | 直觉、推导、应用 | 每章至少做 1 道基础题、1 道变式题、1 道限时题 |
| 第三遍：考试化 | 形成答题动作 | 混排训练内力图、弯曲、应力状态、强度理论、位移、静不定、稳定 |
| 考前 24 小时 | 减少低级失分 | 只看公式血缘、错题根因、边界条件、单位和结论模板 |

## 章节依赖

```text
静力学平衡
-> 截面法与内力图
-> 拉压/剪切/弯曲/扭转应力
-> 应力状态与强度理论
-> 梁位移、能量法、静不定
-> 稳定、动载、疲劳、复合材料与粘弹性
```

## 题型矩阵

| 题型 | 必须先做 | 核心公式/方法 | 最后检查 |
|---|---|---|---|
| 轴向拉压 | 画轴力图 | `sigma=N/A`, `Delta l=NL/EA` | 强度、刚度、拉压号 |
| 扭转 | 画扭矩图 | `tau=T rho/Ip`, `phi=TL/GIp` | 最大剪应力在外缘 |
| 弯曲 | 画剪力图、弯矩图 | `sigma=My/I`, `tau=VQ/Ib` | 最大弯矩截面、危险点 |
| 应力状态 | 先取危险点 | 主应力、莫尔圆、强度理论 | `sigma3=0` 也要参与排序 |
| 梁位移 | 先写弯矩方程 | `EI v''=M(x)`, 单位载荷法 | 边界条件、连续条件 |
| 能量法 | 先求内力函数或单位载荷内力 | `delta=int M m/(EI) dx + int N n/(EA) dx + int T t/(GIp) dx` | 单位载荷方向、积分分段、结果量纲 |
| 静不定 | 先数未知量 | 平衡 + 协调 + 本构 | 不能只列平衡 |
| 压杆稳定 | 判断端部约束 | `Pcr=pi^2 EI/(mu L)^2` | 弱轴、有效长度、安全系数 |
| 动载荷/冲击 | 先判断加载类型 | `Kd=2`, `Kd=1+sqrt(1+2h/delta_st)` | `delta_st` 取静位移，最后做强度校核 |
| 疲劳 | 找循环应力来源 | `sigma_a`, `sigma_m`, Goodman/S-N | 不拿静强度代替疲劳强度 |
| 复合材料 | 判断载荷方向 | 规则混合法、方向性 | 不能当各向同性 |
| 粘弹性 | 判断加载历史 | 蠕变、松弛、`J(t)`, `E(t)` | 时间变量不能漏 |

## 考前检查清单

- 是否先画受力图并标清外力、约束、尺寸、截面位置？
- 是否用截面法求出了 `N,V,M,T`，而不是把外载直接塞进公式？
- 是否统一单位：`N-mm-MPa` 或 `N-m-Pa`？
- 是否区分强度、刚度、稳定、疲劳寿命？
- 是否找到了危险截面和危险点？
- 是否明确写出校核式，例如 `sigma_max <= [sigma]`、`delta_max <= [delta]`、`P <= Pcr/n`？
- 是否说明最终结论：安全/不安全，控制因素是什么？

## 标准答题格式

1. 画受力图。
2. 求支反力。
3. 取截面并列平衡。
4. 画或写内力图/内力函数。
5. 计算应力、应变、位移、转角或临界载荷。
6. 选择强度、刚度、稳定或疲劳准则。
7. 写清结论和单位。
