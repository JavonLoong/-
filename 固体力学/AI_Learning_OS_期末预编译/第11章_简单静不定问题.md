# 第11章 简单的静不定问题

## 1. 章节一句话总纲

静不定问题仅靠平衡方程不够，必须加入变形协调和材料本构，形成“平衡 + 几何 + 本构”的闭合方程组。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 为什么静不定？ | 未知约束力数多于独立平衡方程数 |
| 多余力怎么求？ | 选冗余未知量，写协调条件 |
| 温度/装配误差怎么处理？ | 写自由变形、弹性变形和实际变形关系 |
| 静不定梁怎么做？ | 力法：去约束成静定基本系 |

## 3. 前置知识

- 静定结构反力。
- `Delta=NL/(EA)`。
- `phi=TL/(GIp)`。
- 梁位移或能量法。

## 4. 概念依赖图

```text
原结构 -> 受力图 -> 未知数多于平衡方程 -> 协调条件 -> 本构关系 -> 联立求多余力 -> 内力/应力校核
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `r=未知约束力数-独立平衡方程数` | 静不定次数定义 | 平面静力问题 | 误数约束 |
| `Delta=sum Ni Li/(Ei Ai)` | 轴向变形叠加 | 分段杆 | 漏符号 |
| `phi=sum Ti Li/(Gi Ipi)` | 扭转角叠加 | 阶梯轴 | 忘分段 |
| `delta_P+fX=0` | 力法协调 | 一次静不定 | 正方向不一致 |
| `Delta_T=alpha DeltaT L` | 热膨胀 | 温度均匀 | 自由热变形当实际变形 |

### 公式血缘审计补充

静不定公式的前提是结构线弹性、变形协调关系明确、约束位移已知或为零；不适用于接触状态未判定、间隙未闭合、材料屈服后刚度变化的问题。`X,delta_P,f,Delta_T` 的正方向要统一；轴向刚度 `EA/L` 和扭转刚度 `GIp/L` 单位要对应；典型题入口是两端固定杆、并联杆、温度应力、支座沉降和力法梁。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 两端固定杆 | 设反力 -> 平衡 -> 总伸长为零 |
| 并联杆 | 平衡 `N1+N2=P` -> 协调 `Delta1=Delta2` |
| 固定端阶梯轴 | 平衡扭矩 -> 总扭转角为零 |
| 温度应力 | 自由热变形 + 弹性变形 = 实际约束位移 |
| 简单静不定梁 | 选多余力 -> 去约束 -> 写位移协调 |

## 7. 易错点清单

- 只列平衡方程。
- 凭感觉平均分配内力。
- 忽略不同 `E,A,L`。
- 温度题把自由伸长当实际伸长。
- 求出多余力后忘回代求内力图。

## 8. 最低掌握标准

- 能判断一次静不定。
- 能写平衡方程和协调方程。
- 能处理两端固定杆、并联杆、阶梯轴。

## 9. 高分掌握标准

- 能处理温度、支座沉降、装配误差。
- 能用力法处理简单梁。
- 能解释内力重分配。

## 10. 三遍法学习任务

| 遍数 | 任务 |
|---|---|
| 第一遍 | 每题先数未知力和方程数 |
| 第二遍 | 练温度、装配误差各 2 题 |
| 第三遍 | 用力法做 2 道一次静不定梁 |

## 11. 考试模拟与变式训练

两根不同材料竖杆共同支承刚性板，外力 `P`：  
平衡：`N1+N2=P`。  
协调：`N1 L1/(E1 A1)=N2 L2/(E2 A2)`。  
联立求两杆轴力，再校核 `sigma_i=Ni/Ai`。

## 12. 复习卡片

| 正面 | 背面 |
|---|---|
| 静不定多什么方程？ | 变形协调方程 |
| 三类基本方程？ | 平衡、几何协调、本构 |
| 并联杆内力按什么分配？ | 按刚度 `EA/L` |
| 两端固定杆受热实际总变形？ | 支座不动时为零 |
