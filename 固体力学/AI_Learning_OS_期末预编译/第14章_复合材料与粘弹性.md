# 第14章 复合材料力学性能和聚合物粘弹性力学性能

## 1. 章节一句话总纲

复合材料强调方向性与组分协同，聚合物粘弹性强调时间效应；解题仍要先求内力和应力，再选择对应的材料本构关系。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 复合材料为何不能直接用普通 `E`？ | 各向异性，性能依赖纤维方向 |
| 纵向模量怎么估？ | 等应变规则混合法 |
| 横向模量为什么低？ | 主要受基体和界面控制 |
| 蠕变是什么？ | 恒定应力下应变随时间增大 |
| 松弛是什么？ | 恒定应变下应力随时间降低 |

## 3. 前置知识

- 应力应变关系 `sigma=E epsilon`, `tau=G gamma`。
- 截面法与平均应力。
- 强度和刚度校核。
- 基本函数图像和时间变量。

## 4. 概念依赖图

```text
外载 -> 受力图 -> 截面内力 -> 应力/应变 -> 方向性或时间性本构 -> 变形/强度/长期刚度校核
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `Vf+Vm=1` | 体积分数 | 两相简化复合材料 | 与质量分数混 |
| `E1=Vf Ef+Vm Em` | 纵向等应变 | 沿纤维方向拉伸 | 用于横向 |
| `1/E2=Vf/Ef+Vm/Em` | 横向等应力近似 | 垂直纤维方向 | 与纵向公式混 |
| `epsilon(t)=J(t) sigma` | 蠕变柔量 | 恒定应力 | 漏时间 |
| `sigma(t)=E(t) epsilon` | 松弛模量 | 恒定应变 | 把松弛当蠕变 |
| Maxwell 模型 `dot epsilon=dot sigma/E+sigma/eta` | 弹簧与黏壶串联 | 应力松弛明显 | 与 Kelvin 模型混 |
| Kelvin 模型 `sigma=E epsilon+eta dot epsilon` | 弹簧与黏壶并联 | 蠕变延迟明显 | 误认为能描述瞬时松弛 |

### 公式血缘审计补充

复合材料公式前提是体积分数已知、单向连续纤维、简化等应变/等应力假设；不适用于复杂铺层直接等同单向层、界面失效主导或质量分数未换算。粘弹性公式前提是给定加载历史和 `J(t)` 或 `E(t)`；不适用于时间条件不明。`Vf,Vm,E1,E2,J(t),E(t)` 要写清方向和单位；典型题入口是纵向/横向模量、复合杆拉伸、蠕变和应力松弛。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 单向复合材料纵向模量 | 判断沿纤维 -> `E1=Vf Ef+Vm Em` |
| 横向模量 | 判断垂直纤维 -> 用倒数混合法 |
| 复合材料杆拉伸 | 受力图 -> `N=P` -> `sigma=N/A` -> 沿纤维用 `epsilon=sigma/E1`，横向用 `epsilon=sigma/E2` |
| 蠕变 | 求恒定应力 -> `epsilon(t)=J(t)sigma` |
| 应力松弛 | 给恒定应变 -> `sigma(t)=E(t)epsilon` |

## 7. 易错点清单

- 把复合材料当各向同性。
- 未判断载荷方向和纤维方向。
- 纵向、横向模量公式用反。
- 把蠕变理解成应力增加。
- 把应力松弛理解成应变减少。
- 粘弹性题漏时间和加载历史。

## 8. 最低掌握标准

- 能说明复合材料方向性。
- 能用规则混合法估算纵向模量。
- 能区分蠕变和应力松弛。
- 能按受力图到本构的流程做基础计算。

## 9. 高分掌握标准

- 能根据载荷方向选择 `E1,E2,G12`。
- 能说明纤维、基体、界面对失效的影响。
- 能做长期刚度校核。

## 11. 考试模拟与变式训练

数值题：单向纤维复合材料杆沿纤维方向受拉 `P=10 kN`，截面积 `A=100 mm^2`，长度 `L=500 mm`，`Vf=0.6`，`Ef=200000 MPa`，`Em=3000 MPa`，允许伸长 `[Delta l]=0.20 mm`。

1. 截面法：`N=P=10000 N`。
2. 平均应力：`sigma=N/A=100 MPa`。
3. 纵向模量：`E1=Vf Ef+(1-Vf)Em=0.6*200000+0.4*3000=121200 MPa`。
4. 应变：`epsilon=sigma/E1=100/121200`。
5. 伸长：`Delta l=epsilon L`。
6. 与 `[Delta l]` 比较，写出刚度是否满足。

变式：载荷垂直纤维方向时改用 `E2`；聚合物恒载时用 `epsilon(t)=J(t)sigma`。
