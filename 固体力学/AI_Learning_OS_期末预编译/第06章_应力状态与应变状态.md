# 第06章 应力状态与应变状态分析

## 1. 章节一句话总纲

本章把第 4、5 章求得的某点正应力和剪应力，转化为任意方向截面上的应力、主应力、最大剪应力和应变状态，为强度理论做准备。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 一点应力状态如何表示？ | `sigma_x, sigma_y, tau_xy` |
| 任意斜截面应力如何求？ | 应力变换公式 |
| 主应力和主平面是什么？ | 主平面剪应力为零 |
| 莫尔圆怎么看？ | 圆心、半径、主应力、最大剪应力 |
| 应力与应变如何联系？ | 广义胡克定律 |
| 应变状态如何分析？ | 应变变换、主应变、应变花读数 |

## 3. 前置知识

- 第 4 章正应力公式。
- 第 5 章剪应力公式。
- 坐标与符号约定。
- 胡克定律、泊松比、剪切模量。

## 4. 概念依赖图

```text
构件受力 -> 危险截面 -> 危险点 sigma/tau 或 epsilon/gamma -> 应力/应变变换 -> 主应力/主应变 -> 强度理论/应变花反算
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `sigma_1,2=(sigma_x+sigma_y)/2 +/- sqrt(((sigma_x-sigma_y)/2)^2+tau_xy^2)` | 平面应力变换 | 平面应力状态 | 忘记主应力排序 |
| `tau_max=sqrt(((sigma_x-sigma_y)/2)^2+tau_xy^2)` | 莫尔圆半径 | 面内最大剪应力 | 与三维最大剪应力混淆 |
| `tan 2theta=2tau_xy/(sigma_x-sigma_y)` | 主平面剪应力为零 | 求主方向 | 角度用截面角而非法线角 |
| `epsilon_x=(sigma_x-nu sigma_y)/E` | 平面应力广义胡克定律 | 线弹性 | 漏泊松效应 |
| `gamma_xy=tau_xy/G` | 剪切胡克定律 | 线弹性 | 把工程剪应变和张量剪应变混淆 |
| `epsilon_theta=(epsilon_x+epsilon_y)/2+(epsilon_x-epsilon_y)cos2theta/2+gamma_xy sin2theta/2` | 平面应变变换 | 小变形、工程剪应变 | 漏掉 `gamma_xy/2` |
| `gamma_theta=-(epsilon_x-epsilon_y)sin2theta+gamma_xy cos2theta` | 平面应变变换 | 小变形 | 符号和角度方向混乱 |
| `epsilon_1,2=(epsilon_x+epsilon_y)/2 +/- sqrt(((epsilon_x-epsilon_y)/2)^2+(gamma_xy/2)^2)` | 主应变条件 | 平面应变状态 | 把 `gamma/2` 写成 `gamma` |
| 三向应变花：`epsilon_0=epsilon_x`, `epsilon_45=(epsilon_x+epsilon_y)/2+gamma_xy/2`, `epsilon_90=epsilon_y` | 应变变换特例 | 0/45/90 度应变花 | 读数方向不对应 |

### 公式血缘审计补充

应力/应变变换公式的前提是小变形、同一点处的平面应力或平面应变状态、坐标正方向明确；不适用于跨截面平均值直接混算、塑性大变形或未知角度符号的盲代。`tau_xy` 和 `gamma_xy` 的正负号必须按同一约定；应变花常用 `microstrain`，应力用 `MPa`；典型题入口是主应力、莫尔圆、弯扭组合和应变花反算。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 已知应力单元求主应力 | 写分量 -> 套公式 -> 排序 -> 求最大剪应力 |
| 求斜截面应力 | 明确角度正方向 -> 套变换公式 |
| 莫尔圆 | 定两点 -> 找圆心半径 -> 读主应力 |
| 弯扭组合危险点 | 先求 `M,T` -> 算 `sigma,tau` -> 求主应力 |
| 应变花 | 由 `epsilon_0,epsilon_45,epsilon_90` 求 `epsilon_x,epsilon_y,gamma_xy` -> 主应变 -> 胡克定律反推应力 |

## 7. 易错点清单

- 没有先由内力求危险点应力，直接套主应力。
- 平面应力中 `sigma_z=0`，但三维排序时仍要考虑。
- 主应力面和最大剪应力面相差 45 度。
- `tau_xy` 符号影响主方向。
- 应变变换中最大坑是工程剪应变要以 `gamma/2` 进入莫尔圆半径。
- 应变花题要先确认贴片角度，不能把 `epsilon_45` 当主应变。

## 8. 最低掌握标准

- 会提取危险点 `sigma_x,tau_xy`。
- 会求主应力、主方向、最大剪应力。
- 会画基本莫尔圆。
- 会用平面应力胡克定律。
- 会由 0/45/90 度应变花求 `epsilon_x,epsilon_y,gamma_xy`。

## 9. 高分掌握标准

- 能把弯曲、扭转、拉压组合成一点应力状态。
- 能用解析法和莫尔圆互相校核。
- 能正确处理符号、角度和主应力排序。
- 能从应变花读数反推出主应变、主方向和主应力。

## 10. 三遍法学习任务

## 11. 考试模拟与变式训练

已知 `sigma_x=80 MPa`, `sigma_y=20 MPa`, `tau_xy=30 MPa`，求主应力和最大剪应力。  
变式：把 `tau_xy` 改为 `-30 MPa`，主应力大小不变，主方向改变。

应变花题：某点 0/45/90 度应变花读数为 `epsilon_0=400 microstrain`、`epsilon_45=250 microstrain`、`epsilon_90=-100 microstrain`。  
先得 `epsilon_x=400`、`epsilon_y=-100`、`gamma_xy=2epsilon_45-epsilon_x-epsilon_y=200 microstrain`。  
再用 `epsilon_1,2=(epsilon_x+epsilon_y)/2 +/- sqrt(((epsilon_x-epsilon_y)/2)^2+(gamma_xy/2)^2)` 求主应变。若给 `E,nu`，再用广义胡克定律反求主应力。
