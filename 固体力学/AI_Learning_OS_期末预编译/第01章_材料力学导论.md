# 第01章 材料力学导论

## 1. 章节一句话总纲

材料力学研究杆件在外力作用下的内力、应力、变形与失效风险，解题总流程是：受力图 -> 截面法 -> 平衡 -> 内力 -> 应力/变形 -> 强度、刚度或稳定校核。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 材料力学研究什么？ | 杆件、内力、应力、应变、强度、刚度、稳定性 |
| 为什么引入应力？ | 内力大小还不够，要看单位面积上的受力 |
| 强度、刚度、稳定性区别？ | 强度防破坏，刚度限变形，稳定防失稳 |
| 基本假设是什么？ | 连续、均匀、各向同性、小变形、线弹性 |

## 3. 前置知识

- 静力学平衡：`sum Fx=0`, `sum Fy=0`, `sum M=0`
- 力矩与力偶
- 截面法的基本想法
- 面积、长度、单位换算

## 4. 概念依赖图

```text
外力与约束 -> 受力图 -> 截面法 -> 内力 -> 应力/应变 -> 强度/刚度/稳定校核
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `sigma = F/A` | 平均应力定义 | 应力可平均化，`F` 是截面内力 | 把任意外力当 `F` |
| `epsilon = Delta l/l` | 线应变定义 | 小变形 | 忘记长度单位统一 |
| `sigma = E epsilon` | 胡克定律 | 线弹性范围 | 超过屈服仍套用 |
| `sigma_max <= [sigma]` | 许用应力法 | 许用应力已知 | 只算应力不比较 |

### 公式血缘审计补充

上表每个公式使用时都按这 5 项补齐：推导前提是“小变形、连续介质、线弹性或题目允许平均化”；不适用条件是“大变形、塑性阶段、明显应力集中且题目要求局部应力”；符号含义第一次出现必须写清；单位优先统一为 `N-mm-MPa`；典型题入口是平均应力、基础强度校核和概念辨析题。

## 6. 典型题型表

| 题型 | 标准动作 |
|---|---|
| 概念辨析 | 区分外力、内力、应力、变形 |
| 强度/刚度/稳定判断 | 看题目限制的是应力、位移还是失稳 |
| 平均应力计算 | 先确认截面内力，再用 `sigma=F/A` |
| 单位换算 | 常用 `1 MPa = 1 N/mm^2` |

## 7. 易错点清单

- 把内力当成外力的别名。
- 直接套 `sigma=F/A`，但没有证明 `F` 是截面内力。
- 把“不断裂”当成“设计合格”，漏掉刚度和稳定。
- 忘记多数公式默认小变形、线弹性。

## 8. 最低掌握标准

- 能说出材料力学三大任务：强度、刚度、稳定性。
- 能解释外力、内力、应力、应变、变形。
- 能按总流程启动一道简单题。

## 9. 高分掌握标准

- 能写出每个基本公式的来源和适用条件。
- 能在综合题开头主动补受力图和截面内力。
- 能识别隐藏假设：线弹性、小变形、杆件模型。

## 11. 考试模拟与变式训练

基础题：一杆受轴力 `F=20 kN`，截面面积 `A=200 mm^2`，求平均正应力。  
答案：`sigma=20000/200=100 MPa`。

变式：若 `[sigma]=80 MPa`，则 `100 MPa > 80 MPa`，强度不满足。
