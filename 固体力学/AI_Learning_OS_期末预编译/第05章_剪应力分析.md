# 第05章 弹性杆件横截面上剪应力分析

## 1. 章节一句话总纲

本章研究扭矩和剪力造成的剪应力，并补上扭转变形与刚度链：先由受力图、截面法求 `T` 或 `V`，再计算剪应力、扭转角或梁剪应力，最后做强度或刚度校核。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 圆轴扭转剪应力如何分布？ | 中心为零，外表面最大 |
| 梁横向剪应力如何计算？ | `tau=VQ/(Ib)` |
| 矩形截面剪应力哪里最大？ | 中性轴处，`tau_max=3V/(2A)` |
| 阶梯轴危险段如何判断？ | 同时看 `T` 和直径 |
| 圆轴扭转刚度如何校核？ | 用 `phi=TL/(GIp)` 或分段求和 |

## 3. 前置知识

- 扭矩图、剪力图。
- 极惯性矩 `Ip`、惯性矩 `I`、静矩 `Q`。
- 剪切胡克定律 `tau=G gamma`。

## 4. 概念依赖图

```text
外力偶/横向力 -> 受力图 -> 截面法 -> T/V -> tau=T rho/Ip 或 tau=VQ/(Ib) -> phi=int T/(GIp) dx -> 强度/刚度校核
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `tau=T rho/Ip` | 圆轴扭转平截面假设 | 圆截面弹性扭转 | 用于非圆截面 |
| `tau_max=TR/Ip=T/Wp` | 外表面 `rho=R` | 圆轴最大剪应力 | 忘记最大在外表面 |
| `Ip=pi d^4/32` | 实心圆几何性质 | 实心圆轴 | 直径单位错 |
| `phi=TL/(GIp)` | 扭转角微分关系积分 | 等截面圆轴、扭矩常量、线弹性 | 只校核剪应力，漏刚度 |
| `phi=sum Ti Li/(Gi Ipi)` | 分段扭转角叠加 | 阶梯轴或分段扭矩 | 各段扭矩、刚度混用 |
| `tau=VQ/(Ib)` | 梁剪应力公式 | 细长梁、弹性范围 | `Q` 取错部分 |
| `tau_max=3V/(2A)` | 矩形截面剪应力分布 | 矩形截面 | 当作平均剪应力 |

### 公式血缘审计补充

扭转公式默认圆截面、线弹性、小变形；梁剪应力公式默认细长梁和 Saint-Venant 假设。非圆轴扭转、薄壁开口截面自由扭转、短粗梁剪切变形显著时不能直接套用。`T,V,Q,I,b,Ip,G,L` 必须和具体截面/分段对应；`tau` 单位为 `N/mm^2=MPa`，`phi` 用弧度；典型题入口是圆轴强度/刚度、阶梯轴分段扭转、矩形梁和工字梁剪应力。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 圆轴扭转 | 画扭矩图 -> 找 `Tmax` -> `tau=T/Wp` |
| 圆轴扭转刚度 | 画扭矩图 -> 分段求 `phi=sum Ti Li/(Gi Ipi)` -> 比较 `[phi]` |
| 阶梯轴 | 分段求 `T,d,L,G` -> 各段算 `tau` 和 `phi` -> 强度、刚度都校核 |
| 矩形梁剪应力 | 画剪力图 -> 找 `Vmax` -> `tau_max=3V/(2A)` |
| 工字梁剪应力 | 找腹板处 `Q,b,I` -> `tau=VQ/(Ib)` |
| 剪切设计 | 由 `tau_max <= [tau]` 反求尺寸 |

## 7. 易错点清单

- 把弯矩最大处当剪应力最大处。
- 圆轴中心误认为剪应力最大。
- `Q` 不是剪力，是静矩。
- 阶梯轴只比较扭矩，忽略直径四次方影响。
- 扭转题只算 `tau`，漏掉题目要求的扭转角或单位长度扭转角。
- 阶梯轴扭转角要分段求和，不能用一个平均直径替代。

## 8. 最低掌握标准

- 会画扭矩图和剪力图。
- 会用圆轴扭转公式。
- 会用 `phi=TL/(GIp)` 或分段形式做扭转刚度校核。
- 会用 `tau=VQ/(Ib)` 和矩形截面最大剪应力公式。

## 9. 高分掌握标准

- 能判断剪应力最大位置。
- 能处理空心轴、阶梯轴、工字梁。
- 能完成“扭矩图 -> 最大剪应力 -> 扭转角 -> 强度/刚度结论”的闭环。
- 能综合正应力与剪应力进入强度理论。

## 11. 考试模拟与变式训练

实心圆轴受扭矩 `T`、直径 `d`：  
`Ip=pi d^4/32`，`tau_max=16T/(pi d^3)`。

扭转刚度变式：圆轴长度 `L`、剪切模量 `G`，若全长扭矩为 `T`，则  
`phi=TL/(GIp)`。若许用扭转角为 `[phi]`，必须同时满足 `phi <= [phi]`。  
阶梯轴变式：分段列 `Ti,Li,Gi,Ipi`，总扭转角 `phi=sum Ti Li/(Gi Ipi)`。

矩形梁剪力为 `V`，面积 `A`：  
`tau_max=3V/(2A)`。
