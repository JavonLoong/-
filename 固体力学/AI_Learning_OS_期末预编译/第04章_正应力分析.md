# 第04章 弹性杆件横截面上正应力分析

## 1. 章节一句话总纲

本章解决横截面正应力从哪里来、怎样分布、如何校核的问题：先由受力图和截面法求 `N` 或 `M`，再用 `sigma=N/A`、`sigma=My/I` 或叠加公式判断危险点。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 拉压正应力如何算？ | `sigma=N/A` |
| 弯曲正应力如何分布？ | 沿截面高度线性分布，中性轴处为零 |
| 中性轴、惯性矩、截面模量作用？ | `I` 控制分布，`W=I/ymax` 控制最大值 |
| 偏心拉压怎么做？ | 等效为轴力 `N` 与弯矩 `M=Ne` 叠加 |

## 3. 前置知识

- 支反力、内力图、截面法。
- 形心、惯性矩 `I`、截面模量 `W`。
- 胡克定律与平截面假设。

## 4. 概念依赖图

```text
外载 -> 受力图 -> 截面法 -> N/M -> sigma=N/A 或 sigma=My/I -> 危险点 -> 强度校核
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `sigma=N/A` | 轴向内力均匀分布 | 直杆轴向拉压 | 忽略应力集中 |
| `sigma=My/I` | 平截面假设 + 胡克定律 + 截面平衡 | 梁弯曲线弹性 | `y` 取错 |
| `sigma_max=M/W` | `y=ymax` | 边缘最大应力 | `I` 与 `W` 混用 |
| `sigma=N/A +/- My/I` | 线弹性叠加 | 拉压弯曲组合 | 符号乱 |

### 公式血缘审计补充

正应力公式的推导前提是平截面假设、小变形、线弹性和截面几何量已知；不适用于深梁、明显局部压溃、屈服后塑性分布。`M,y,I,W,N,A` 的方向和截面位置必须写明；若 `M` 用 `N*mm`、`I` 用 `mm^4`，则 `sigma` 为 `MPa`；典型题入口是弯矩图后求危险截面、偏心拉压和截面尺寸设计。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 拉压强度 | 求轴力图 -> 找 `Nmax` -> `sigma=N/A` |
| 梁弯曲正应力 | 求反力 -> 弯矩图 -> `Mmax` -> `sigma=M/W` |
| 指定点应力 | 求该截面 `M` -> 找该点 `y` -> `sigma=My/I` |
| 偏心拉压 | 等效 `N` 和 `M=Ne` -> 边缘点叠加 |
| 截面设计 | 由 `M/W <= [sigma]` 反求 `W` 或尺寸 |

## 7. 易错点清单

- 把剪力最大点当正应力最大点。
- `y` 不是到边界距离，而是到中性轴距离。
- 非对称截面未先找形心。
- 只算一个边缘点，漏掉另一侧。

## 8. 最低掌握标准

- 会用 `sigma=N/A`, `sigma=My/I`, `sigma=M/W`。
- 能从弯矩图找危险截面。
- 能做简单偏心受压。

## 9. 高分掌握标准

- 能解释弯曲正应力线性分布。
- 能处理组合拉压弯曲。
- 能根据强度条件反求尺寸或许用载荷。

## 10. 三遍法学习任务

| 遍数 | 任务 |
|---|---|
| 第一遍 | 默写正应力公式链 |
| 第二遍 | 轴拉压、纯弯曲、偏心受压各 2 题 |
| 第三遍 | 限时训练危险截面和危险点判断 |

## 11. 考试模拟与变式训练

简支梁中央受集中力 `P`：`Mmax=PL/4`，最大弯曲正应力 `sigma_max=Mmax/W`。  
变式：集中力移到距左端 `a`，先重新求支反力和弯矩图。

偏心压力 `P`、偏心距 `e`：`sigma=P/A +/- Pey/I`，检查是否出现拉应力。

## 12. 复习卡片

| 正面 | 背面 |
|---|---|
| 弯曲正应力最大在哪？ | 危险截面上距中性轴最远处 |
| `I` 与 `W` 区别？ | `I` 算指定点，`W=I/ymax` 算最大应力 |
| 偏心力如何等效？ | 轴力 `N=P` + 弯矩 `M=Pe` |
