# 第12章 动载荷与动应力概述

## 1. 章节一句话总纲

动载荷题把速度、加速度、冲击和突加载荷转化为等效内力或能量关系，再求最大动应力并进行强度校核。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 动载和静载区别？ | 惯性效应、能量转换、应力放大 |
| 突加载荷系数？ | `Kd=2` |
| 冲击怎么处理？ | 用能量法和静位移 `delta_st` |
| 加速构件怎么处理？ | 牛顿第二定律或达朗贝尔原理 |

## 3. 前置知识

- 静应力公式。
- 轴向变形和梁挠度。
- 动能、势能、功。
- `F=ma`。

## 4. 概念依赖图

```text
动载类型 -> 受力图/能量图 -> 等效动载或动载系数 -> 动内力 -> 动应力 -> 校核
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `Kd=sigma_d/sigma_st=P_d/P` | 动载放大定义 | 线弹性、同受力形式 | 与安全系数混淆 |
| `Kd=2` | 突加载荷能量法 | 瞬时从 0 加到 `P` | 当缓慢加载用 |
| `Kd=1+sqrt(1+2h/delta_st)` | 重物冲击能量守恒 | 线弹性冲击 | `delta_st` 取错 |
| `sigma_d=Kd sigma_st` | 线弹性比例 | 应力与载荷线性 | 超弹性范围套用 |
| `T=W(1+a/g)` | 牛顿第二定律 | 向上加速提升 | 加速度方向错 |

### 公式血缘审计补充

动载公式的前提是可等效为单自由度线弹性响应、阻尼忽略或按题意忽略；不适用于超过屈服后的冲击、复杂振动或多自由度耦合题直接套系数。`delta_st` 必须是同一载荷作为静载时的位移；`h,delta_st` 单位一致；典型题入口是突加载荷、重物冲击、加速吊索和动强度校核。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 突加载荷杆 | 求静应力 -> 乘 `Kd=2` |
| 重物冲击杆 | 求 `delta_st` -> 求 `Kd` -> `sigma_d` |
| 加速吊索 | 画受力图 -> `T-W=ma` -> `sigma=T/A` |
| 动强度校核 | 求最大动应力 -> 比 `[sigma]` |

## 7. 易错点清单

- 把突加载荷当静载。
- 冲击题把 `delta_st` 写成 `delta_d`。
- 漏掉落高对应的能量。
- 惯性力方向取错。
- 只算应力不做校核。

## 8. 最低掌握标准

- 能区分缓慢加载、突加载荷、冲击。
- 会用 `Kd=2` 和冲击系数。
- 会处理简单加速吊索。

## 9. 高分掌握标准

- 能从能量守恒解释冲击公式。
- 能正确求结构静位移。
- 能给出降低动应力的措施。

## 10. 三遍法学习任务

| 遍数 | 任务 |
|---|---|
| 第一遍 | 判断载荷类型并写对应公式 |
| 第二遍 | 3 道冲击题，重点求 `delta_st` |
| 第三遍 | 每题最后写强度校核和改进建议 |

## 11. 考试模拟与变式训练

数值题：重物重量 `W=1 kN` 从高度 `h=20 mm` 落到竖直杆端，杆长 `L=1000 mm`，面积 `A=100 mm^2`，弹性模量 `E=200000 MPa`，许用应力 `[sigma]=180 MPa`。

1. 静位移：`delta_st=WL/(EA)=1000*1000/(200000*100)=0.05 mm`。
2. 动载系数：`Kd=1+sqrt(1+2h/delta_st)=1+sqrt(1+40/0.05)=1+sqrt(801)`。
3. 最大动应力：`sigma_d=Kd W/A`。
4. 校核：若 `sigma_d <= 180 MPa`，强度满足；否则不满足。

变式：`h=0` 时成为突加载荷，`Kd=2`。

## 12. 复习卡片

| 正面 | 背面 |

| 突加载荷动载系数？ | `Kd=2` |

| 动应力校核？ | `sigma_d <= [sigma]` |
