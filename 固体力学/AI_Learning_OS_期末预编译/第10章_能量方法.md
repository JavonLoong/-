# 第10章 材料力学中的能量方法

## 1. 章节一句话总纲

能量方法把构件应变能作为桥梁，在先求内力函数的基础上，用卡氏定理或单位载荷法求位移、转角和静不定多余力。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 应变能怎么写？ | 按 `N,M,T,V` 分项积分 |
| 如何求指定点位移？ | 单位载荷法或卡氏定理 |
| 无载荷点位移怎么求？ | 加虚拟力，最后令其为零 |
| 转角怎么求？ | 加单位力矩 |
| 能量法第一步是什么？ | 仍然是受力图和内力函数 |

## 3. 前置知识

- 内力图和内力函数。
- 拉压、弯曲、扭转基本公式。
- 梁位移与刚度。
- 线弹性、小变形、叠加原理。

## 4. 概念依赖图

```text
受力图 -> 截面法 -> N(x)/M(x)/T(x) -> 应变能 U -> 卡氏定理/单位载荷法 -> 位移/转角
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `U_N=int N^2/(2EA) dx` | 轴向应变能 | 拉压杆 | 把外力当内力 |
| `U_M=int M^2/(2EI) dx` | 弯曲应变能 | 梁弯曲 | 写成 `M/(EI)` |
| `U_T=int T^2/(2GIp) dx` | 扭转应变能 | 圆轴扭转 | 漏 `GIp` |
| `U_V=int k V^2/(2GA) dx` | 剪切应变能 | 短粗梁或题目明确要求考虑剪切 | 细长梁常规题盲目加入 |
| `delta=int M m/(EI) dx + int N n/(EA) dx + int T t/(GIp) dx` | 虚功/单位载荷法 | 线弹性 | 单位力方向错 |
| `delta_i=partial U/partial P_i` | 卡氏定理 | 线弹性 | 忘记虚拟力最后令零 |

### 公式血缘审计补充

能量公式的前提是线弹性、小变形和内力表达正确；不适用于路径相关塑性、大变形或漏分段的结构。`N,M,T,V` 必须来自截面法，不能直接用外载替代；位移单位为长度、转角为弧度、能量为 `N*mm`；典型题入口是梁端位移、指定点位移、桁架节点位移和力法静不定。

剪切应变能口径：细长梁弯曲位移题通常忽略 `V` 项，除非题目写明“考虑剪切变形”、梁很短粗，或给出剪切修正系数 `k`。桁架、轴向杆和圆轴扭转题则按相应 `N` 或 `T` 项处理。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 卡氏定理求梁端位移 | 求 `M(x)` -> 写 `U` -> 对 `P` 求偏导 |
| 单位载荷法求挠度 | 原结构 `M` + 单位力结构 `m` -> 积分 |
| 求转角 | 加单位力矩 -> 积分 |
| 桁架节点位移 | 原杆力 `Ni` + 单位力杆力 `ni` -> `sum Ni ni li/(EA)` |
| 静不定力法 | 用能量法求柔度系数 |

## 7. 易错点清单

- 未分段写内力函数。
- 求无载荷点位移忘加虚拟力。
- 单位力方向与位移方向不一致。
- 转角题加了单位力而非单位力矩。
- 变截面题把 `EI` 提出积分号。

## 8. 最低掌握标准

- 会写拉压、弯曲、扭转应变能。
- 会用单位载荷法求简单挠度/转角。
- 会处理卡氏定理的虚拟力。

## 9. 高分掌握标准

- 能处理多段梁、组合变形、桁架位移。
- 能用能量法建立静不定协调方程。
- 能做量纲检查。

## 10. 三遍法学习任务

| 遍数 | 任务 |
|---|---|
| 第一遍 | 抄写四类应变能公式并标适用条件 |
| 第二遍 | 2 道卡氏定理题 + 2 道单位载荷题 |
| 第三遍 | 限时完成组合变形或桁架节点位移题 |

## 11. 考试模拟与变式训练

悬臂梁端部受 `P`，取自由端为原点，`M=Px`：  
`U=int_0^L P^2 x^2/(2EI) dx=P^2 L^3/(6EI)`。  
`delta=partial U/partial P=PL^3/(3EI)`。

变式：求转角时在自由端加虚拟力矩。

## 12. 复习卡片

| 正面 | 背面 |
|---|---|
| 弯曲应变能？ | `U=int M^2/(2EI) dx` |
| 单位载荷法核心？ | `delta=int M m/(EI) dx` 等 |
| 求转角加什么？ | 单位力矩 |
| 无载荷点位移？ | 加虚拟力，求完令零 |
