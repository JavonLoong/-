# 第13章 疲劳强度与构件寿命估算概述

## 1. 章节一句话总纲

疲劳问题研究构件在循环应力下的裂纹萌生、扩展和断裂，即使最大应力低于静强度，也必须按应力幅、平均应力和寿命曲线进行校核。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 为什么低于屈服也会断？ | 循环载荷导致裂纹逐步扩展 |
| 循环应力怎么描述？ | `sigma_max`, `sigma_min`, `sigma_a`, `sigma_m`, `R` |
| 寿命怎么估？ | S-N 曲线或疲劳极限 |
| 平均应力怎么修正？ | Goodman 等准则 |
| 什么因素最危险？ | 缺口、粗糙表面、尺寸大、残余拉应力、腐蚀 |

## 3. 前置知识

- 内力图和危险截面。
- `sigma=N/A`, `sigma=My/I`, `tau=T rho/Ip`。
- 弯扭组合强度理论。
- 材料强度指标。

## 4. 概念依赖图

```text
循环外载 -> 受力图 -> 危险截面 -> 循环内力 -> sigma_max/min -> sigma_a/sigma_m -> S-N/Goodman -> 寿命或安全系数
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `sigma_a=(sigma_max-sigma_min)/2` | 应力幅定义 | 循环应力 | 与平均应力混 |
| `sigma_m=(sigma_max+sigma_min)/2` | 平均应力定义 | 循环应力 | 符号错 |
| `R=sigma_min/sigma_max` | 应力比 | 描述循环类型 | 忽略压应力负号 |
| `sigma_a/sigma_-1 + sigma_m/sigma_b <= 1/n` | Goodman 准则 | 拉伸平均应力修正 | 静强度代替疲劳 |
| `sigma_-1,修=k_a k_b k_c k_d / K_f * sigma_-1` | 表面、尺寸、载荷、可靠性、缺口修正的简化乘积 | 实际构件疲劳极限估算 | 用一个 `k` 淹没所有修正来源 |

### 公式血缘审计补充

疲劳公式的前提是循环应力能由内力分析得到，材料 S-N 或疲劳极限数据适用；不适用于一次静载破坏、低周塑性疲劳未给模型、腐蚀/高温无修正数据的问题。`sigma_max,sigma_min,sigma_a,sigma_m,R` 必须保留拉压符号；应力单位通常为 `MPa`，寿命为循环次数；典型题入口是轴类弯扭疲劳、缺口疲劳和 S-N 寿命估算。

疲劳修正拆解：`k_a` 常代表表面质量，`k_b` 代表尺寸，`k_c` 代表载荷类型，`k_d` 代表可靠性或环境，`K_f` 代表疲劳应力集中。若题目只给总修正系数 `k`，可用 `sigma_-1,修=k sigma_-1`；若分别给各项，必须逐项代入，不能把缺口影响漏掉。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 拉压疲劳 | 求循环轴力 -> 求 `sigma_max/min` -> `sigma_a/m` -> 校核 |
| 弯曲疲劳 | 画弯矩图 -> 找危险截面 -> 求交变弯曲应力 |
| 弯扭组合疲劳 | 求 `M,T` -> 等效应力幅/平均值 -> 疲劳准则 |
| 缺口疲劳 | 名义应力 -> 乘疲劳应力集中系数 `Kf` |
| 寿命估算 | 实际应力幅 -> 查 S-N 曲线 -> 与要求寿命比 |

## 7. 易错点清单

- 把疲劳强度当静强度。
- 没有先求循环内力。
- `sigma_a` 和 `sigma_m` 算反。
- 脉动循环误当对称循环。
- 缺口题不考虑应力集中。

## 8. 最低掌握标准

- 能解释疲劳三阶段。
- 能求 `sigma_a`, `sigma_m`, `R`。
- 能读简单 S-N 曲线。
- 能判断疲劳危险因素。

## 9. 高分掌握标准

- 能处理弯扭组合疲劳。
- 能说明疲劳极限修正，并区分表面、尺寸、载荷、可靠性和缺口来源。
- 能提出改进方案：圆角、抛光、表面强化、降载。

## 10. 三遍法学习任务

| 遍数 | 任务 |
|---|---|
| 第一遍 | 做循环应力参数速查卡 |
| 第二遍 | 拉压、弯曲、弯扭组合各 1-2 题 |
| 第三遍 | 限时做轴类疲劳综合题 |

## 11. 考试模拟与变式训练

阶梯轴受交变弯矩和恒定扭矩：  
先画轴受力图，找台阶圆角危险截面；再求 `M,T`；由 `sigma=Mc/I`, `tau=Tc/Ip` 得应力；计算 `sigma_a,sigma_m` 并考虑 `Kf`；用 Goodman 或题给准则求安全系数。
