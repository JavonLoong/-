# 第08章 梁的位移分析与刚度设计

## 1. 章节一句话总纲

本章在强度校核之外，研究梁的转角和挠度：由弯矩方程建立 `EI v''=M(x)`，通过边界条件和连续条件求位移，并用刚度条件校核。

## 2. 核心问题

| 问题 | 考试抓手 |
|---|---|
| 为什么还要校核刚度？ | 强度满足但变形过大也不合格 |
| 如何由弯矩求挠度？ | `EI v''=M(x)` 两次积分 |
| 边界条件怎么写？ | 固端 `v=0, theta=0`，简支端 `v=0` |
| 最大挠度在哪里？ | 对称梁常在对称轴；一般要由 `theta=0` 判断 |
| 设计时用什么量？ | 刚度设计多反求 `I` |

## 3. 前置知识

- 支反力、剪力图、弯矩图。
- 弯曲正应力 `sigma=My/I`。
- 惯性矩 `I`。
- 分段积分和边界条件。

## 4. 概念依赖图

```text
梁外载 -> 受力图 -> 支反力 -> 截面法 -> M(x) -> EI v''=M(x) -> theta/v -> 刚度校核
```

## 5. 公式血缘表

| 公式 | 来源 | 适用条件 | 常见误用 |
|---|---|---|---|
| `EI v''=M(x)` | 小挠度曲率 + 弯曲曲率公式 | 线弹性梁、小变形 | 弯矩符号不统一 |
| `theta=v'` | 挠曲线几何关系 | 小转角 | 与挠度混淆 |
| `delta=int M m/(EI) dx` | 单位载荷法 | 求指定点位移 | 单位力方向取错 |
| 固端 `v=0,theta=0` | 约束条件 | 固定端 | 漏转角为零 |
| 简支端 `v=0` | 约束条件 | 铰支/滚支 | 误令转角为零 |

### 公式血缘审计补充

梁位移公式的推导前提是线弹性、小挠度、弯曲变形为主；不适用于大挠度、塑性铰或短粗梁剪切变形主导的问题。`v,theta,M,E,I,x` 的正方向要和弯矩方程一致；`EI v''=M` 中若 `M` 为 `N*mm`、`EI` 为 `N*mm^2`，则 `v''` 为 `1/mm`；典型题入口是悬臂梁、简支梁、分段载荷和刚度设计。

## 6. 典型题型表

| 题型 | 标准流程 |
|---|---|
| 悬臂梁端力 | 写 `M(x)` -> 两次积分 -> 固端条件 |
| 简支梁均布载荷 | 求反力 -> `M(x)` -> 两端挠度为零，对称点转角为零 |
| 分段载荷梁 | 分段写 `M(x)` -> 每段积分 -> 连接处 `v,theta` 连续 |
| 指定点位移 | 原结构弯矩 `M` + 单位力弯矩 `m` -> 积分 |
| 刚度设计 | 求 `delta_max` -> 令 `delta_max <= [delta]` |

## 7. 易错点清单

- 未先求支反力，弯矩函数全错。
- 简支端转角误设为零。
- 分段梁漏连续条件。
- 最大弯矩位置不一定是最大挠度位置。
- 强度设计用 `W`，刚度设计用 `I`，不能混。

## 8. 最低掌握标准

- 能写出简单梁的 `M(x)`。
- 能用 `EI v''=M(x)` 积分求转角和挠度。
- 能背常见梁挠度公式。
- 能做刚度校核。

## 9. 高分掌握标准

- 能处理分段载荷和连续条件。
- 能用叠加法和单位载荷法。
- 能判断强度控制还是刚度控制。

## 11. 考试模拟与变式训练

悬臂梁端部集中力 `P`：  
`theta=PL^2/(2EI)`，`delta=PL^3/(3EI)`。

简支梁跨中力 `P`：  
`delta_max=PL^3/(48EI)`。

简支梁全跨均布载荷 `q`：  
`delta_max=5qL^4/(384EI)`。
