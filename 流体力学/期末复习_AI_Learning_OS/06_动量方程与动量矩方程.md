## 章节一句话总纲

动量方程与动量矩方程的核心不是背一个“力等于 rho QV 差”的壳子，而是先把控制体、坐标正方向、进出口速度矢量和所有外力画清楚，再用“动量通量的净流出 = 外力合力”求边界反力、喷流冲击力、弯管支座力和叶轮转矩。

## 期末考试抓手

1. 一看到“冲击力、弯管受力、法兰螺栓、喷嘴、支座反力、射流冲击、叶轮转矩”，优先想到动量方程或动量矩方程。
2. 动量题先画图，不先代公式：控制体边界、入口/出口、外法线、坐标正方向、速度矢量、压力力、重力、壁面/支座反力必须同时出现。
3. 定常一维总流常用式：`sum F = sum_out beta rho Q V - sum_in beta rho Q V`。这是矢量式，算题时必须写成 x、y、z 分量。
4. 外力项按“谁作用在控制体内流体上”列：压强力（也常写作压力力）、重力、壁面对流体的作用力、支座/法兰传来的作用力、剪切力。最后若题问“流体对管壁/平板的力”，要取反作用力。
5. 压力力用表压最稳：若控制面外侧都受同一大气压，表压会自动抵消；用绝压也可以，但必须把大气压作用在所有相关外表面上的力一并列入。
6. 动量通量符号不是“入口正出口负”，而是外法线约定下 `V·n` 的符号：出口为正，入口为负；若改写成“出 - 入”，也必须保持速度分量方向。
7. 喷流冲击平板抓三点：射流自由表面通常取环境表压为零；平板法向速度分量改变产生冲击力；光滑平板切向若忽略摩擦，切向外力近似为零。
8. 弯管题抓两次反号：先求管壁对水的力，再取反得到水对管的力；若还要求支座反力，再对管件整体平衡取反或相加。
9. 动量矩题抓“切向速度分量”：对转轴的力矩只由 `r V_theta` 的角动量通量变化决定，径向速度本身不直接产生绕轴转矩。
10. 与连续方程/伯努利联用的顺序通常是：连续方程求速度关系，伯努利/能量方程求速度或压强，动量方程求力，动量矩方程求转矩/功率。

## 核心问题

1. 为什么控制体动量方程比直接跟踪系统更适合求管壁、平板、喷嘴受到的力？
2. 动量方程中的 `sum F` 到底是哪些力？为什么支座反力、压力力、重力不能随手漏掉？
3. 为什么动量通量是矢量通量，不能只用速度大小？
4. 进出口速度方向改变时，为什么即使速度大小不变也会产生力？
5. 为什么喷流冲击自由平板常可把压力力取为零，而弯管题不能随便忽略压力力？
6. 为什么用表压通常比用绝压更省事？什么时候用绝压会把大气压力漏算？
7. 如何从“管壁对流体的力”转换到“流体对管壁/支座的力”？
8. 动量矩方程与普通动量方程的差别是什么？为什么旋转机械看入口/出口切向速度？
9. 连续方程、伯努利方程和动量方程在同一道题中各负责求什么？
10. 动量方程能求总作用力，但为什么不能告诉你边界上力的分布？

| --- | --- | --- |
| 系统与控制体 | 把系统动量定理转成控制体方程 | 不知道力是作用在流体上还是管壁上 |
| 雷诺输运定理 | 推出动量方程和动量矩方程积分形式 | 只背稳态公式，遇到非定常储存项就空白 |
| 连续方程 | 求 `Q`、截面平均速度、流量分配 | 动量方程中 `rho QV` 的 `Q` 写错 |
| 伯努利/能量方程 | 求喷口速度、截面压强、水头损失 | 动量题力列对了但速度/压强来源不明 |
| 压强与表压/绝压 | 列压力力 `pA`，处理大气压抵消 | 用绝压只算开口压力，漏掉大气压外力 |
| 速度矢量分解 | 写动量通量的 x/y 分量 | 把出口速度大小直接相减，方向错 |
| 受力分析 | 外力合力必须包括压力、重力、支座/壁面反力 | 少一个外力导致结果方向或大小严重错 |
| 反作用力 | 从“壁面对流体”转为“流体对壁面” | 结果差一个负号 |
| 力矩与叉乘 | 动量矩方程求转矩、功率、扬程 | 把径向速度也当成绕轴转矩来源 |
| 动量修正系数 | 非均匀截面用 `beta` 修正动量通量 | 把最大速度当平均速度，或默认 `beta=1` 不说明 |

## 三遍法

### 第一遍：直觉理解

动量方程的直觉是“速度矢量被改了，必然有外力在改它”。

水流进弯管前向右，出弯管后向上，即使速度大小不变，动量矢量方向也变了；这个变化必须由压力、重力、管壁作用力共同提供。反过来，水也会给管壁一个反作用力，这就是弯管支座为什么受力。

射流冲击平板也是同一件事：入流本来沿 x 方向有动量，撞上平板后沿板面流走，x 方向动量减少。谁让 x 方向动量减少？平板给水一个反向力。水给平板的冲击力就是这个力的反作用。

动量矩方程的直觉是“切向动量改变，必然有转矩”。流体进出叶轮时，如果绝对速度的切向分量变了，绕轴角动量通量就变了，轴就必须给流体转矩，或流体反过来给轴转矩。

最容易混的四对关系：

1. 控制体内流体受力 vs 流体对固体的作用力：方向相反。
2. 速度大小 vs 速度矢量：动量方程看矢量。
3. 入口/出口标签 vs 分量正负：出口速度的 x 分量也可能是负。
4. 表压 vs 绝压：表压省掉的是共同大气压，不是省掉所有压力力。

### 第二遍：公式推导

从系统动量定理出发：

```text
系统动量随时间的变化率 = 作用在系统上的外力合力
d/dt int_sys V dm = sum F
```

用雷诺输运定理把系统表达转为控制体表达，令广延量为动量 `B=mV`，单位质量广延量为 `b=V`：

```text
partial_t int_CV rho V dV
+ oint_CS rho V (V·n)dA
= sum F
```

第一项是控制体内动量储存变化率，第二项是穿过控制面的动量净流出率。定常时第一项为零：

```text
sum F = oint_CS rho V (V·n)dA
```

若只有若干一维进出口，且截面速度可用平均速度代表：

```text
sum F = sum_out beta rho Q V - sum_in beta rho Q V
```

投影到 x 轴：

```text
sum F_x = sum_out beta rho Q V_x - sum_in beta rho Q V_x
```

这一步是考场得分核心。必须先规定 x 正方向，再把每个外力和每个进出口速度矢量投影到 x 轴。y、z 方向同理。

动量矩方程同样来自系统动量矩定理。对点 O 取矩，系统角动量为：

```text
H_O = int_sys r x V dm
```

用雷诺输运定理：

```text
partial_t int_CV rho (r x V)dV
+ oint_CS rho (r x V)(V·n)dA
= sum M_O
```

定常、入口出口均匀、绕 z 轴取矩：

```text
T_z = sum_out rho Q r V_theta - sum_in rho Q r V_theta
```

再乘以角速度 `omega` 得功率：

```text
P = T_z omega = dot m (u_2 V_theta2 - u_1 V_theta1)
```

其中 `u=omega r` 是叶轮圆周速度。泵/压缩机向流体输入能量时，按所取正方向通常表现为轴对流体做正功；涡轮/水轮机从流体取能时符号相反，具体仍以题目坐标和转向为准。

### 第三遍：题目应用

动量题标准动作：

1. 写研究对象：控制体取“流体”还是“流体+固体”。期末最常用是控制体只取流体。
2. 画控制体：截面要取在流速/压强可近似均匀或渐变处；喷流题控制面要包住全部散射水流。
3. 标坐标正方向：不要让正方向在解题中途改变。
4. 标速度矢量：入口/出口速度都画箭头，再写分量，如 `V_1x=V_1`、`V_2x=V_2 cos theta`。
5. 标外力：压强力 `pA`、重力 `G`、壁面/支座对流体的作用力 `R`、剪切力或其他已知力。
6. 决定压强量度：优先用表压；若用绝压，必须补全大气压对外表面的作用。
7. 列连续方程：求 `Q`、`V`、分流量，必要时判断不可压/定常。
8. 列伯努利/能量方程：求喷口速度、截面压强或水头差；有损失则写损失项。
9. 列动量分量方程：按 `出 - 入 = 外力`，不要在同一题里混用符号约定。
10. 取反作用力：题问“水对管”“射流对板”“支座受力”时，和“管/板对水”的力方向相反。
11. 检查量纲和极限：`rho QV` 是 N；速度变成 0 时冲击力应变小；180 度反向偏转比 90 度平板力大。

动量矩题标准动作：

1. 选取矩点/转轴，规定正转向。
2. 画入口/出口半径 `r_1,r_2` 和绝对速度切向分量 `V_theta1,V_theta2`。
3. 写 `T_z = 出口角动量通量 - 入口角动量通量`。
4. 用 `P=T omega` 求功率，或用 `H=P/(rho g Q)` 求理论扬程。
5. 分清绝对速度 `V`、叶轮圆周速度 `u`、相对速度 `W`。考试通用补充：速度三角形中 `V = u + W`。

## 易错点与反例

1. 错误：动量方程里 `sum F` 是流体对固体的力。  
   反例：控制体取弯管内水流时，`sum F` 是压力、重力、管壁对水的作用力；水对弯管的力要取反。

2. 错误：只要是出口，速度分量就写正。  
   反例：若出口速度朝负 x 方向，虽然它是出口，`V_x` 仍为负；“出口为正”只适用于通量的流出/流入拆项，不适用于坐标分量。

3. 错误：动量通量写成 `rho Q V` 后就可以丢掉箭头。  
   反例：90 度弯管中入口和出口速度大小相等，但方向不同，动量变化不为零。

4. 错误：喷流冲击平板一律 `2rho QV`。  
   反例：正冲固定平板后水沿板面流走，入流方向动量变为 0，冲击力为 `rho QV`；只有流动近似反向折回 180 度时才可能接近 `2rho QV`。

5. 错误：自由射流题也要列大气压力力。  
   反例：射流外边界与大气相通，若全程用表压，环境压力为 0；但若用绝压，必须把大气压力在整个控制面上的作用算全，否则更容易错。

6. 错误：弯管题可以忽略压力力，只用动量变化求支座反力。  
   反例：有压管内入口/出口压力 `p_1A_1,p_2A_2` 常与动量项同量级，漏掉会让支座力严重偏差。

7. 错误：重力总能忽略。  
   反例：水平弯管 y 方向可不列重力，但竖直弯管、挑流坎、动量实验竖直射流都可能需要重力或水体重量。

8. 错误：绝压和表压混用。  
   反例：入口用绝压 `p_abs A`，出口大气处又取 0，会多算一个 `p_a A`；期末管路受力题一般统一用表压最稳。

9. 错误：动量方程能给出边界上每一点的压力分布。  
   反例：恒定总流动量方程只能给合力或合力矩，朱德军课件也强调它避开内部细节；若问分布，需要更细的压力/粘性分析。

10. 错误：先用伯努利求力。  
    反例：伯努利给能量/压强/速度关系，不能直接给管壁总反力；力必须由动量方程或静力压力积分给出。

11. 错误：动量矩方程用径向速度算转矩。  
    反例：绕轴角动量通量是 `r V_theta`，纯径向速度 `V_r` 不直接贡献绕轴角动量；它通过流量影响 `rho Q`，不是切向项。

12. 错误：叶轮入口无预旋就表示入口速度为零。  
    反例：无预旋通常指入口绝对速度切向分量 `V_theta1=0`，径向/轴向流量速度仍存在。此条为考试通用补充。

13. 错误：已知流量就不需要连续方程。  
    反例：变径弯管给的是某一截面速度或直径，另一个截面的速度必须先由连续方程求出，否则动量通量错。

14. 错误：动量修正系数和能量修正系数是同一个。  
    反例：动量通量用 `beta`，能量方程速度水头用 `alpha`；层流中两者数值也不同。

## 最低掌握标准/高分掌握标准

最低掌握标准：

1. 能写出控制体动量方程积分形式，并解释储存项、净流出项、外力项。
2. 能把定常总流动量方程写成 x/y 分量形式。
3. 能在喷流冲击平板题中正确得到 `F=rho QV` 的方向和反作用力。
4. 能在弯管题中列出压力力、重力、壁面/支座反力、动量通量。
5. 能说明为什么入口速度通量为流入、出口为流出，并能处理速度分量正负。
6. 能统一使用表压或绝压，不混用。
7. 能说出动量矩方程关注 `r x V`，旋转机械中切向速度分量决定转矩。

高分掌握标准：

1. 能根据题目主动选择控制体：只含流体、含流固整体、或包住全部射流散开区域。
2. 能先用连续方程和伯努利方程补齐速度/压强，再用动量方程求力。
3. 能处理二维弯管/挑流坎的 x、y 分量，并把“壁面对流体”和“流体对壁面”转换清楚。
4. 能解释表压法为什么可行，且知道绝压法必须补全大气压力作用。
5. 能在非均匀截面或题目提到修正系数时正确使用 `beta`，不与能量修正系数 `alpha` 混淆。
6. 能写出动量矩方程并用于叶轮/旋转喷头的转矩、功率、理论扬程判断。
7. 能用极限情形检查答案：无流量则力为零，速度加倍动量力约增为四倍，偏转角增大冲击力增大。

要求：

## 复习卡片

### 卡片 1：动量方程一句话

问：控制体动量方程的物理意义是什么？

答：控制体内动量储存变化率 + 穿过控制面的动量净流出率 = 作用在控制体内流体上的外力合力。

### 卡片 2：定常总流公式

问：定常一维总流动量方程怎么写？

答：`sum F = sum_out beta rho Q V - sum_in beta rho Q V`，这是矢量式，实际题目要投影到坐标轴。

### 卡片 3：外力清单

问：动量方程中的外力通常包括哪些？

答：压强力、重力、壁面/支座对流体的作用力、剪切力以及题目给出的其他外力。

### 卡片 4：表压 vs 绝压

问：弯管/喷嘴受力题为什么常用表压？

答：如果控制体外侧同受大气压，使用表压可把共同大气压抵消；用绝压时必须把大气压对所有相关面的作用列全。

### 卡片 5：通量符号

问：入口为什么在积分通量中为负？

答：控制面法向取外法向，入口速度指向控制体内，与外法向相反，所以 `V·n<0`；改写一维式时表现为“出 - 入”。

### 卡片 6：速度分量

问：出口速度分量一定为正吗？

答：不一定。出口只说明通量为流出，速度分量正负取决于坐标轴方向。

### 卡片 7：射流正冲平板

问：水平射流正冲固定平板，忽略损失时冲击力是多少？

答：水对平板的力沿来流方向，大小 `rho QV = rho A V^2`。

### 卡片 8：弯管反力

问：弯管题中求得 `R` 后为什么常要取反？

答：动量方程中 `R` 常设为管壁对流体的力；题目若问水对管壁或支座受力，需要用牛顿第三定律取反。

### 卡片 9：动量修正系数

问：什么时候不能随便取 `beta=1`？

答：截面速度分布明显非均匀，或题目要求考虑动量修正系数时；渐变流简化题常可说明近似取 `beta=1`。

### 卡片 10：动量矩方程

问：动量矩方程看什么通量？

答：看角动量通量 `rho(r x V)(V·n)`；绕轴一维时核心是 `rho Q r V_theta`。

### 卡片 11：叶轮入口概念

问：叶轮转矩为什么看绝对速度切向分量？

答：控制体角动量是相对惯性坐标系的 `r x V`，所以用绝对速度 `V` 的切向分量，而不是相对叶片速度。

### 卡片 12：联用顺序

问：连续方程、伯努利方程、动量方程在综合题中怎么分工？

答：连续方程定流量和速度关系，伯努利/能量方程定速度或压强，动量方程定合力或支座反力。
