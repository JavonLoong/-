### 抢分优先级

1. 先写“读数 -> 物理量”：液面高差不是压强，砝码质量不是力，量水箱水位差不是流量，测压管高度不是局部损失系数。
2. 再写“方程适用条件”：静水看是否静止同种连通；能量方程看定常、不可压、缓变断面、损失项；动量方程看控制体和外力。
3. 计算前统一单位：cm -> m，g -> kg，kPa -> Pa，L/min -> m3/s，水头 m 与压强 Pa 不混写。
4. 报结果时带单位和有效数字：实验题结果通常保留 3 位有效数字或与原始读数精度相配。
5. 图表题必须写坐标、单位、趋势和异常点处理：不是只画散点，而要说明斜率、拟合区间、物理意义。

│  ├─ 静水总压力：M0 = m g L0 -> P实测 = M0/L1

│  ├─ 总压力理论值与力臂
│  └─ 喷嘴出口速度、射流重力修正
└─ 温度 T

## 三遍法

### 第一遍：直觉理解

实验题不是先背公式，而是先把“表上读到的数”翻译成“方程能吃的物理量”。

1. 测压管液面高度是水头，不是压强；要乘 `rho g` 才是压强差。
2. 量水箱上升高度是体积变化，不是流量；要乘水箱面积再除以时间。
3. 砝码质量是质量，不是力；要乘 `g`，还要看是否有杠杆臂。
4. 两个测点的水头差不一定就是损失；若管径变化，还要把速度水头差放回能量方程。
5. 损失系数没有单位，但它绑定了一个参考速度水头；参考速度不写，答案不完整。
6. 综合题本质是分工：连续方程管流量，能量方程管压强和速度，动量方程管力。

### 第二遍：公式推导

#### 1. 从测压管读数到压强

静止同种液体中：

```text
z + p/(rho g) = C
```

所以两点之间：

```text
p2 - p1 = rho g (z1 - z2)
```

若测压管一端通大气，测管液面处表压为 0，测点压强可由测管液面相对测点的高度差得到。

#### 2. 从体积法到速度和雷诺数

量水箱体积增量：

```text
Delta V = A_tank Delta H
```

流量：

```text
Q = Delta V / t
```

管内平均速度：

```text
v = Q / A_pipe = 4Q/(pi d^2)
```

雷诺数：

```text
Re = v d / nu
```

这条链条是文透里、动量、沿程、局部损失实验的共同底座。

#### 3. 从能量方程到文透里和损失

实际总流能量方程：

```text
z1 + p1/(rho g) + alpha1 v1^2/(2g)
= z2 + p2/(rho g) + alpha2 v2^2/(2g) + h_w
```

文透里理论流量率定先忽略损失，并由连续方程消去 `v1`：

```text
Q理 = K sqrt(Delta h)
mu = Q实 / Q理
```

沿程损失在等直径直管中速度水头相同，常直接由测压管水头差得到：

```text
hf = H1 - H2
lambda = hf / ((l/d) v^2/(2g))
```

局部损失若前后管径不同，必须保留速度水头差：

```text
hj = H1 - H2 + (alpha1 v1^2 - alpha2 v2^2)/(2g)
zeta = hj / (v_ref^2/(2g))
```

#### 4. 从动量方程到射流冲击力

取射流转向前后的水流为控制体，沿来流方向列：

```text
sum F_x = rho Q (V_out,x - V_in,x)
```

平板使出流来流方向速度分量约为 0，则板对水的力与水对板的力大小为：

```text
R = rho Q v
```

曲面板偏转角为 `beta` 时：

```text
R = rho Q v (1 - cos beta)
```

竖直向上射流要先用能量修正冲击速度：

```text
v_hit = sqrt(v0^2 - 2gz)
R = rho Q v_hit (1 - cos beta)
```

### 第三遍：题目应用

实验/综合题标准动作：

1. 圈出原始读数：液面高程、砝码、时间、尺寸、温度、压力表读数。
2. 写单位换算：cm、mm、g、kPa、L/min 先转成统一单位。
3. 写仪器到物理量的翻译链：`Delta h -> Delta p`，`Delta H/t -> Q`，`m -> mg`。
4. 判压强基准：全题用表压或全用绝压；自由液面和出口通常用表压 0。
5. 选控制体或断面：能量题取渐变断面，动量题取包住流向变化的控制体。
6. 列主方程：静水方程、连续方程、能量方程、动量方程、损失定义。
7. 代入数值前做量纲检查：水头 m、压强 Pa、力 N、流量 m3/s、系数无量纲。
8. 写误差/图表结论：相对误差、曲线趋势、斜率物理意义、异常点原因。
9. 报结果：单位、有效数字、方向或作用点齐全。
10. 反查极限：流量越大损失应增大；文透里压差越大流量越大；突扩损失为正；实测系数通常不应无理由为负。

## 易错点与反例

1. 错误：把测压管液面读数直接当压强。
   反例：测压管显示的是 `z+p/(rho g)` 的水头位置；压强差还要乘 `rho g`。
   避坑：先写“读数单位是 cm 水柱”，再换成 Pa 或 N/cm2。

2. 错误：同一水平面永远等压。
   反例：不同液体、不连通、非静止或有加速度时不能直接等压。
   避坑：写出“同种、连通、静止、仅重力场”。

3. 错误：表压和绝压混用。
   反例：自由液面取表压 0，而压力表读数又被加上大气压，能量方程会多出一个 `p_a`。
   避坑：每道题开头写“全用表压”或“全用绝压”。

4. 错误：砝码 g 数直接当 N。
   反例：100 g 砝码对应约 `0.981 N`，不是 `100 N`。
   避坑：质量 `m` 先转 kg，再乘 `g`。

5. 错误：体积法流量忘除时间。
   反例：量水箱水位升高 10 cm 只是体积变化，时间不同流量不同。
   避坑：固定写 `Q=A Delta H/t`。

6. 错误：文透里 `Q` 与 `Delta h` 成正比。
   反例：理论式是 `Q=K sqrt(Delta h)`，压差水头加倍，流量不是加倍。
   避坑：看到压差式流量计，立刻写平方根。

7. 错误：沿程损失系数 `lambda` 有单位。
   反例：`hf=lambda(l/d)v^2/(2g)` 中除 `hf` 和速度水头外其余都是无量纲。
   避坑：系数结果后不写 m、Pa。

8. 错误：局部损失只等于两测压管水头差。
   反例：突扩/突缩前后管径不同，速度水头也变，必须用能量方程算 `hj`。
   避坑：先看 `d1=d2` 是否成立。

9. 错误：`zeta` 不说明参考速度。
   反例：突扩可写 `zeta1` 或 `zeta2`，分别对应上游或下游速度水头，数值差很多。
   避坑：答案写成 `zeta_1=... based on v1` 或 `zeta_2=... based on v2`。

10. 错误：动量方程求出的力就是题目问的力。
    反例：控制体取水流时，方程中的壁面力是“固体对水”的力；题问“水对固体”需取反。
    避坑：在答案末尾写“由牛顿第三定律，所求力方向相反”。

11. 错误：误差分析只写“人为误差、仪器误差”。
    反例：实验指导书明确强调读数稳定、排气调平、水流恒定、喷嘴对准、断面选择、流区筛选。
    避坑：每个实验至少写一个对应机理的误差源。

12. 错误：双对数图不分流区。
    反例：沿程实验层流 `hf` 与 `v` 近似一次方，紊流是 `1.75~2.0` 次方，混合拟合没有物理意义。
    避坑：先按 `Re` 划分层流/过渡/紊流，再拟合。

13. 错误：有效数字越多越好。
    反例：测压管读到 0.1 cm，最后损失系数写 0.582394 没有实验意义。
    避坑：结果通常保留 3 位有效数字，误差百分比保留 1-2 位小数。

## 综合题解题流程

```text
1. 审题分类
   静水 / 能量 / 动量 / 损失 / 实验数据 / 多方程综合

2. 标注已知量
   几何尺寸、液面高、压力表、流量、时间、质量、温度、损失系数

3. 统一基准
   高程基准、压强基准、单位系统、坐标方向、控制体边界

4. 翻译读数
   Delta h -> Delta p 或水头差
   Delta V/t -> Q
   m -> mg
   T -> nu 或密度修正

5. 先连续
   Q=Av，必要时用直径平方关系

6. 再能量
   理想时写伯努利；实际管路写 h_w=h_f+sum h_j；有泵/涡轮另列机械项

7. 后动量
   需要力时再取控制体，列 x/y 分量和所有外力

8. 算系数或误差
   lambda、zeta、mu、relative error、图表斜率

9. 检查
   单位、有效数字、方向、作用点、损失正值、系数量级、极限趋势
```

## 最低掌握标准

达到最低掌握，期末实验/综合题至少能保住步骤分：

1. 能把测压管读数、量水箱读数、砝码质量分别转成压强/水头、流量、力。
2. 能统一表压/绝压，不在同一方程里混用压强基准。
3. 能写出静水压强、静水总压力、连续方程、能量方程、动量方程的基本形式。
4. 能区分沿程损失 `hf` 和局部损失 `hj`，会写 `h_w=hf+sum hj`。
5. 能计算 `lambda`、`zeta`、`mu`、`Re` 和相对误差。
6. 能在动量题中画控制体，并说明求到的是“固体对流体”还是“流体对固体”。
7. 能完成 cm/g/kPa/L/min 到 SI 的单位换算。
8. 能写出 2-3 条具体误差来源，不只写空泛原因。

## 高分掌握标准

达到高分掌握，能把实验题和压轴综合题做成稳定得分点：

1. 能根据题面自动选择静水、能量、动量或组合方程，并写出适用条件。
2. 能说明测压管水头、压强水头、速度水头、损失水头之间的物理关系。
3. 能正确处理变径局部损失中的速度水头差，并明确 `zeta` 的参考速度。
4. 能用 `Re` 判断流区，并在双对数图中解释斜率与理论指数。
5. 能解释文透里流量系数小于 1、孔板流量系数更小、`mu` 随 `Re` 变化后趋稳。
6. 能把短管/长管题中的入口、出口、弯头、阀门、突扩、沿程损失全部放进能量账本。
7. 能把喷嘴/弯管法兰题中的压力力、重力、壁面力、动量通量分量列完整。
8. 能对异常数据提出物理解释：排气不净、未恒定、读数视差、喷嘴未对准、断面选错、沿程损失未扣除。
9. 能用有效数字和单位把答案写得像实验报告，而不是只给一个裸数。

## 复习卡片

### 卡片 1：测压管读数

问：测压管液面高程到底代表什么？

答：代表测点的测压管水头 `z+p/(rho g)`。通大气测管液面处表压为 0，测点压强由液面与测点的高差换算。

### 卡片 2：压强基准

问：能量方程里一边用表压一边用绝压可以吗？

答：不可以。全题必须统一压强基准；自由液面和射流出口常用表压 0。

### 卡片 3：静水总压力

问：平面总压力为什么不能只报 `P`？

答：完整答案还要有方向和作用点。实验中还要由力矩平衡求 `P实测` 并与 `P理论` 比较。

### 卡片 4：体积法流量

问：量水箱读数怎样转流量？

答：`Q=A_tank Delta H/t`。注意面积和水位差单位要配套。

### 卡片 5：文透里

问：文透里流量与压差水头是什么关系？

答：理论上 `Q理=K sqrt(Delta h)`；实际流量 `Q实=mu Q理`。

### 卡片 6：沿程损失

问：沿程损失系数怎么从实验算？

答：等直径管中 `hf` 可由测压管水头差得出，再用 `lambda=hf/((l/d)v^2/(2g))`。

### 卡片 7：局部损失

问：为什么局部损失不能总是等于测压管水头差？

答：前后速度可能不同，能量方程中还要加上速度水头差：`hj=H1-H2+(alpha1 v1^2-alpha2 v2^2)/(2g)`。

### 卡片 8：损失系数

问：`zeta` 的答案为什么必须说明参考速度？

答：`zeta=hj/(v_ref^2/(2g))`。突扩等变径构件用上游或下游速度会得到不同数值。

### 卡片 9：动量实验

问：动量方程最容易漏哪一步？

答：漏控制体和反作用力。方程中常先求固体对流体的力，题问流体对固体时要取反。

### 卡片 10：单位换算

问：实验题四个高频单位坑是什么？

答：cm -> m，g -> kg，kPa -> Pa，cm3/s -> m3/s。

### 卡片 11：图表

问：双对数图的斜率代表什么？

答：若 `hf ∝ v^m`，则 `ln hf - ln v` 图斜率为 `m`；若 `lambda ∝ Re^n`，则 `ln lambda - ln Re` 图斜率为 `n`。

### 卡片 12：误差分析

问：实验误差分析怎样避免空泛？

答：把误差写回仪器和机理：读数视差、水面波动、未恒定、排气不净、断面未在渐变流、喷嘴未对准、沿程损失未扣除、温度粘度取值不准。

### 卡片 13：综合题

问：连续、能量、动量三方程如何分工？

答：连续求速度关系，能量求压强/流量/损失，动量求力和反力。

## 专业拓展附注（非期末主线）

车院热泵空调、压气机性能实验属于专业拓展材料，当前资料无法确认其为流体力学期末主线。冲刺阶段不要把它放入必考路径；若题面真的出现，只按“实验读数 -> 物理量 -> 效率/性能参数”的通用数据处理思路处理。

- 热泵：关注温度、流量、功率和 `COP=收益热量/压缩机功率`。
- 压气机：关注质量流量、压比、转速、效率以及总温总压修正。
- 与本套主线的连接：量纲相似、可压缩流总参数、实验数据处理。
