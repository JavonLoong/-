## 高风险条件总表

| 先问什么 | 如果答案是 | 优先章节 | 禁忌 |
| --- | --- | --- | --- |
| 流体是否静止？ | 是 | 02、14 | 不要套伯努利或动量方程 |
| 是否只求流量/速度关系？ | 是 | 04 | 不能忘密度是否变化 |
| 是否沿同一流线求压强/速度/水头？ | 是 | 05 | 有明显损失时必须加 `h_w` |
| 是否求力、反力、喷流冲击、弯管受力？ | 是 | 06、14 | 不能只用伯努利，必须选控制体 |
| 是否涉及管道损失？ | 是 | 05、10、14 | 先判 `Re`，再选 `lambda` 和局部损失 |
| 是否是圆管层流解析题？ | 是 | 09 | `lambda=64/Re` 不能乱用到紊流 |
| 是否高速气体或喷管？ | 是 | 12 | 不要套低速不可压伯努利 |
| 是否跨激波或膨胀波？ | 是 | 13 | 等熵关系不能跨激波 |
| 是否是实验数据表？ | 是 | 14 | 原始读数必须先转 SI 物理量 |

## 公式索引

| 主线公式 | 何时用 | 何时不能用 | 细节 |
| --- | --- | --- | --- |
| `tau = mu du/dy` | 牛顿流体粘性切应力 | 非牛顿流体、速度梯度方向不清 | 01 |
| `p = p0 + rho g h` | 静止同种液体压强分布 | 流体运动、密度显著变化 | 02、14 |
| `P = p_C A` | 平面静水总压力大小 | 曲面总压力不能直接套 | 02、14 |
| `Q = int_A V·n dA`，一维 `Q=Av` | 流量、连续方程入口 | 变密度时不能直接写 `A1v1=A2v2` | 04 |
| `partial rho/partial t + div(rho V)=0` | 局部质量守恒 | 激波等不连续面优先用积分式 | 04 |
| `z+p/(rho g)+v^2/(2g)=C` | 理想流体伯努利 | 有泵/涡轮/损失/非定常/可压缩高速/跨激波 | 05 |
| `H1 + Hp = H2 + h_w + Ht` | 实际总流能量方程 | 未说明断面和损失项时不完整 | 05、10、14 |
| `sum F = partial_t int rho V dV + oint rho V(V·n)dA` | 控制体受力、喷流、弯管 | 把流体受力与壁面受力方向混淆 | 06、14 |
| `Gamma = oint V·ds`，`omega = curl V` | 有旋/无旋、环量 | 点涡奇点和多连通区域要单独说明 | 07 |
| `pi` 定理 | 变量多、要找无量纲群 | 变量不独立、重复变量含待求量 | 08 |
| `Re = vd/nu` | 层流/紊流和相似判据 | 速度、长度尺度取错 | 08、09、10 |
| `Q = pi R^4 Delta p/(8 mu L)` | 圆管充分发展层流 | 紊流、入口段、非圆管、非牛顿 | 09 |
| `h_f = lambda (L/D) v^2/(2g)` | 沿程损失 | `lambda` 口径不明、Fanning/Darcy 混用 | 10、14 |
| `h_j = zeta v_ref^2/(2g)` | 局部损失 | 不说明参考速度 | 10、14 |
| `a = sqrt(kRT)`，`Ma=V/a` | 可压缩气体判别 | 用滞止温度声速代替当地声速 | 12、13 |
| 等熵滞止关系 | 喷管等熵段 | 跨激波、强损失、非绝热 | 12 |
| 面积-Ma 关系 | 课件/公式表要求喷管定量时 | 未要求定量求支时以状态判断为主 | 12 |
| 正激波关系 | 激波垂直于来流 | 斜激波要先取法向马赫数 | 13 |
| Prandtl-Meyer 入口 | 超声速膨胀扇 | 压缩转角不能用膨胀扇公式 | 13 |

## 题型索引

| 题型 | 解题链条 | 章节 |
| --- | --- | --- |
| 静水压强/U 形管 | 定基准 -> 找等压面 -> 逐段 `rho g h` | 02、14 |
| 平面/曲面总压力 | 画压强图 -> 求大小 -> 求作用线/分力 | 02、14 |
| 流线、迹线、涡量判断 | 看定常 -> 写微分方程/求 `curl V` | 03、07 |
| 多入口多出口流量 | 画控制面 -> 进出通量守恒 | 04 |
| 文丘里/皮托管/虹吸 | 连续 + 伯努利；必要时加损失 | 04、05、14 |
| 喷嘴/弯管/射流冲击 | 连续/能量求速度 -> 动量求力 | 05、06、14 |
| 管路流量/水头损失 | 能量账本 -> `Re` -> `lambda/zeta` -> 迭代或代算 | 05、10、14 |
| Poiseuille 层流 | 压差驱动 -> 抛物线速度 -> 流量/剪应力 | 09 |
| Moody 图题 | 算 `Re` 和相对粗糙度 -> 查 `lambda` -> 能量方程 | 10 |
| 量纲分析 | 列变量 -> 写量纲矩阵 -> 选重复变量 -> 组 `pi` | 08 |
| 外流阻力/边界层 | 判 Re -> 看分离/阻力构成 -> 用阻力系数 | 11 |
| 喷管堵塞 | 总参数 -> 临界压力比 -> 喉部 Ma=1 -> 分段 | 12 |
| 激波/膨胀扇 | 先判断压缩还是膨胀 -> 正/斜激波或 PM -> 参数变化 | 13 |
| 实验数据处理 | 原始读数 -> SI 物理量 -> 方程 -> 误差/图表 | 14 |

## 跨章衔接

| 衔接 | 关键提醒 |
| --- | --- |
| 01 -> 05 -> 12 | 不可压缩不是一刀切：低速液体/低速气体可近似不可压；喷管、激波、高 Ma 气体必须进入 12/13。 |
| 04 -> 05/06 | 先问“流线还是控制体”：求压强/速度常走 05，求力/反力必须走 06；两者都要先画边界。 |
| 05 -> 10 -> 14 | 伯努利不是和损失对立；05 给能量账本，10 给 `h_f/h_j` 算法，14 给实验读数入口。 |
| 09 -> 10 | 09 的 Poiseuille 层流解接到 10 的工程损失框架，桥梁是 Darcy `lambda=64/Re`。 |
| 12 -> 13 | 12 的等熵关系只在等熵段用；过激波后总压下降，必须分段，不可一条等熵公式从头套到尾。 |
| 08 -> 10/11/14 | 量纲分析提供 `Re/Fr/Eu/Ma/We` 等准则，具体物理模型仍回对应章节。 |

## 符号禁混表

| 符号 | 可能含义 | 禁混提醒 |
| --- | --- | --- |
| `alpha` | 01 体膨胀系数；05/10 动能修正系数 | 看上下文和单位，修正系数通常无量纲 |
| `beta` | 01 压缩系数；06 动量修正系数；13 斜激波角 | 斜激波角是角度，压缩系数有量纲 |
| `mu` | 01 动力粘度；05/14 流量系数；13 马赫角常写作 `mu_M` | 粘度有单位 Pa·s，系数无量纲 |
| `p0` | 05 低速总压；12/13 可压缩滞止压 | 低速 `p+rho v^2/2` 不能跨到高 Ma 激波题 |
| `f` | 摩阻系数，也可能是 Fanning | 本套管损默认 Darcy 口径；英文资料先查 `64/Re` 还是 `16/Re` |
| `H` | 总水头、液面高度、几何高差 | 写清 `z+p/(rho g)+v^2/(2g)` 还是实验读数 |
| `zeta` | 局部损失系数 | 必须写参考速度 `v_ref` |

## 何时不能用

| 不能这样做 | 正确处理 |
| --- | --- |
| 伯努利跨激波 | 激波前后分段；跨激波用 13 的激波关系 |
| 两点不在同一流线仍套理想伯努利 | 若是总流断面，用总流能量方程；若求力，用控制体动量 |
| `A1v1=A2v2` 不看密度 | 可压缩流用 `rho A v` 守恒 |
| `64/Re` 套紊流 | 先判 Re；紊流查 Moody 图或经验式 |
| 局部损失只写 `zeta` 不写参考速度 | 明确 `h_j=zeta v_ref^2/(2g)` |
| 斜激波直接用整体 `M1` 套正激波 | 用法向马赫数 `M_n` |
| 膨胀转角用激波关系 | 膨胀扇用 Prandtl-Meyer 入口或方向判断 |
| 实验读数不转单位 | cm、mm、g、L/min 全部转 SI 后再算 |
