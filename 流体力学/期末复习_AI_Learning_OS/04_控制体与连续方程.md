## 章节一句话总纲

控制体与连续方程的核心不是背 `A_1V_1=A_2V_2`，而是把“同一批流体质点的质量守恒”翻译成“穿过控制体边界的法向质量通量平衡”：先选控制体和控制面，再看外法向速度，再判断密度是否变化、流动是否定常、是否不可压。

## 期末考试抓手

1. 概念题抓“系统 vs 控制体”：系统跟着同一批流体质点走，边界可变形、无质量交换；控制体是流场中确定空间区域，边界叫控制面，可以有质量进出。
2. 填空/简答抓雷诺输运定理的文字含义：系统内物理量的随体变化率 = 控制体内储存量变化率 + 通过控制面的净流出通量。
3. 推导题抓连续方程血缘：质量守恒 `dm_sys/dt=0`，在雷诺输运定理中令广延量为质量，即得积分形式连续方程。
4. 计算题抓三步：画控制体边界，标进出口外法向，列 `储存项 + 净流出项 = 0`。
5. 判断题抓条件：`nabla·V = 0` 的条件是不可压缩，不是“定常”或“理想流体”；定常可压缩流只能写 `nabla·(rho V)=0`。
6. 管流题抓质量流量：可压缩一维定常流用 `rho A V = 常数`，不可压缩才化为 `A V = Q = 常数`。
7. 多入口多出口题抓符号：按“流出为正、流入为负”写积分形式，或直接写 `sum dot m_in = sum dot m_out`（定常时）/ `dM_CV/dt = sum dot m_in - sum dot m_out`（非定常时）。

1. 为什么流体力学不直接跟踪系统，而要改用控制体？

3. 雷诺输运定理到底把什么“翻译”成了什么？
4. 连续方程为什么是质量守恒，而不是“体积守恒”？
5. 控制面上的通量为什么只看速度的法向分量？
6. 什么时候保留密度 `rho`，什么时候可以把 `rho` 消掉？
7. 定常和不可压分别消掉连续方程中的哪一类项？
8. 为什么 `A_1V_1=A_2V_2` 不是所有管流题都能直接用？

## 前置知识

| 前置知识 | 本章用法 | 不稳时的症状 |
| --- | --- | --- |
| 连续介质假设 | 允许把密度、速度看成空间时间连续函数 | 不知道为什么能写微分方程 |
| 拉格朗日法/欧拉法 | 系统对应拉格朗日，控制体对应欧拉 | 系统和控制体混用 |
| 密度、体积、质量 | `dm = rho dV`，质量通量为 `rho V_n dA` | 把体积流量当质量流量 |
| 速度分量与法向分量 | 控制面通量只取 `V·n` | 侧壁也误算通量，或进出口符号反 |
| 散度与高斯定理 | 积分形式和微分形式互相转换 | 会背 `nabla·V=0` 但不会检查速度场 |
| 定常/非定常 | 定常表示局部时间变化项为零 | 把定常等同于速度不变或不可压 |
| 不可压缩 | 流体微团密度随体不变，常见水流可近似 | 在可压气流中误消密度 |

→ 任意微小控制体 + 高斯定理
→ 微分形式连续方程
→ 条件简化：定常 / 不可压 / 一维均匀截面
→ 管流、液面变化、多入口多出口流量平衡

## 三遍法

### 第一遍：直觉理解

本章的直觉图像是“查账”：

- 系统账本：盯住同一批流体质点，质量永远不丢不增；难点是这批流体会跑、会变形，不方便工程计算。
- 控制体账本：在空间里画一个盒子，只统计盒子里存了多少、边界流进流出多少；工程上好用。
- 雷诺输运定理：把“跟着流体查账”翻译成“站在固定盒子旁查账”。
- 连续方程：专门查质量账。盒子里质量增加，必然来自净流入；盒子里质量减少，必然变成净流出。

最关键的边界直觉：

- 控制面上只有穿过边界的速度分量会带来质量交换。
- `V·n > 0`：沿外法向穿出去，是流出。
- `V·n < 0`：逆外法向穿进来，是流入。
- `V·n = 0`：沿壁面切向流或固壁无穿透，不贡献质量通量。

### 第二遍：公式推导

从系统质量守恒出发：

```text
系统质量 m_sys 不随时间变化
→ Dm_sys/Dt = 0
```

雷诺输运定理对任意广延量 `B`：

```text
系统中 B 的变化率
= 控制体内 B 的储存变化率
+ 通过控制面的 B 的净流出率
```

取 `B=m`，单位质量的质量 `b=1`：

```text
0 = partial/partial t int_CV rho dV + oint_CS rho(V·n)dA
```

这就是积分形式连续方程。对任意微小控制体成立，并用散度定理把面积分转成体积分：

```text
partial rho/partial t + nabla·(rho V) = 0
```

条件简化要分清：

1. 定常：`partial/partial t = 0`，得到 `nabla·(rho V)=0` 或积分式中控制体储存项为零。
2. 不可压缩：密度随体不变，常密度时可从散度里提出，得到 `nabla·V=0`。
3. 一维均匀截面：把面积分化成截面平均量，得到 `rho A V`。
4. 一维定常不可压：密度相同再消去，得到 `A V = Q`。

### 第三遍：题目应用

通用解题动作：

1. 写方程名：根据连续方程/质量守恒。
2. 选控制体：管段、水箱、三通或给定空间区域。
3. 画控制面：标出入口、出口、固壁、自由液面。
4. 标外法向：所有入口的 `V·n` 应为负，出口为正。
5. 判断密度：水/油常按不可压；气体、喷管、燃烧、密度给定变化时保留 `rho`。
6. 判断定常：若控制体内质量不随时间变，储存项为零；液面变化、充排水、密闭容器通常非定常。
7. 写质量平衡：优先写质量流量，再在不可压时化为体积流量。
8. 检查单位：`rho A V` 是 kg/s，`A V` 是 m^3/s，不能混加。

## 易错点与反例

1. 错误：系统就是控制体。
   反例：系统是一团被“染色”的流体，随流运动且无质量穿过其边界；控制体是一段管道空间，水不断进出。

2. 错误：控制面上所有速度都产生通量。
   反例：管壁上流体可以有切向速度，但无穿透时 `V·n=0`，质量通量为零。

3. 错误：入口通量写正。
   反例：外法向指向控制体外，入口处速度指向控制体内，所以 `V·n<0`；若用积分式，入口项天然为负。

4. 错误：定常流一定满足 `nabla·V=0`。
   反例：定常可压缩喷管中 `partial rho/partial t=0`，但密度沿程变化，正确形式是 `nabla·(rho V)=0`。

5. 错误：不可压缩必须定常。
   反例：水箱液面随时间升降，流动非定常，但水仍可按不可压处理；可用 `dV_water/dt = Q_in - Q_out`。

6. 错误：连续方程就是体积守恒。
   反例：不同密度气体混合或可压缩流中，守恒的是质量，不能直接令体积流量相等。

7. 错误：任何管流都用 `A_1V_1=A_2V_2`。
   反例：气体喷管或燃烧烟气密度变化时，应使用 `rho_1A_1V_1=rho_2A_2V_2`。

8. 错误：截面速度给了函数也用 `Q=AV`。
   反例：抛物线速度分布 `V=V_max(1-r^2/R^2)` 时，`Q=int_0^R V 2pi r dr = V_max A/2`。

9. 错误：多出口只看主出口。
   反例：一个入口两个出口的定常水流，应写 `rho_1A_1V_1 = rho_2A_2V_2 + rho_3A_3V_3`。

10. 错误：密闭水箱只算水的质量变化。
    反例：若题目明确液面上方有空气且控制体取整个密闭容器，控制体内空气体积也随 `h` 变；材料中的期中指南已将此作为特殊提醒。若题目是开口水箱，空气自由进出，一般不计空气储存项。

## 最低掌握标准/高分掌握标准

最低掌握标准：

1. 能用一句话区分系统和控制体。
2. 能写出雷诺输运定理的文字意义。
3. 能写出积分形式连续方程和微分形式连续方程。
4. 能判断 `nabla·V=0` 的条件是不可压缩。

## 复习卡片

### 卡片 1：系统 vs 控制体

问：系统和控制体最本质的区别是什么？

答：系统是同一批流体质点，质量不穿过系统边界；控制体是空间区域，流体可以穿过控制面进出。

### 卡片 2：雷诺输运定理

问：RTT 的一句话含义是什么？

答：系统中某广延量的随体变化率 = 控制体内该量的储存变化率 + 穿过控制面的净流出率。

### 卡片 3：通量正负

问：为什么入口通量为负？

答：控制面法向取外法向，入口速度指向控制体内部，与外法向相反，所以 `V·n<0`。

### 卡片 4：积分连续方程

问：积分形式连续方程怎么写？每项什么意思？

答：`partial_t int_CV rho dV + oint_CS rho(V·n)dA=0`。第一项是控制体内质量储存变化率，第二项是质量净流出率。

### 卡片 5：微分连续方程

问：可压非定常流动的连续方程是什么？

答：`partial rho/partial t + nabla·(rho V)=0`。

### 卡片 6：不可压缩简化

问：`nabla·V=0` 的真正条件是什么？

答：不可压缩。它不要求定常，也不等同于理想流体。

### 卡片 7：定常可压缩

问：定常可压缩流的连续方程是什么？

答：`nabla·(rho V)=0`；一维时为 `rho A V=常数`。

### 卡片 8：管径变为一半

问：不可压定常圆管流中直径变为一半，平均速度怎样变？

答：面积变为四分之一，速度变为四倍。

### 卡片 9：水箱液面

问：开口水箱不可压水流的液面变化率怎么记？

答：`A_0 dh/dt = Q_in - Q_out`，进多出少升，出多进少降。

### 卡片 10：多入口多出口

问：多入口多出口定常流怎样列连续方程？

答：先列质量流量：`sum dot m_in = sum dot m_out`；只有密度相同且不可压时，才能化为 `sum Q_in = sum Q_out`。
