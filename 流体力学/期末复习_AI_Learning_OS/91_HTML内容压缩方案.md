## 1. 14 章压缩卡片

| 章 | 1 句总纲 | 3 个关键词 | 1 个必考公式/判断 | 1 个禁用条件 | 补充/拓展标记 |
| --- | --- | --- | --- | --- | --- |
| 01 流体的物理性质 | 把真实流体先抽象成连续介质，再用密度、压缩性、粘性和表面张力判断后续方程能否简化。 | 连续介质；粘性；不可压近似 | 牛顿内摩擦定律：`tau = mu du/dy`，切应力看速度梯度，不看速度本身。 | 非牛顿流体、微纳/稀薄气体、速度梯度方向不清时，不直接套 `tau=mu du/dy`。 | `重度/比重`、`球面附加压强`、`Ma<0.3` 属考试通用补充/跨章衔接。 |
| 02 流体静力学 | 静止或相对静止流体中，压强梯度与质量力平衡，再把分布压强积分成压力、作用线和浮力。 | 静压分布；压力中心；浮力 | 同种静止液体：`p = p0 + rho g h`；平面总压力：`P = p_C A`。 | 流体明显运动、密度显著变化、不同液体跨界时，不能一段到底套常密度静压式。 | 浮体稳定性、旋转/加速相对平衡为考试通用补充；主线仍是静压、总压力、浮力。 |
| 03 流体运动学 | 把流动写成速度场，再用定常性、流线/迹线、物质导数、变形率和涡量把运动变成可计算对象。 | 速度场；物质导数；涡量 | `a = partial V/partial t + (V·nabla)V`；定常流也可能有迁移加速度。 | 非定常流求流线时不能把 `t` 当积分变量，必须先固定时刻。 | 速度梯度矩阵写法属考试通用补充，只作统一记忆，不替代课件基本定义。 |
| 04 控制体与连续方程 | 质量守恒不是体积守恒，而是控制体内储存量与穿过控制面的法向质量通量平衡。 | 控制体；质量通量；不可压简化 | 一般式：`partial rho/partial t + div(rho V)=0`；一维定常：`rho A V = const`。 | 可压缩流、密度变化流不能直接写 `A1 V1 = A2 V2`。 | 移动控制体、质量源项、激波积分式为考试通用补充；主线用固定控制体。 |
| 05 能量方程与伯努利方程 | 先判断机械能能否沿同一流线或缓变总流守恒，再把位置、压强、速度、损失、泵和涡轮写成水头账本。 | 伯努利；水头账本；损失项 | 实际总流：`H1 + Hp = H2 + h_w + Ht`，其中 `H=z+p/(rho g)+alpha v^2/(2g)`。 | 有损失、泵/涡轮、非定常、高速可压、跨激波或两点不在同一流线时，不能套理想伯努利。 | 无旋流全场伯努利、低速总压与可压缩滞止压区别为通用补充/跨章提醒。 |
| 06 动量方程与动量矩方程 | 先画控制体、外力、速度矢量和正方向，再用动量通量变化求喷流冲击、弯管反力和转矩。 | 控制体受力；动量通量；反作用力 | 定常一维总流：`sum F = sum_out beta rho Q V - sum_in beta rho Q V`，必须按分量列。 | 求力/反力不能只用伯努利；动量方程里的 `sum F` 是外界对控制体内流体的力，题问壁面受力要取反。 | 可动控制体、叶轮机欧拉公式、效率修正属考试通用补充。 |
| 07 理想流体有旋与有势流动 | 用环量和涡量判有旋/无旋，把二维不可压无旋流转成势函数或流函数的边值问题。 | 环量；势函数；流函数 | `Gamma = oint V·ds`，`Omega = curl V`；二维不可压无旋时 `nabla^2 phi=0`、`nabla^2 psi=0`。 | 不能凭流线弯曲判断有旋；理想势流固壁是不穿透，不是无滑移。 | Kelvin 定理、多连通区域、镜像法等只做考试通用补充。 |
| 08 量纲分析与相似原理 | 把多变量有量纲问题压缩成少数无量纲数，并按主导力选择模型相似准则。 | π 定理；重复变量；相似准则 | Buckingham：`k` 个变量、量纲矩阵秩 `r`，得到 `k-r` 个 `pi` 项。 | 变量不独立、重复变量含待求因变量或量纲不独立时，不能机械组 `pi` 项。 | 量纲矩阵秩、`St/Ca/We` 等扩展准则属考试通用补充；主线是 `Re/Fr/Eu/Ma` 和模型律取舍。 |
| 09 粘性不可压缩内部流动 | 圆管层流由压力梯度驱动、壁面无滑移制造速度梯度，最终连到 Poiseuille、压降和沿程损失。 | 层流；Poiseuille；充分发展 | 圆管充分发展层流：`Q = pi R^4 Delta p/(8 mu L)`，且 `lambda=64/Re`。 | 紊流、入口段、非圆管、非牛顿、可压缩显著或局部损失主导时，不能套 Poiseuille。 | 入口长度 `Le/d≈0.05Re`、N-S 粘性项直觉为考试通用补充。 |
| 10 紊流与管路损失 | 先由 `Re` 判流态，再用能量方程把沿程、局部、泵/涡轮和水头线放进同一本账。 | Re 判流；Moody；局部损失 | `h_f=lambda (L/D) v^2/(2g)`，`h_j=zeta v_ref^2/(2g)`；`lambda` 先判流态再选。 | 不算 `Re` 就选 `lambda`、不说明 `zeta` 参考速度、Darcy/Fanning 混用，都不能进入最终答案。 | Darcy/Fanning 区分是考试通用补充但需保留弱标签；孔口/管嘴是损失应用接口。 |
| 11 边界层与绕流阻力 | 高 Re 外流的粘性影响集中在薄边界层和分离尾涡区，真实阻力由摩擦阻力与压差阻力共同决定。 | 边界层；分离；阻力系数 | 分离抓手：`tau_w = mu (partial u/partial y)_w = 0`；阻力：`F_D=C_D(1/2)rho U^2 A`。 | 分离后尾涡区、强三维/非定常/强压缩流中，不能继续机械套平板边界层公式。 | Stokes 阻力、卡门涡街频率等为考试通用补充，不做本章主线。 |
| 12 可压缩流体一维流动 | 用马赫数打开可压缩模型，用等熵/绝热关系解释喷管加速、临界状态和质量流量堵塞。 | Ma；滞止参数；壅塞 | `Ma=V/a`、`a=sqrt(kRT)`；等熵临界：`p*/p0=[2/(k+1)]^(k/(k-1))`。 | 绝热不等于等熵；有摩擦、激波或强损失时，不能跨段用等熵总压关系。 | 面积-Ma 关系、质量流量函数、Fanno 趋势为考试通用补充；需按老师公式表优先。 |
| 13 可压缩超声速流动 | `Ma>1` 时压缩转角走激波、膨胀转角走膨胀扇；激波守恒但不可逆、熵增、总压下降。 | 正激波；斜激波；膨胀扇 | 斜激波第一步：`M1n=M1 sin beta`；正激波后 `M2<1`，斜激波弱解后整体 `M2` 可仍超声速。 | 斜激波不能用整体 `M1` 直接套正激波；激波跨越不能用等熵关系；压缩转角不能用 PM 膨胀公式。 | PM 函数、正激波总压比、等熵膨胀参数为考试通用补充，只保留查表/入口，不升主线。 |
| 14 实验与综合题抢分 | 先把读数翻译成 SI 物理量，再用静压、连续、能量、动量、损失和误差分析串成可评分链条。 | 读数翻译；单位换算；误差图表 | 体积法 `Q=A_tank Delta H/t`；实验统一流程：读数 -> SI 物理量 -> 方程 -> 误差/图表。 | 原始读数不转单位、表压绝压混用、砝码质量不乘 `g`、`zeta` 不写参考速度时，不能直接报结果。 | 热泵 COP、压气机性能为专业拓展（非期末主线），仅章末折叠附注。 |

## 2. 全局公式速查（10 条）

| 序号 | 速查公式 | 何时用 | 禁用提醒 | 对应章 |
| --- | --- | --- | --- | --- |
| F01 | `tau = mu du/dy`，`nu=mu/rho` | 牛顿流体剪切、粘性切应力、管流近壁 | 不能用 `nu` 直接代 `mu`；非牛顿/方向不清慎用 | 01、09、11 |
| F02 | `p=p0+rho g h`，`P=p_C A` | 静水压强、测压管、平面静水总压力 | 曲面压力不能直接套 `P=p_C A`；不同液体逐段算 | 02、14 |
| F03 | `Q=int_A V·n dA`，`dot m=int_A rho V·n dA`，一维 `rho A V=const` | 流量、控制体质量守恒、可压缩一维流 | 不可压才可化成 `A V=const`；非均匀截面要积分 | 03、04、12 |
| F04 | `a=partial V/partial t+(V·nabla)V`，`Omega=curl V` | 速度场加速度、涡量、有旋/无旋判断 | 定常不等于零加速度；流线弯曲不等于有旋 | 03、07 |
| F05 | `z+p/(rho g)+v^2/(2g)=C`；实际流 `H1+Hp=H2+h_w+Ht` | 伯努利、总流能量、泵管路、水头线 | 理想伯努利不能跨损失、泵、涡轮、非同流线、高速可压、激波 | 05、10、14 |
| F06 | `sum F = partial_t int rho V dV + oint rho V(V·n)dA`；定常一维 `sum F=out-in` | 喷嘴、弯管、射流冲击、支座反力 | `sum F` 是外界对流体的力；题问固体受力需取反 | 06、14 |
| F07 | `Gamma=oint V·ds`，`V=nabla phi`，二维不可压 `u=psi_y, v=-psi_x` | 有旋/无旋、势流、流函数、圆柱绕流 | 流函数不要求无旋；势函数满足 Laplace 还需不可压 | 07 |
| F08 | `n_pi=k-r`；`Re=UL/nu`，`Fr=U/sqrt(gL)`，`Ma=U/a` | 量纲分析、模型相似、主导力判断 | 变量不独立或比尺方向不清时，不能直接换算模型量 | 08、10、11、12 |
| F09 | `Q=pi R^4 Delta p/(8mu L)`；`lambda=64/Re`；`h_f=lambda(L/D)v^2/(2g)` | 圆管层流、沿程损失、Poiseuille 与 Darcy 衔接 | `64/Re` 只给圆管充分发展层流；紊流查 Moody/经验式 | 09、10 |
| F10 | `a=sqrt(kRT)`，`T0/T=1+(k-1)M^2/2`；正激波/斜激波按分段关系 | 可压缩喷管、壅塞、激波/膨胀方向判断 | 等熵关系不能跨激波；斜激波先用法向马赫数 | 12、13 |

## 3. 题型路径（10 条）

| 路径 | 题面信号 | 主 agent 可放入 HTML 的步骤 |

| P01 静水压强 / U 形管 | 静止水箱、压力表、测压管、油水银 | 定基准 -> 判表压/绝压 -> 找同种连通静止等压面 -> 逐段 `rho g h` -> 单位换 Pa 或水头 |
| P02 总压力 / 曲面 / 浮力 | 闸门、平板、圆弧、曲面、漂浮体 | 画压强图 -> 平面用形心深度求 `P` -> 压力中心取矩 -> 曲面拆 `Fx/Fz` -> 浮力用排开液体重量 |
| P03 速度场 / 流线 / 涡量 | 已知 `u,v,w,t`，求加速度、流线、迹线、有旋 | 拆分量 -> 判定常/维数 -> 流线固定时刻、迹线解 ODE -> 物质导数求加速度 -> `curl V` 判有旋 |
| P04 连续方程 / 多口流量 | 变径管、三通、多入口出口、水箱液面变化 | 画控制体和外法向 -> 写质量守恒 -> 定常用 `in=out` -> 非定常保留储存项 -> 不可压再消 `rho` |

| P06 动量受力 / 喷流弯管 | 喷嘴、射流冲击、弯管、法兰、支座反力 | 连续/能量先求速度和压强 -> 控制体包住流体 -> 列压力力、重力、壁面力 -> 动量分量式 -> 对固体受力取反 |
| P07 管流损失 / Poiseuille / Moody | 层流、紊流、沿程、局部、长管、短管、串并联 | `Q-A-v` -> 算 `Re` -> 层流用 `64/Re`/Poiseuille -> 紊流查 Moody/经验式 -> 列 `h_f + sum h_j` -> 串联损失加、并联损失等 |

## 4. “何时不能用”（8 条）

| 禁用项 | 不能用的场景 | 正确转向 |
| --- | --- | --- |
| C01 理想伯努利 | 有明显损失、泵/涡轮、非定常、高速可压、跨激波、两点不在同一流线 | 用实际总流能量方程；求力时转控制体动量；跨激波进第 13 章 |
| C02 `A1V1=A2V2` | 可压缩气流、密度变化、非均匀截面、多入口多出口未判清 | 用 `rho A V=const` 或面积分连续方程 |
| C03 `p=p0+rho g h` | 流体正在流动、相对静止条件未处理、密度随高度显著变化 | 静止才用静压；运动题转能量/动量；不同液体逐段算 |
| C04 `P=p_C A` | 曲面压力、两侧受压、封闭液面有均匀附加压强却未取矩 | 平面压力积分/压力中心；曲面拆水平分力与压力体 |
| C05 `64/Re` / Poiseuille | 紊流、过渡区、入口段、非圆管、非牛顿、短管局部损失主导 | 先判 Re；紊流用 Moody/经验式；入口和局部损失单列 |
| C06 `h_j=zeta v^2/(2g)` | 没说明参考速度，或断面选在急变/旋涡未恢复区域 | 明确 `v_ref` 是上游/下游/管内平均速度，断面取渐变段 |
| C07 正激波关系直接套斜激波 | 斜激波、楔形体绕流、给 `beta/delta` 的超声速转角题 | 先取 `M1n=M1 sin beta`，法向按正激波，最后还原整体 `M2` |
| C08 等熵关系跨不可逆段 | 激波、强摩擦、节流、非绝热、有轴功或总压损失段 | 分段：等熵段用滞止关系，激波段用激波关系，摩擦段只做趋势或按题给公式 |

## 6. 主 agent 整合注意事项

1. 章节卡片默认只显示：总纲、3 关键词、主公式/判断、禁用条件；补充内容只以 badge 或折叠条显示。
2. 全局速查优先服务考前最后一小时，公式不宜超过 10 条；如需更多公式，放入章节全文折叠。
3. “何时不能用”应放在右侧或顶部高风险区，优先于完整正文展示。
4. 第 12、13 章所有等熵/激波内容必须有“分段”提示；不要把等熵公式视觉上贯穿激波前后。
