## 章节一句话总纲

可压缩超声速流动的核心是：只在 `Ma>1` 的流场里讨论有限扰动如何变成激波或膨胀波；压缩转角通常对应激波、膨胀转角对应膨胀扇，激波按“绝热、无轴功、薄间断面”列守恒但过程不可逆、熵增且总压下降，斜激波计算必须抓法向分量和 `beta-delta-Ma` 关系。

## 期末考试抓手

1. 看到“超声速、激波、膨胀波、马赫线、喷管非设计工况、背压、楔形体绕流”，先写 `Ma>1`；若来流不是超声速，就不能直接套本章激波/膨胀波图像。
2. 正激波是垂直于来流的压缩间断：波前超声速，波后一定亚声速；静压、密度、温度升高，速度、马赫数降低，总温不变，总压降低。
3. 激波基本假设必须写清：定常坐标系下薄激波控制体、绝热、无轴功、忽略体力、理想气体；不是等熵过程。
4. 斜激波不是重新发明一套公式：先分解速度，切向分量不变，法向分量突变；用 `M1n=M1 sin beta` 代替正激波公式中的 `M1`。
5. 斜激波角 `beta` 与气流偏转角 `delta` 不是同一个角：`beta` 是激波面与来流方向夹角，`delta` 是气流转向角；给定 `M1` 和 `delta` 时常有弱解与强解两个 `beta`。
6. 弱解直觉：`beta` 小、靠近马赫角、压缩弱、波后常仍超声速，实际附着斜激波通常取弱解。强解直觉：`beta` 大、靠近正激波、压缩强、波后多为亚声速，通常需要下游高背压等约束。
7. `delta>delta_max` 时附着斜激波不能满足边界转角，激波脱体；脱体激波正中近似正激波，两侧逐渐过渡为斜激波。
8. 膨胀波是超声速气流绕凸角或向低压区膨胀形成的连续波组：速度和马赫数升高，压力、密度、温度降低；理想膨胀扇近似等熵，总压不损失。
9. 喷管题先比背压与设计出口压、出口正激波临界背压、喉部激波消失背压：背压低于设计出口压出外部膨胀波；略高于设计出口压出外部斜激波；再高时正激波进入渐扩段。
10. 喷管流量题抓喉部是否壅塞：第十三讲明确 `p2' < p2m` 时背压变化而喉部临界参数不变，流量保持不变；`p2' > p2m` 后流量减小，`p2'=p1` 停止流动。
11. 考试选择题常考“等熵/不可逆”陷阱：膨胀波可近似等熵，激波一定熵增；正激波熵增和波阻最大。
12. 计算题优先用参数比值，少算中间绝对量：`p2/p1`、`rho2/rho1`、`T2/T1`、`M2`，最后再求速度或压强。

## 核心问题

1. 为什么本章所有激波和膨胀波图像都必须先满足 `Ma>1`？
2. 激波为什么能用连续、动量、能量方程列式，却不能说是等熵过程？
3. 正激波前后哪些静参数增加，哪些速度参数降低，总温与总压怎样变化？
4. 正激波为什么一定把超声速流变为亚声速流？
5. 斜激波为什么只改变法向速度分量，而切向速度分量不变？
6. `M1n=M1 sin beta` 在斜激波题里为什么是第一公式？
7. `delta` 与 `beta` 的关系怎样决定弱解、强解和最大折转角？
8. 为什么实际附着斜激波通常取弱解，而大 `beta` 强解常“不自然”？
9. 膨胀波为什么是连续扇形波组而不是突跃间断？
10. Prandtl-Meyer 膨胀关系解决的是哪类“超声速转角”问题？
11. 喷管背压变化时，外部膨胀波、外部斜激波、出口正激波、管内正激波分别在什么条件下出现？
12. 激波熵增、总压下降、波阻之间怎样联系？

## 前置知识

| 前置知识 | 本章用法 | 不稳时的症状 |
| --- | --- | --- |
| 声速与马赫数 | 判断是否进入超声速扰动传播图像，`Ma=V/a` | 亚声速题误画马赫线或激波 |
| 马赫角 | 微弱扰动线角度 `mu=arcsin(1/Ma)` | 把马赫角和斜激波角混为一谈 |
| 理想气体状态方程 | 用 `p=rho R T` 联立参数比 | `p2/p1`、`rho2/rho1`、`T2/T1` 互相矛盾 |
| 一维定常绝热流 | 接第 12 章喷管、滞止参数、临界状态 | 不知道喉部是否壅塞 |
| 控制体守恒 | 对薄激波列质量、动量、能量方程 | 把激波当普通连续等熵流 |
| 动量方程 | 推出激波压强突升与速度突降 | 压力升高却速度也写升高 |
| 能量方程 | 说明绝热无功激波总温不变 | 把总温也写成下降或上升 |
| 熵与可逆性 | 判断激波不可逆、总压下降 | 误称“正激波等熵压缩” |
| 速度分解 | 斜激波分解法向/切向分量 | 用整体 `M1` 直接套斜激波静参数 |
| 三角函数与角度 | 处理 `beta`、`delta`、`mu` 的关系 | 度/弧度混用，或把角定义反 |

## 概念依赖图

```text
可压缩流动基础
-> 声速 a、马赫数 Ma、马赫角 mu=asin(1/Ma)
-> Ma>1：扰动只在马赫锥/马赫线内向下游传播
   -> 凹壁面/压缩转角：微弱压缩波汇聚
      -> 有限压缩间断：激波
         -> 正激波：波面垂直来流
            -> 薄激波控制体
            -> 连续 + 动量 + 绝热无轴功能量
            -> 静压/密度/温度升高，速度/Ma 降低
            -> 熵增、总压下降、波阻
         -> 斜激波：波面倾斜，流向偏转
            -> 速度分解：Vn 突变，Vt 不变
            -> M1n=M1 sin beta
            -> 用正激波关系计算法向压缩
            -> beta-delta-Ma 关系
            -> 弱解/强解、delta_max、脱体激波
   -> 凸壁面/膨胀转角：微弱膨胀波展开
      -> 膨胀波组 / Prandtl-Meyer 膨胀扇
      -> 近似等熵：V、Ma 升高；p、rho、T 降低；总压保持
-> 工程入口
   -> 超声速绕流：压缩角用斜激波，膨胀角用膨胀扇
   -> 缩放喷管非设计工况：背压决定外部波系或管内正激波
   -> 风洞/模型：正激波或斜激波参数计算
```

| Prandtl-Meyer 函数（考试通用补充） | `nu(M)=sqrt((k+1)/(k-1))*atan sqrt((k-1)(M^2-1)/(k+1))-atan sqrt(M^2-1)` | 微弱等熵膨胀波积分 | 二维定常、理想气体、无粘等熵、`M>1` | 凸角膨胀扇，`delta=nu(M2)-nu(M1)` | 压缩转角不能用膨胀扇代替激波 | `nu` 为角度，注意弧度/度 | 把膨胀波当激波套参数突跃 | 求膨胀后马赫数 |
| 等熵膨胀参数（考试通用补充） | `T0`、`p0` 不变；`T2/T1=(1+(k-1)M1^2/2)/(1+(k-1)M2^2/2)`；`p2/p1=(T2/T1)^(k/(k-1))` | 等熵滞止关系 | 理想膨胀扇，无激波损失 | 膨胀波前后静参数 | 激波跨越绝不能用等熵关系 | 静压/静温比无量纲 | 膨胀后压力写升高 | 膨胀扇计算 |
| 喷管壅塞质量流量接口 | 喉部 `M=1` 后，若背压继续降低但喉部临界状态不变，则质量流量不变 | 一维等熵喷管 + 临界状态 | 缩放喷管，入口滞止参数给定 | 判断背压变化对流量影响 | 激波进入喉前后需重新判断 | `G` 或 `m_dot` kg/s | 以为背压越低流量永远越大 | 喷管非设计工况 |

## 三遍法

### 第一遍：直觉理解

本章的直觉是“超声速流动来不及提前商量”。

在亚声速流里，压力扰动能向上游传播，流体可以提前调整；在 `Ma>1` 里，扰动被限制在马赫线/马赫锥下游，前方流体不知道后面有物体或转角。于是：

- 遇到压缩转角：许多压缩扰动会互相追赶、叠加，最后形成很薄的压缩间断，也就是激波。
- 遇到膨胀转角：许多微弱膨胀扰动按角度展开，形成连续膨胀波组，不需要突跃间断。
- 激波像“突然刹车并压缩”：速度和马赫数降，压力、密度、温度升，机械能的一部分不可逆地变成内能，总压损失。
- 膨胀波像“连续放松并加速”：速度和马赫数升，压力、密度、温度降；理想情况下没有熵增，总压不降。
- 正激波是最强的正面刹车；斜激波是斜着刹车，只刹法向分量，切向分量继续滑过去。

### 第二遍：公式推导

**1. 正激波：从薄控制体守恒出发。**

取一个随激波固定的薄控制体，假设绝热、无轴功、忽略体力和厚度内细节。对波前 1、波后 2：

```text
质量：rho1 V1 = rho2 V2
动量：p1 + rho1 V1^2 = p2 + rho2 V2^2
能量：h1 + V1^2/2 = h2 + V2^2/2
理想气体：h=cpT, p=rhoRT
```

这四类关系联立，得到正激波参数比。因为能量方程没有热和轴功，`T0` 不变；但动量方程允许压强突升，热力学第二定律要求过程熵增，所以 `p0` 必须下降。

**2. 斜激波：把速度拆成法向与切向。**

对斜激波，波面切向没有压力差，因此切向速度保持：

```text
V1t = V2t
rho1 V1n = rho2 V2n
M1n = M1 sin beta
```

法向方向上完全像一个正激波，所以静压比、密度比、温度比都可把正激波公式中的 `M1` 换成 `M1n`。波后整体马赫数还要由几何关系恢复：

```text
M2n = M2 sin(beta - delta)
```

再结合速度方向转过 `delta`，得到 `beta-delta-M` 关系。固定 `M1` 时，`delta(beta)` 有最大值；小 `beta` 是弱解，大 `beta` 是强解。

**3. 膨胀波：把有限膨胀拆成无穷多个微弱等熵转角。**

膨胀扇不是突跃压缩，而是一串微弱扰动波。每个微小转角都可近似等熵，积分后得到 Prandtl-Meyer 函数：

```text
delta = nu(M2) - nu(M1)
```

膨胀后 `M2>M1`，`p2<p1`，`T2<T1`，`rho2<rho1`。这部分函数公式在第十三讲课件中未完整展开，作为考试通用补充掌握到“会判断、会查表/代公式入口”即可。

**4. 喷管非设计工况：背压控制波系位置。**

缩放喷管设计工况下，喉部达到临界状态，渐扩段继续膨胀到超声速，出口压力等于设计背压。背压改变时：

```text
背压低于设计出口压 -> 出口外部膨胀波，喷管内仍按设计膨胀
背压略高于设计出口压 -> 出口外部斜激波/压缩波，喷管内仍可保持超声速设计曲线
背压继续升高 -> 出口正激波，再到正激波进入渐扩段
背压高到激波退至喉部并消失 -> 渐扩段全亚声速，喷管像文丘里管
背压等于入口压力 -> 无流动
```

### 第三遍：题目应用

通用解题动作：

1. 判状态：来流是否 `Ma>1`，题目是压缩转角、膨胀转角、正激波、斜激波，还是喷管背压题。
2. 写假设：激波题写薄激波、定常、绝热、无轴功、理想气体；膨胀波题写理想等熵膨胀；喷管题写一维等熵段与可能的激波损失。
3. 正激波：从 `M1` 查或算 `p2/p1`、`rho2/rho1`、`T2/T1`、`M2`，再求速度、压强或总压。
4. 斜激波：先画图标 `beta` 和 `delta`；用 `M1n=M1 sin beta`；把法向马赫数代入正激波关系；最后用 `M2 sin(beta-delta)=M2n` 还原整体 `M2`。
5. 给定 `M1,delta` 求斜激波：用 `beta-delta-M` 关系或图表找两个解；考试默认通常取弱解，除非题目说明强激波或下游约束。
6. 膨胀波：用 `delta=nu(M2)-nu(M1)` 求 `M2`，再用等熵关系求 `p,T,rho`；判断方向时记住膨胀后压力下降、速度上升。
7. 喷管题：画压力沿程曲线，比较背压区间；判断波在出口外、出口截面、渐扩段内、喉部，还是全管亚声速。
8. 检查参数趋势：激波后静压/静温/密度必须升高，总压必须下降；膨胀扇后静压/静温/密度必须降低，总压近似不变。
9. 检查单位与角度：三角函数角度是度还是弧度；`M` 无量纲；温度必须用 K。
10. 写结论：不仅给数值，还要写“弱解/强解”“附着/脱体”“总压下降/熵增”等物理判断。

## 易错点与反例

1. 错误：看到压缩就用等熵关系。  
   反例：激波压缩是突跃压缩，熵增；同一压比下激波压缩后温度高于等熵压缩，密度低于等熵压缩。  
   避坑：激波跨越不用等熵关系，波前和波后各自到滞止态时才可用等熵关系。

2. 错误：正激波后仍写 `Ma2>1`。  
   反例：第十三讲明确“超声速气流通过正激波后一定变成亚声速”。  
   避坑：正激波算完必须检查 `M2<1`。

3. 错误：斜激波后也一定亚声速。  
   反例：斜激波后法向分量为亚声速，但整体速度还含切向分量；弱斜激波后常仍为超声速。  
   避坑：只说 `M2n<1`，整体 `M2` 要另算。

4. 错误：斜激波用整体 `M1` 直接套正激波公式。  
   反例：同一 `M1` 下，小 `beta` 弱斜激波压缩很弱，远小于正激波。  
   避坑：第一步永远写 `M1n=M1 sin beta`。

5. 错误：混淆斜激波角 `beta` 与气流偏转角 `delta`。  
   反例：`delta=0` 时既可能退化为马赫波，也可能是正激波；但 `beta` 分别为马赫角或 `90 deg`。  
   避坑：图上标清“波面与来流”为 `beta`，“流线转过”为 `delta`。

6. 错误：给定 `M1,delta` 只求出一个斜激波角。  
   反例：当 `delta<delta_max` 时通常有弱/强两个 `beta`。  
   避坑：说明两个解；若无特别约束，工程附着斜激波多取弱解。

7. 错误：任何楔角都能产生附着斜激波。  
   反例：`delta>delta_max` 时形成脱体激波，正中近似正激波。  
   避坑：大转角先比 `delta_max`。

8. 错误：膨胀波也让压力升高。  
   反例：第十三讲膨胀波处 `V2>V1`，压力、密度、温度下降。  
   避坑：膨胀 = 加速降压；激波 = 减速升压。

9. 错误：膨胀扇有总压损失。  
   反例：理想 Prandtl-Meyer 膨胀是连续等熵波组，总压保持；激波才有总压下降。  
   避坑：把“等熵膨胀”和“不可逆激波”分开。

10. 错误：激波“能量损失”表示总温下降。  
    反例：绝热无功激波总焓和总温不变，损失体现在熵增和总压下降。  
    避坑：总温看能量，总压看可逆性。

11. 错误：喷管背压低于设计出口压时，喷管内部压力曲线也改变。  
    反例：第十三讲指出微弱扰动波不能逆流越过马赫线向上传播，喷管内仍按设计曲线膨胀。  
    避坑：超声速出口外扰动不能逆传入喷管。

12. 错误：喷管内出现正激波后，激波后渐扩段仍按超声速规律加速降压。  
    反例：正激波后为亚声速，亚声速在渐扩段减速增压。  
    避坑：先判激波后 `M<1`，再用亚声速扩压直觉。

13. 错误：背压从 `p2k` 升到 `p2m` 时流量也随背压变化。  
    反例：第十三讲选择题对应区间中，激波向喉部移动、强度变小、流量不变。  
    避坑：只要喉部临界参数不变且仍壅塞，流量不变。

14. 错误：运动激波直接用实验室坐标速度套驻激波公式。  
    反例：复习注意事项强调激波理论是驻激波理论，匀速运动激波需做坐标系转换。  
    避坑：换到激波固定坐标系，用相对速度作波前速度。

15. 错误：用斜激波总压表项时不看说明。  
    反例：复习注意事项提醒斜激波不能用正激波关系表中的 `p02/p1` 直接替代。  
    避坑：静参数用 `M1n`；总压比也按法向激波损失处理，但不要把整体正激波表项照搬。

## 最低掌握标准 / 高分掌握标准

### 最低掌握标准

1. 能写出本章只针对 `Ma>1` 的超声速扰动图像。
2. 能背出正激波后参数趋势：`p,rho,T` 升，`V,M` 降，`T0` 不变，`p0` 降，熵增。
3. 能写出正激波控制方程：连续、动量、绝热无轴功能量。
4. 能用给定 `M1` 计算正激波 `p2/p1`、`rho2/rho1`、`T2/T1`、`M2`。
5. 能说明斜激波“法向按正激波，切向不变”。
6. 能写 `M1n=M1 sin beta`，知道 `beta` 与 `delta` 不同。
7. 能判断弱解/强解、小转角附着、大转角脱体。
8. 能说清膨胀波使气流加速降压，理想过程等熵。
9. 能按背压区间判断喷管出口外膨胀波、出口外斜激波、管内正激波、全亚声速的大致图像。
10. 能完成 10 分钟内的正激波基础计算。

### 高分掌握标准

1. 能从三守恒方程推导正激波关键参数式，并解释每个假设的物理意义。
2. 能由 `beta-delta-M` 关系分析两个斜激波解，并说明实际为什么常取弱解。
3. 能把斜激波计算完整做成“`M1n` -> 静参数比 -> `M2n` -> `M2` -> 总压损失”的链条。
4. 能判断 `delta>delta_max` 的脱体激波，并描述正中正激波、两侧斜激波的结构。
5. 能用 Prandtl-Meyer 入口式处理膨胀扇，至少会写 `delta=nu(M2)-nu(M1)` 和等熵参数变化。
6. 能处理“压缩角 + 膨胀角”的超声速绕流综合题，正确画波系。
7. 能解释激波熵增、总压下降、波阻之间的联系。
8. 能在喷管非设计工况中判断激波位置、强度、流量随背压变化的趋势。
9. 能识别运动激波题需要换到激波固结坐标系。
10. 能在选择题中快速排除“激波等熵、总压升高、斜激波后必亚声速”等陷阱。

## 复习卡片

| 卡片 | 正面 | 背面 |
| --- | --- | --- |
| 概念卡 1 | 本章为什么先看 `Ma>1`？ | 只有超声速时扰动不能向全场提前传播，才有马赫线、激波、膨胀扇这套图像。 |
| 概念卡 2 | 激波是绝热无功，为什么还不可逆？ | 绝热无功只保证总焓/总温不变；激波内部有粘性和热传导效应的不可逆结果，熵增、总压下降。 |
| 趋势卡 1 | 正激波后哪些量升高？ | 静压 `p`、密度 `rho`、静温 `T`、熵 `s` 升高。 |
| 趋势卡 2 | 正激波后哪些量降低？ | 速度 `V`、马赫数 `M`、总压 `p0` 降低；总温 `T0` 不变。 |
| 公式卡 1 | 正激波三守恒是什么？ | `rho1V1=rho2V2`；`p1+rho1V1^2=p2+rho2V2^2`；`h1+V1^2/2=h2+V2^2/2`。 |
| 公式卡 2 | 正激波 `M2^2` 公式？ | `M2^2=(1+(k-1)M1^2/2)/(kM1^2-(k-1)/2)`。 |
| 条件卡 1 | 激波公式的基本假设？ | 定常薄激波控制体、绝热、无轴功、忽略体力、理想气体；跨激波不可逆。 |
| 斜激波卡 1 | 斜激波计算第一步？ | 写 `M1n=M1 sin beta`，先判断 `M1n>1`。 |
| 斜激波卡 2 | 斜激波前后哪个速度分量不变？ | 切向分量不变：`V1t=V2t`；法向分量按正激波突变。 |
| 角度卡 1 | `beta` 和 `delta` 分别是什么？ | `beta` 是激波面与来流方向夹角；`delta` 是气流通过激波后的偏转角。 |
| 角度卡 2 | 给定 `M1,delta` 时弱/强解怎样分？ | 小 `beta` 为弱激波，大 `beta` 为强激波；通常附着斜激波取弱解。 |
| 脱体卡 | 什么时候出现脱体激波？ | `delta>delta_max(M1)` 或钝头/大转角时，附着斜激波无法满足偏转。 |
| 膨胀卡 1 | 膨胀波后参数怎么变？ | `V,M` 升高；`p,rho,T` 降低；理想情况下等熵，总压保持。 |
| 膨胀卡 2 | Prandtl-Meyer 入口公式？ | `delta=nu(M2)-nu(M1)`，用于超声速凸角等熵膨胀。 |
| 喷管卡 1 | 背压低于设计出口压会怎样？ | 出口外出现膨胀波，称膨胀不足；喷管内仍按设计膨胀。 |
| 喷管卡 2 | 背压略高于设计出口压会怎样？ | 出口外出现斜激波/压缩波，背压继续升高可到出口正激波。 |
| 喷管卡 3 | 管内正激波随背压升高如何移动？ | 从出口附近向喉部移动，强度减弱；在对应区间流量仍不变。 |
| 易错卡 1 | “斜激波后一定亚声速”对吗？ | 不对。法向分量波后亚声速，整体 `M2` 可能仍超声速。 |
| 易错卡 2 | “激波后总压升高”对吗？ | 不对。静压升高，总压因熵增而下降。 |
| 易错卡 3 | “膨胀波和激波都可用正激波表”对吗？ | 不对。膨胀波用等熵/Prandtl-Meyer，激波用突跃关系。 |
