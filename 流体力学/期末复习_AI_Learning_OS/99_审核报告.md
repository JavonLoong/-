## 总索引需要补充的衔接说明

建议主 agent 在 `00_期末复习总索引.md` 明确补上以下衔接：

1. **`01 -> 05 -> 12`：不可压缩判据不是一刀切**

4. **`09 -> 10`：Poiseuille 与 Darcy 的关系**
