# 02_流体静力学

## 章节一句话总纲

流体静力学的核心是：在静止或相对静止条件下，用压强梯度与质量力平衡确定压强分布，再把分布压强对平面、曲面或物体表面积分成总压力、作用线和浮力。

本章做题必须先写清四件事：

1. **静止条件**：绝对静止，还是相对某个加速坐标系静止；静止流体内无相对运动，切应力不起作用。
2. **坐标方向**：常用 `z` 轴竖直向上，深度 `h` 向下；若改坐标，公式符号必须跟着改。
3. **压强基准**：用绝对压强、表压强还是真空度；水面通大气时，总压力题通常用表压。
4. **作用点**：总压力不仅要大小和方向，还要作用线或压力中心。

### 标准解题顺序

1. 判定流体是否静止或相对静止；若流动明显，转去能量方程或动量方程。
2. 画坐标：建议 `z` 向上，深度 `h` 向下；在图上标出自由液面、交界面和所求点。
3. 选压强基准：通大气面表压为 0；封闭容器或真空计题先换算表压/绝压。
4. 写压强分布：同种静止流体中 `p = p0 + rho g h`，或 `z + p/(rho g) = C`。
5. 总压力题先画压强分布图：压力方向垂直受压面，压强随竖直深度线性增大。
6. 平面题求 `F` 和压力中心；曲面题求 `Fx/Fz`、压力体方向和合力作用线。
7. 最后检查单位、方向、基准、作用点是否齐全。

1. 为什么静止流体中只有法向压强、没有切应力？

3. 重力场中为什么压强只随竖直高度变化？

5. 绝对压强、表压强、真空度如何互换？
6. 液柱式测压计如何逐段列压强方程？
7. 平面上的总压力为什么等于形心处压强乘面积？
8. 压力中心为什么通常在形心下方？
9. 曲面总压力为什么必须分解为水平分力和竖直分力？
10. 压力体如何画，实压力体和虚压力体如何定方向？
11. 浮力为什么等于排开液体重量，作用线为什么过浮心？

| 公式名称 | 表达式 | 来源定律 | 推导前提 | 适用条件 | 不适用条件/易错 |
| --- | --- | --- | --- | --- | --- |
| 静压强定义 | `p = lim(Delta F_n / Delta A)` | 表面力定义 | 取微小面积，法向受压 | 流体内部或流固接触面 | 压强是标量，压力方向还要由作用面法线确定 |
| 静压强方向 | `dF = p dA`，方向垂直并指向受压面 | 静止流体剪切平衡 | 流体质点之间无相对运动 | 静止或相对静止流体 | 流动粘性剪切问题不能套用“无切应力” |
| 欧拉平衡方程 | `nabla p = rho f` | 微元体力平衡 | 静止，单位质量力为 `f` | 绝对静止或相对静止并把惯性力并入 `f` | 漏掉惯性力会误判加速容器、旋转容器 |
| 分量式 | `partial p/partial x = rho f_x`，`partial p/partial y = rho f_y`，`partial p/partial z = rho f_z` | 欧拉平衡方程 | 直角坐标 | 写坐标方向后使用 | `z` 轴改向时符号要重写 |
| 全微分式 | `dp = rho(f_x dx + f_y dy + f_z dz)` | `dp = nabla p · dr` | 压强连续可微 | 液柱逐段推导、等压面判断 | 不同液体密度不同，不能一段到底 |
| 重力场静压分布 | `dp = -rho g dz` | 欧拉平衡方程，`f = -g k` | `z` 竖直向上 | 静止、重力场、同种均质液体 | 若用深度 `h` 向下，写成 `dp = rho g dh` |
| 基本方程 | `p = p0 + rho g h` | 对 `dp = rho g dh` 积分 | `rho` 常数，同一液体 | 水、油等近似不可压液体 | 气体密度随高度变时不能直接用常密度式 |
| 测压管水头 | `z + p/(rho g) = C` | 重力场基本方程 | 同种静止液体，`rho` 常数 | 测压管、连通器 | 跨不同液体时 `rho` 变，常数不同 |
| 等压面条件 | `dp = 0`，即 `f_x dx + f_y dy + f_z dz = 0` | 全微分式 | 静止流体 | 找同压点、液体交界面 | “水平面等压”还需同种、连通、仅重力场 |
| 表压 | `p_g = p_abs - p_a` | 压强基准换算 | 当地大气压已知 | 压力表读数、开口水箱 | 表压可正可负 |
| 真空度 | `p_v = p_a - p_abs = -p_g` | 压强基准换算 | `p_abs < p_a` | 真空计、虹吸/密闭容器 | 真空度不是绝对压强 |
| 液柱差压 | `Delta p = rho g Delta h` | 静压基本方程 | 同种液柱、竖向高差 | 测压管、U 形管 | 上下方向符号错最常见 |
| 平面总压力大小 | `F = int_A p dA = p_C A = rho g h_C A` | 分布压力积分 | 表压，且自由液面表压为 0 | 任意形状平面 | 封闭液面 `p0 != 0` 时应写 `F = (p0 + rho g h_C)A` |
| 平面压力中心 | `y_D = y_C + I_C/(y_C A)` | 对自由液面交线取矩 | `y` 沿斜面向下，`p = rho g y sin theta` | 自由液面表压为 0 的倾斜平面 | 这里的 `y` 是斜面距离，不是竖直深度 |
| 压力中心深度 | `h_D = h_C + I_C sin^2 theta/(h_C A)` | 上式换成竖直深度 | `h = y sin theta` | 需报竖直水深时 | 不能把 `I_C/(y_C A)` 直接当竖直增量 |
| 平面带液面压力 | `y_D = (p0 A y_C + rho g sin theta I_x)/(p0 A + rho g sin theta A y_C)` | 分布压力取矩 | 液面表压 `p0` 为常数 | 密闭容器、加压水箱 | 均匀压力项作用在形心，会把压力中心往形心拉 |
| 曲面水平分力 | `F_x = int p dA_x = rho g h_Cx A_x` | 曲面压力投影 | 投影到垂直于 `x` 的平面 | 曲面水平方向分力 | `A_x` 是投影面积，不是曲面面积 |
| 曲面竖直分力 | `F_z = rho g V_p` | `int p dA_z` 化为压力体重量 | 压力体从曲面竖直画到自由液面或延长面 | 曲面竖直分力 | 实压力体向下，虚压力体向上 |
| 曲面合力 | `F = sqrt(F_x^2 + F_z^2)` | 正交分力合成 | 二维剖面或柱面问题 | 圆弧闸门、圆柱面 | 方向要写 `tan theta = F_z/F_x`，还要写作用线 |
| 圆弧特判 | 合力作用线通过圆心 | 各微元压力法线过圆心 | 圆弧/圆柱面 | 圆弧闸门压轴题 | 只适用于圆弧法线共心的曲面 |
| 浮力 | `F_B = rho_l g V_dis` | 曲面总压力积分，阿基米德定律 | 静止液体 | 潜体、浮体 | `V_dis` 是排开液体体积，不一定是物体总体积 |
| 相对平衡直线等压面 | `p = C - rho g z - rho a x`，等压面 `a x + g z = const` | 欧拉方程加入惯性力 | 容器以加速度 `a` 平动 | 加速水箱、斜液面 | 考试中先画等效重力方向 |
| 旋转相对平衡 | `z = C + omega^2 r^2/(2g)` | 离心惯性力与重力合成 | 刚体式旋转，无相对运动 | 旋转圆筒液面 | 这是相对静止，不是普通水平等压面 |

## 三遍法

### 第一遍：直觉理解

1. 静止流体为什么只“压”不“剪”？
   - 若存在切应力，流体微团会被切向拉动而产生相对运动，这与静止矛盾。
   - 所以静止流体对任意面只提供法向压力，方向垂直作用面并指向受压面。

2. 为什么越深压强越大？
   - 某点上方液柱的重量必须由下方更大的压强托住。
   - 在同一种静止液体中，向下走一段 `Delta h`，压强增加 `rho g Delta h`。

3. 为什么同一水平面不总是等压？
   - 在“同种、连通、静止、质量力仅为重力”的液体中，同一水平面等压。
   - 若中间隔着不同液体、不连通、容器加速或旋转，直接套水平等压会错。

4. 为什么平面压力中心在形心下方？
   - 总压力大小可以用形心压强平均化，但实际压强下大上小。
   - 因为下部权重更大，合力作用点被压强分布“拉”向更深处。

5. 为什么曲面要分解？
   - 曲面上每个微元的法线方向不同，压力不是平行力系。
   - 先把每个微元压力投影到水平和竖直方向，分别合成再还原总力。

### 第二遍：公式推导

#### 1. 欧拉平衡微分方程

取静止微元流体，单位体积所受质量力为 `rho f`，压强合力等效为 `-nabla p`。平衡条件为：

```text
rho f - nabla p = 0
```

所以：

```text
nabla p = rho f
```

分量式：

```text
partial p/partial x = rho f_x
partial p/partial y = rho f_y
partial p/partial z = rho f_z
```

全微分式：

```text
dp = rho(f_x dx + f_y dy + f_z dz)
```

物理意义：静止流体中单位体积质量力由压强梯度平衡。

#### 2. 重力场基本方程

若 `z` 轴竖直向上，重力为 `f = -g k`：

```text
partial p/partial x = 0
partial p/partial y = 0
partial p/partial z = -rho g
```

因此：

```text
dp = -rho g dz
```

对同种均质液体积分：

```text
p + rho g z = C
z + p/(rho g) = C'
```

以自由液面为 `z = 0`，液面压强为 `p0`，向下深度为 `h`：

```text
p = p0 + rho g h
```

#### 3. 平面总压力

对平面面积 `A`，表压 `p = rho g h`：

```text
F = int_A p dA = rho g int_A h dA
```

设形心深度为 `h_C`：

```text
int_A h dA = h_C A
F = rho g h_C A = p_C A
```

若沿斜面从自由液面交线向下取坐标 `y`，倾角为 `theta`，则 `h = y sin theta`：

```text
F = rho g sin theta int_A y dA
  = rho g sin theta y_C A
```

压力中心由力矩定义：

```text
F y_D = int_A y dF
      = rho g sin theta int_A y^2 dA
      = rho g sin theta I_x
```

所以：

```text
y_D = I_x/(y_C A)
```

由平行轴定理 `I_x = I_C + A y_C^2`：

```text
y_D = y_C + I_C/(y_C A)
```

#### 4. 曲面总压力

曲面微元压力 `dF = p dA`，方向沿法线。对某水平轴向的分力：

```text
dF_x = p dA_x
F_x = int p dA_x
```

这等于曲面在相应铅直投影面上的静水总压力：

```text
F_x = rho g h_Cx A_x
```

竖直方向：

```text
F_z = int p dA_z = rho g int h dA_z = rho g V_p
```

其中 `V_p` 是压力体体积。实压力体方向向下，虚压力体方向向上。

### 第三遍：题目应用

#### A. 液柱测压题动作模板

1. 把所有已知自由液面、压力表、真空计换成同一压强基准。
2. 选已知点起算：通大气液面表压为 0；绝压题可从 `p_a` 起算。
3. 沿管路走：
   - 同种液体向下 `+rho g h`；
   - 同种液体向上 `-rho g h`；
   - 到同一连续液体的同一水平面可横向跨过去。
4. 每跨一种液体换一次 `rho`。
5. 到所求点列等式，最后说明表压/绝压/真空度。

#### B. 平面总压力题动作模板

1. 判断用表压还是绝压；若闸门两侧都通大气，通常用表压。
2. 找受压面积 `A` 和形心竖直深度 `h_C`。
3. 写大小：`F = rho g h_C A`。
4. 找作用点：
   - 标准倾斜平面用 `y_D = y_C + I_C/(y_C A)`；
   - 矩形压强图可用三角形/梯形分布的形心；
   - 两侧均有水时，对两侧分别算力和力矩，再求净合力作用点。
5. 写方向：压力垂直于平面并指向受压面。

#### C. 曲面总压力题动作模板

1. 画曲面剖面和宽度，别漏进深 `b/L`。
2. 水平分力：取曲面在铅直平面上的投影面积，找投影形心深度。
3. 竖直分力：从曲面边界竖直画到自由液面或延长面，得到压力体。
4. 判方向：液体和压力体在曲面同侧为实压力体，向下；异侧为虚压力体，向上。
5. 合成：`F = sqrt(F_x^2 + F_z^2)`，`tan theta = F_z/F_x`。
6. 作用线：
   - `F_x` 过投影面压力中心；
   - `F_z` 过压力体重心；
   - 圆弧/圆柱面合力作用线通过圆心。

#### D. 浮力题动作模板

1. 判断物体是全浸没还是漂浮。
2. 求排开液体体积 `V_dis`。
3. 写 `F_B = rho_l g V_dis`，方向竖直向上。
4. 作用线过排开液体体积的形心，即浮心。
5. 漂浮平衡还要写 `F_B = W`。

| 等压面判断 | 多个水平断面、油水、水银 | `dp=0`，同种连通重力场 | 先列等压面成立条件 | 只看“同高”不看是否同种连通 |
| 测压管水头 | 测压管、静水实验 | `z+p/(rho g)=C` | 统一基准面，读液面高程 | 把测压管读数当绝压 |
| U 形管/复式差压计 | 多液体、多高差 | 逐段 `+/- rho g h` | 从已知点一路走到未知点 | 水银段方向、真空度符号错 |
| 平面闸门总压力 | 矩形闸门、倾斜平板 | `F = rho g h_C A` | 先形心深度，再面积 | 漏宽度，漏方向 |
| 平面压力中心 | 问作用点/压力中心 | `y_D = y_C + I_C/(y_C A)` | 指明 `y` 沿斜面，报清距液面/顶端/底端 | 把压力中心写成形心 |
| 两侧受水的闸门 | 左右水深不同、不同密度 | 分别算 `F1,F2` 和力矩 | 净力 `F=F1-F2`，净作用点用力矩 | 直接相减两个压力中心 |
| 曲面总压力 | 圆柱面、圆弧闸门 | `F_x=rho g h_C A_x`，`F_z=rho g V_p` | 画投影和压力体 | 用曲面面积算 `F_x` |
| 圆弧闸门作用线 | 题干提示圆心/转轴 | 合力作用线过圆心 | 用分力合成方向确定作用点深度 | 忘写作用线过圆心 |
| 浮力 | 潜体、浮体、排开液体 | `F_B=rho g V_dis` | 方向向上，过浮心 | 用物体密度代替液体密度 |
| 相对平衡 | 容器加速/旋转、液面倾斜 | 加入惯性力后用欧拉方程 | 等压面垂直等效质量力 | 仍按水平等压面处理 |

## 易错点与反例

1. **把压强当力**
   - 错误反例：压力表读数 `39228 Pa`，直接说水箱受力是 `39228 N`。
   - 正确处理：`Pa = N/m^2`，要求力必须再乘受压面积；求水深则用 `p = rho g h`。

2. **没声明压强基准**
   - 错误反例：真空度 `0.98 kPa` 写成 `p = 0.98 kPa`。
   - 正确处理：若用表压，真空度对应 `p_g = -0.98 kPa`；若用绝压，`p_abs = p_a - 0.98 kPa`。

3. **滥用“同一水平面压强相等”**
   - 错误反例：油和水中同高度两点直接令压强相等。
   - 正确处理：只有同种、连通、静止、重力场中的水平面才可直接等压；不同液体交界面处压强连续，但密度改变。

4. **坐标方向和公式符号打架**
   - 错误反例：设 `z` 向上，却写 `dp = rho g dz`。
   - 正确处理：`z` 向上时 `dp = -rho g dz`；深度 `h` 向下时 `dp = rho g dh`。

5. **倾斜平面用斜长当竖直深度**
   - 错误反例：倾斜闸门长度中心为 `L/2`，直接写 `h_C = L/2`。
   - 正确处理：`h_C` 是竖直深度；若闸门与水平夹角为 `theta`，相应竖直增量是 `y sin theta`。

6. **压力中心写成形心**
   - 错误反例：平面总压力作用点在形心。
   - 正确处理：除非压强均匀，否则压力中心在形心下方；用 `y_D = y_C + I_C/(y_C A)`。

7. **带压液面还用零表压公式**
   - 错误反例：封闭水箱液面上方有气压，还直接 `F = rho g h_C A`。
   - 正确处理：用 `F = (p0 + rho g h_C)A`，作用点也要把均匀压力项计入力矩。

8. **曲面水平分力用曲面面积**
   - 错误反例：圆弧面 `F_x = rho g h_C A_曲面`。
   - 正确处理：`F_x` 用铅直投影面积 `A_x`。

9. **压力体方向判反**
   - 错误反例：只要画出压力体就默认 `F_z` 向下。
   - 正确处理：液体与压力体在曲面同侧为实压力体，向下；异侧为虚压力体，向上。

10. **漏进深宽度**
    - 错误反例：二维圆弧剖面题只用弧长/投影高度，忘乘闸门宽度 `b`。
    - 正确处理：投影面积和压力体体积都要乘进深。

11. **两侧水压力直接相减数值，不求净作用点**
    - 错误反例：左侧 `F1`、右侧 `F2` 得出 `F=F1-F2` 后结束。
    - 正确处理：若问作用点，必须对同一参考点取矩：`F l = F1 l1 - F2 l2`。

12. **把浮力体积取成物体总体积**
    - 错误反例：漂浮木块浮力用整个木块体积。
    - 正确处理：浮力取排开液体体积；漂浮时由 `rho_l g V_dis = W` 反求浸没体积。

13. **相对静止题仍按普通水平液面**
    - 错误反例：加速容器中仍令自由液面水平。
    - 正确处理：自由液面是等压面，必须垂直等效质量力；平动倾斜，旋转成抛物面。

## 最低掌握标准 / 高分掌握标准

### 最低掌握标准

达到以下水平，本章基础题可以保住：

1. 能写出静压强两个特性：垂直作用面、各向同性。
2. 能从 `nabla p = rho f` 写出重力场 `p = p0 + rho g h`。
3. 能区分绝对压强、表压强、真空度，并做基本换算。
4. U 形管题能按“向下加、向上减、换液换密度”列式。
5. 平面总压力会算 `F = rho g h_C A`。
6. 压力中心知道在形心下方，能用矩形三角形压强图求简单作用点。
7. 曲面题知道分解为水平分力和竖直分力。
8. 能用 `F_B = rho g V_dis` 求基础浮力。

### 高分掌握标准

达到以下水平，本章压轴题才稳：

1. 能在答案第一行说明静止/相对静止条件和坐标方向。
2. 会处理封闭容器带压液面、真空计、多个液体密度的组合测压题。
3. 平面总压力会用惯性矩公式求压力中心，并能处理两侧水压力的净作用点。
4. 曲面压力能准确画压力体，区分实/虚压力体方向。
5. 圆弧闸门题能利用“合力作用线过圆心”快速定位作用线。
6. 能解释为什么绝压和表压在两侧同大气作用时会互相抵消。
7. 能处理相对平衡基础题：平动液面倾斜、旋转液面抛物面。
8. 考试通用补充：知道浮体稳定性基础判据 `GM > 0` 稳定、`GM = 0` 中性、`GM < 0` 不稳定；材料未确认高频计算，掌握概念即可。

## 复习卡片

### 概念卡

**Q1：静止流体压强的两个特性是什么？**  
A：方向垂直并指向受压面；同一点静压强大小与作用面方位无关。

**Q2：欧拉平衡方程的物理意义是什么？**  
A：静止流体中，单位体积质量力与压强梯度平衡，即 `nabla p = rho f`。

**Q3：为什么绝对静止均质液体的等压面是水平面？**  
A：重力竖直向下，等压面与质量力正交，所以等压面水平。

**Q4：压力中心是什么？**  
A：分布压力合力作用线与受压面的交点。

**Q5：压力体是什么？**  
A：为求曲面竖直分力，从曲面竖直画到自由液面或其延长面形成的几何体；其“重量”给出竖直分力大小。

### 公式卡

**Q6：重力场静压强基本公式是什么？**  
A：`p = p0 + rho g h`，其中 `h` 是从参考面向下的竖直深度。

**Q7：测压管水头公式是什么？**  
A：同种静止液体中 `z + p/(rho g) = C`。

**Q8：表压、绝压、真空度如何换算？**  
A：`p_g = p_abs - p_a`，`p_v = p_a - p_abs = -p_g`。

**Q9：平面总压力大小如何算？**  
A：`F = p_C A = rho g h_C A`；若液面表压不为零，`F = (p0 + rho g h_C)A`。

**Q10：平面压力中心公式是什么？**  
A：自由液面表压为零时，沿斜面坐标 `y_D = y_C + I_C/(y_C A)`。

**Q11：曲面水平分力和竖直分力如何算？**  
A：`F_x = rho g h_Cx A_x`，`F_z = rho g V_p`。

**Q12：浮力公式是什么？**  
A：`F_B = rho_l g V_dis`，方向竖直向上，作用线过浮心。

### 条件卡

**Q13：什么时候可以说同一水平面压强相等？**  
A：静止、连通、同种均质流体、质量力仅为重力。

**Q14：什么时候不能直接用 `p = p0 + rho g h`？**  
A：流体不静止、密度显著变化、跨多种液体、坐标方向未定义、相对平衡中未加入惯性力时。

**Q15：总压力题为什么通常用表压？**  
A：若受压面另一侧也受同一大气压，大气压产生的均匀压力相互抵消；但封闭容器或单侧受压要明确基准。

### 易错卡

**Q16：U 形管最容易错在哪里？**  
A：上下方向符号、不同液体密度、真空度符号、等压面条件。

**Q17：平面总压力最容易错在哪里？**  
A：把斜长当深度、漏乘宽度、压力中心写成形心。

**Q18：曲面总压力最容易错在哪里？**  
A：水平分力用曲面面积、压力体画错、实/虚压力体方向判反、漏写作用线。

**Q19：圆弧闸门的快速检查是什么？**  
A：合力作用线必须通过圆心；若答案没有体现这一点，要回查方向和作用点。

**Q20：浮体稳定性基础怎么记？**  
A：考试通用补充：小倾斜后若稳心 `M` 在重心 `G` 之上，即 `GM > 0`，浮体稳定；反之不稳定。
