### 第 2 页：第 2 章 流体静力学（整页，重点放图 2/3/4）

- 2.1 静水压强与等压面：`p = p_0 + rho g h`
- 2.2 静力学微分方程：`dp/dx = rho X`，`dp/dy = rho Y`，`dp/dz = rho Z`
- 2.3 平面壁总压力
  - 合力：`F = (p_0 + rho g h_c) A`
  - 压力中心：
    - 沿斜面：`y_D = y_C + J_C / (y_C A)`
    - 垂深形式：`h_D = h_c + J_C sin^2(theta) / (h_c A)`
  - 常见图形：矩形、三角形、圆的 `y_C`、`J_C`
- 2.4 曲面壁总压力
  - 水平分力：`F_H = (p_0 + rho g h_{cH}) A_H`
  - 竖直分力：`F_V = p_0 A_V + rho g V`
  - 合力：`F = sqrt(F_H^2 + F_V^2)`，`tan(alpha) = F_V / F_H`
  - 压力体：判断“上压下托”
- 2.5 非惯性坐标系下的静止液体
  - 平移：`tan(alpha) = a_x / (g + a_z)`
  - 匀速转动：`z = omega^2 r^2 / (2g) + C`

这一页必须保留：

### 第 3 页：第 3 章 + 第 4 章前半（这一页开始向第 4 章倾斜）

第 3 章 流体运动学基础：

- 3.1 描述流体运动的分析方法
  - 拉格朗日法
  - 欧拉法
- 3.2 流场几何描述及质点导数
  - 流线方程：`dx/u = dy/v = dz/w`
  - 迹线方程：`dx/dt = u`，`dy/dt = v`，`dz/dt = w`
  - 质点导数：`D/Dt = partial/partial t + u partial/partial x + v partial/partial y + w partial/partial z`
  - 加速度分量：
    - `a_x = u_t + u u_x + v u_y + w u_z`
    - `a_y = v_t + u v_x + v v_y + w v_z`
- 3.3 流体微团运动分析
  - 线变形率：`epsilon_x = partial u / partial x`
  - 角变形率：`epsilon_xy = 1/2 (partial u / partial y + partial v / partial x)`
- 3.4 流场旋度
  - `omega_z = 1/2 (partial v / partial x - partial u / partial y)`
- 3.5 势函数/流函数
  - 流函数：`u = partial psi / partial y`，`v = - partial psi / partial x`
  - 势函数：`u = partial phi / partial x`，`v = partial phi / partial y`
- 图 1 对应内容要单列小框：
  - `dy/dx = v/u`
  - 若化为一阶线性方程：`y' + P(x) y = Q(x)`
  - 积分因子：`exp(int P(x) dx)`

### 第 4 页：第 4 章后半（整页，必须铺满）

- 4.4 欧拉积分形式的基本方程
  - 质量守恒：
    - `partial/partial t int_cv rho dV + int_cs rho (V·n) dA = 0`
  - 动量方程：
    - `sum F = partial/partial t int_cv rho V dV + int_cs rho V (V·n) dA`
  - 动量矩方程：
    - `sum M = partial/partial t int_cv r x rho V dV + int_cs r x rho V (V·n) dA`
  - 能量方程：保留总式和使用条件
- 4.5 理想流体 Bernoulli 方程
  - `p + 1/2 rho V^2 + rho g z = C`
  - 或水头式：`p/(rho g) + alpha V^2/(2g) + z = const`
  - 连续方程联立：`Q = VA`，`A_1 V_1 = A_2 V_2`
  - 条件单列标红：
    - 定常
    - 不可压
    - 无粘
    - 沿流线
- 4.6 流体动力学微分型基本方程
  - 应力矢量/应力张量：`p_n = n · P`
  - 微分连续方程：`partial rho / partial t + nabla · (rho V) = 0`
  - 不可压：`nabla · V = 0`
  - 欧拉方程：`rho DV/Dt = rho f - nabla p`
  - 若课上涉及，再补 Navier-Stokes 结构式
