# 水波实验中光的折射校正 —— 理论推导

## 一、问题描述

在实验中，我们将手机（相机）固定在水槽上方某一特定位置进行俯拍。由于水有一定深度，光线从水中传播到空气中时会发生**折射**，导致我们在图像中观测到的水波位置与其在水中的真实位置之间存在偏差。

本文档推导从**观测坐标**到**真实坐标**的换算公式，并说明程序实现的思路。

---

## 二、物理模型

### 2.1 坐标系建立

```
  手机相机 C (x_c, y_c, H)        ← 相机位于水面上方高度 H 处
       │
       │  空气 (折射率 n₁ = 1)
       │
  ─────┼──────────────────────  ← 水面 (z = 0)
       │
       │  水 (折射率 n₂ ≈ 1.33)
       │
       ●                        ← 水下目标点 P_real (x, y, -h)
  ─────────────────────────────  ← 水槽底部 (z = -h)
```

- **相机位置**：固定在水面正上方，光心投影到水面的点为 $(x_c, y_c)$，距水面高度 $H$
- **水深**：$h$（从水面到水槽底部/目标物的深度）
- **折射率**：空气 $n_1 = 1$，水 $n_2 = n \approx 1.33$

### 2.2 Snell 定律（折射定律）

光线从水中射入空气时，在水面处满足 Snell 定律：

$$n_1 \sin\theta_1 = n_2 \sin\theta_2$$

即：

$$\sin\theta_1 = n \sin\theta_2$$

其中：
- $\theta_1$：光线在空气中与法线（竖直方向）的夹角
- $\theta_2$：光线在水中与法线的夹角

## 三、公式推导

### 3.1 几何关系推导

设水下目标点的真实位置为 $(x, y)$，相机观测到的表观位置为 $(X, Y)$。

**在空气中**（从水面到相机）：

光线在空气中的水平偏移量：
$$\Delta_{air} = H \tan\theta_1$$

**在水中**（从目标点到水面）：

光线在水中的水平偏移量：
$$\Delta_{water} = h \tan\theta_2$$

**观测位置**是相机沿直线看到的表观位置，即：

$$X - x_c = (H + h) \cdot \frac{\Delta_{air}}{H} \quad \text{(若不考虑折射)}$$

但实际上由于折射，真实位置为：

$$x - x_c = \Delta_{air} + \Delta_{water} = H\tan\theta_1 + h\tan\theta_2$$

### 3.2 引入比例系数 k

相机观测到的方向角对应的水平距离为：

$$X - x_c = H \tan\theta_1$$

所以 $\tan\theta_1$ 可以直接从观测坐标得到：

$$\tan\theta_1 = \frac{X - x_c}{H}$$

（注意：这里 $X - x_c$ 和 $H$ 需要使用同一单位，都用像素或都用物理长度）

真实位置与观测位置的关系：

$$x - x_c = H\tan\theta_1 + h\tan\theta_2$$

由 Snell 定律：$\sin\theta_1 = n\sin\theta_2$

$$\tan\theta_2 = \frac{\sin\theta_2}{\cos\theta_2} = \frac{\sin\theta_1 / n}{\sqrt{1 - \sin^2\theta_1 / n^2}}$$

$$= \frac{\sin\theta_1}{\sqrt{n^2 - \sin^2\theta_1}}$$

$$= \frac{\tan\theta_1 \cos\theta_1}{\sqrt{n^2 - \sin^2\theta_1}}$$

因此：

$$x - x_c = H\tan\theta_1 + h \cdot \frac{\tan\theta_1 \cos\theta_1}{\sqrt{n^2 - \sin^2\theta_1}}$$

$$= \tan\theta_1 \left( H + \frac{h\cos\theta_1}{\sqrt{n^2 - \sin^2\theta_1}} \right)$$

又因为 $X - x_c = H\tan\theta_1$，所以：

$$x - x_c = \frac{X - x_c}{H} \left( H + \frac{h\cos\theta_1}{\sqrt{n^2 - \sin^2\theta_1}} \right)$$

$$\boxed{x = x_c + (X - x_c) \cdot \left(1 + \frac{h\cos\theta_1}{H\sqrt{n^2 - \sin^2\theta_1}}\right)}$$

### 3.3 整理为通用公式

利用 $\sin^2\theta_1 = \frac{\tan^2\theta_1}{1 + \tan^2\theta_1}$ 和 $\cos\theta_1 = \frac{1}{\sqrt{1+\tan^2\theta_1}}$：

令 $r^2 = (X - x_c)^2 + (Y - y_c)^2$，$\tan^2\theta_1 = \frac{r^2}{H^2}$

代入化简得：

$$x = x_c + (X - x_c) \cdot \frac{H}{H + \frac{h}{\sqrt{n^2 + (n^2 - 1)\tan^2\theta_1}}}$$

$$\boxed{x = x_c + (X - x_c) \cdot \frac{H}{H + \frac{h}{\sqrt{n^2 + (n^2-1) \cdot \frac{(X-x_c)^2 + (Y-y_c)^2}{H^2}}}}}$$

**y 方向同理**：

$$\boxed{y = y_c + (Y - y_c) \cdot \frac{H}{H + \frac{h}{\sqrt{n^2 + (n^2-1) \cdot \frac{(X-x_c)^2 + (Y-y_c)^2}{H^2}}}}}$$

### 3.4 隐含方程的说明

> ⚠️ **重要提示**：上述公式中，等号右侧出现了 $(X, Y)$ 即观测坐标，但在某些推导路径下，$\tan\theta_1$ 的表达式中会出现真实坐标 $(x,y)$，使其成为隐含方程。
>
> **在本实验的实际情况中**：由于 $\tan\theta_1 = (X - x_c)/H$ 是从相机端直接确定的（相机看到的角度），所以**公式可以直接计算，无需迭代**。

但如果需要更高精度（考虑水面不完全平整等因素），可以使用迭代法求解。我们的程序中同时提供了直接计算和迭代求解两种模式。

## 四、特殊情况分析

### 4.1 小角度近似（相机正下方附近）

当 $\theta_1 \to 0$ 时：

$$x - x_c \approx (X - x_c) \cdot \left(1 + \frac{h}{nH}\right)$$

**校正量正比于水深 h，反比于相机高度 H。**

### 4.2 垂直正下方（$\theta_1 = 0$）

无偏差：$x = X$，$y = Y$

### 4.3 误差估算

对于典型实验参数（$H = 30$ cm, $h = 10$ cm, $n = 1.33$）：

| 距画面中心距离 | $\theta_1$ | 位置偏差 |
|:---:|:---:|:---:|
| 0 cm | 0° | 0 mm |
| 5 cm | 9.5° | ~1.2 mm |
| 10 cm | 18.4° | ~2.6 mm |
| 15 cm | 26.6° | ~4.3 mm |
| 20 cm | 33.7° | ~6.5 mm |

可见偏差是**不可忽略的**，校正是必要的。
